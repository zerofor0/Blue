#### 2.3 概率连续性的应用：Borel-Cantelli引理

**（一）Borel-Cantelli引理**

【定义】设 $A_n$ 是事件列，$\limsup_{n\to\infty} A_n$ 表示事件列中无穷多次发生的事件（即“有无穷多个 $A_n$ 发生”）。

【重点】**Borel-Cantelli引理（BC引理）**
设 $A_n$ 是事件列。
1. 若 $\sum_{n=1}^{\infty} P(A_n) < \infty$，则 $P(\limsup_{n\to\infty} A_n) = 0$。
2. 若 $A_n$ 相互独立，且 $\sum_{n=1}^{\infty} P(A_n) = \infty$，则 $P(\limsup_{n\to\infty} A_n) = 1$。

【易错点】
- 结论2必须要求事件列 $A_n$ **相互独立**，而结论1不需要独立性条件。
- $\limsup_{n\to\infty} A_n$ 的对偶事件是 $\liminf_{n\to\infty} A_n^c$，即 $P(\liminf A_n) = 1 - P(\limsup A_n^c)$。

【例题：待补充】（课件P21抛硬币例题）
试验：抛掷一枚均匀硬币无穷多次。记事件 $H_n$ 表示第 $n$ 次抛掷得到正面，$P(H_n) = 1/2$。
- 求无穷多次出现正面的概率：由于 $H_n$ 独立且 $\sum P(H_n) = \infty$，由BC引理结论2知 $P(\limsup H_n) = 1$。
- 求某时刻后一直为正面的概率：$P(\liminf H_n) = 1 - P(\limsup H_n^c) = 1 - 1 = 0$。

#### 2.4 条件概率

**（一）条件概率的定义与性质**

【定义】设 $(\Omega, \mathcal{F}, P)$ 是概率空间，$A, B \in \mathcal{F}$，且 $P(A) > 0$。在事件 $A$ 发生的条件下，事件 $B$ 发生的条件概率定义为：
$$P(B|A) = \frac{P(A \cap B)}{P(A)}$$

【重点】条件概率是概率测度
用 $P_A(\cdot) = P(\cdot|A)$ 表示在事件 $A$ 发生条件下的条件概率，则 $(\Omega, \mathcal{F}, P_A)$ 也是一个概率空间。
- 非负性：$P(B|A) \ge 0$
- 规范性：$P(\Omega|A) = 1$
- 可列可加性：对互不相容的事件列 $B_i$，有 $P(\bigcup_{i=1}^{\infty} B_i | A) = \sum_{i=1}^{\infty} P(B_i|A)$
【记忆】条件概率具有概率的所有性质，原有概率公式中的无条件概率均可替换为条件概率。

【易错点】条件概率 $P(B|A)$ 与无条件概率 $P(B)$ 的大小关系**不确定**。
- 当 $B \subset A$ 时，$P(B|A) = \frac{P(B)}{P(A)} \ge P(B)$
- 当 $A \cap B = \emptyset$ 时，$P(B|A) = 0 \le P(B)$

**（二）基于条件概率的概率模型计算**

【解题步骤】利用条件概率计算复杂事件概率（如雷达探测问题）：
1. 明确已知条件概率（如 $P(B|A), P(B|A^c)$）和无条件概率（如 $P(A)$）。
2. 利用乘法公式求交集概率：$P(A \cap B) = P(A)P(B|A)$。
3. 利用全概率公式求边缘概率：$P(B) = P(A)P(B|A) + P(A^c)P(B|A^c)$。
4. 利用贝叶斯公式（定义）求后验概率：$P(A|B) = \frac{P(A \cap B)}{P(B)}$。

【例题：待补充】（课件P29雷达探测例题）
已知 $P(A)=0.05, P(B|A)=0.99, P(B|A^c)=0.10$。
求 $P(A|B)$。
解：
$P(A \cap B) = P(A)P(B|A) = 0.05 \times 0.99 = 0.0495$
$P(B) = P(A)P(B|A) + P(A^c)P(B|A^c) = 0.05 \times 0.99 + 0.95 \times 0.1 = 0.1445$
$P(A|B) = \frac{P(A \cap B)}{P(B)} = \frac{0.0495}{0.1445} \approx 0.34$

#### 2.5 乘法公式

**（一）序贯树形图**

【解题步骤】利用序贯树形图计算概率：
1. 建立树形图，关心的事件处于叶子节点，根到叶子的路径代表事件发生。
2. 在路径的每个分枝上写上相应的条件概率。
3. 叶子事件发生的概率 = 根到该叶子路径上所有分枝条件概率的乘积。

**（二）乘法公式定理**

【必背】**乘法公式**
设 $(\Omega, \mathcal{F}, P)$ 是概率空间，$A_i \in \mathcal{F}, i=1,\dots,n$，且 $P(A_1 \cap A_2 \cap \dots \cap A_{n-1}) > 0$，则：
$$P\left(\bigcap_{i=1}^{n} A_i\right) = P(A_1) \prod_{i=2}^{n} P(A_i | A_1 \cap A_2 \cap \dots \cap A_{i-1})$$

【易错点】
- 公式成立的前提是 $P(A_1 \cap \dots \cap A_{n-1}) > 0$，以保证所有条件概率均有定义。
- 事件的顺序可任意换序，共有 $n!$ 个等价公式，选择合适的顺序是解题关键。

**（三）乘法公式的典型应用：配对问题**

【高频】**配对问题（错排问题）**
$n$ 封信装入 $n$ 个信封，求没有一个正确（$q_0$）和恰有 $r$ 个正确（$q_r$）的概率。
设 $A_i$ = “第 $i$ 个信封地址正确”。
利用乘法公式求交集概率：
- $P(A_i) = \frac{1}{n}$
- $P(A_i \cap A_j) = P(A_i)P(A_j|A_i) = \frac{1}{n} \cdot \frac{1}{n-1} = \frac{(n-2)!}{n!}$
- $P(A_1 \cap \dots \cap A_k) = \frac{(n-k)!}{n!}$

结合容斥原理求至少一个正确的概率：
$P(\bigcup_{i=1}^n A_i) = \sum P(A_i) - \sum P(A_i \cap A_j) + \dots + (-1)^{n-1} P(\bigcap A_i)$
$= \frac{1}{1!} - \frac{1}{2!} + \dots + (-1)^{n-1} \frac{1}{n!}$

【必背】配对问题核心结论：
1. 全错排概率：$q_0 = 1 - P(\bigcup A_i) = \sum_{k=0}^{n} \frac{(-1)^k}{k!}$。当 $n \to \infty$ 时，$q_0 \to e^{-1} \approx 0.368$。
2. 恰有 $r$ 个正确的概率：$q_r = \frac{1}{r!} \sum_{k=0}^{n-r} \frac{(-1)^k}{k!}$。

#### 2.6 全概率公式

**（一）全概率公式思想**

【重点】全概率公式的核心思想是“化整为零”。将样本空间 $\Omega$ 分割为互不相容的事件 $A_1, A_2, \dots, A_n$（即 $A_i \cap A_j = \emptyset$ 且 $\bigcup A_i = \Omega$），则事件 $B$ 的概率可以表示为各分割块上条件概率的加权和。
$$P(B) = \sum_{i=1}^{n} P(A_i)P(B|A_i)$$

#### 2.7 本节总结与速查

**（一）核心结论**

1. **BC引理**：级数收敛则无穷多次发生概率为0；独立且级数发散则无穷多次发生概率为1。
2. **条件概率**：本质是样本空间的缩减（从 $\Omega$ 缩减至 $A$），它完全符合概率测度的所有公理和性质。
3. **乘法公式**：用于求多个事件同时发生的概率，关键在于合理拆解事件发生的先后顺序。
4. **配对问题**：经典模型，结合乘法公式与容斥原理，记住 $e^{-1}$ 的极限结果。

| 公式名称 | 表达式 | 适用条件 |
| :--- | :--- | :--- |
| 条件概率 | $P(B\|A) = \frac{P(A \cap B)}{P(A)}$ | $P(A) > 0$ |
| 乘法公式 | $P(\bigcap_{i=1}^n A_i) = P(A_1) \prod_{i=2}^n P(A_i\|A_1 \dots A_{i-1})$ | $P(A_1 \dots A_{n-1}) > 0$ |
| 全概率公式 | $P(B) = \sum_{i=1}^n P(A_i)P(B\|A_i)$ | $A_i$ 互不相容且并为 $\Omega$ |
