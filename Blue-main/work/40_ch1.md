#### 第1讲 概率模型与性质

##### 1.1 概率模型的基本构成

###### （一）样本空间与事件
*   **【定义】样本空间（Ω）**：一个试验所有可能结果的集合。试验结果必须满足**互斥**且**完整**。
*   **【定义】事件（A）**：样本空间的子集，即某些试验结果的集合。
*   **样本空间的分类**：
    *   **离散型**：试验结果为有限个或可列无限个。常通过序贯树形图刻画（如连续抛掷骰子多次）。
    *   **连续型**：试验结果不可列，通常为实数轴上的区间或多维空间中的区域（如到达时间的延迟）。
*   **【易错点】** 样本空间的选择具有主观性。对同一试验，应根据关心的信息选择合适的模型，既要包含足够细节，又要避免不必要的繁琐。连续抛三次硬币只能算作**一次试验**，不能认为是三次试验。

###### （二）概率与概率公理
*   **【定义】概率（P）**：为事件 $A$ 分配的一个非负数 $P(A)$，刻画了事件 $A$ 发生的信念程度。
*   **【必背】概率三大公理**：
    1.  **非负性**：对一切事件 $A$，满足 $P(A) \ge 0$。
    2.  **归一化**：$P(\Omega) = 1$。
    3.  **可列可加性**：若 $A_1, A_2, \dots$ 是互不相交的事件，那么
        $$P(A_1 \cup A_2 \cup \dots) = P(A_1) + P(A_2) + \dots$$
*   **【易错点】** 公理3是**可列可加性**（针对无限个互斥事件），而非仅仅是有限可加性。由公理1和2已可推导出 $P(A) \le 1$，无需额外增加公理。并非任意样本空间的子集都可以分配概率（涉及测度论，课程中通常不作深入展开，但需建立此意识）。

##### 1.2 典型概率模型

###### （一）古典概型（离散均匀概率）
*   **【定义】古典概型**：样本空间 $\Omega$ 由 $n$ 个等可能性的试验结果组成。
*   **【解题步骤】** 对于事件 $A$：
    $$P(A) = \frac{\text{含于事件 } A \text{ 的试验结果数}}{n}$$
*   **适用条件**：样本空间有限，且每个基本事件等可能。

###### （二）几何概型（连续均匀概率）
*   **【定义】几何概型**：样本空间 $\Omega \subset \mathbb{R}^r$ 的体积 $m(\Omega)$ 为正数，且 $\Omega$ 中每个试验结果等可能。
*   **【解题步骤】** 对于事件 $A \subset \Omega$：
    $$P(A) = \frac{m(A)}{m(\Omega)}$$
    其中 $m(A)$ 表示 $A$ 的体积（一维为长度，二维为面积，三维为体积）。
*   **【例题】**（来源：Lecture01_handout.pdf P14-15）
    甲乙约定见面，延迟时间均在 $0 \sim 1$ 小时。先到者等15分钟（1/4小时）后离开。求两人能相会的概率。
    *解析*：设甲乙延迟时间分别为 $x, y \in [0,1]$。样本空间 $\Omega = [0,1] \times [0,1]$，面积 $m(\Omega)=1$。
    相会条件为 $|x-y| \le 1/4$。
    事件 $M = \{(x,y) \mid |x-y| \le 1/4, 0 \le x \le 1, 0 \le y \le 1\}$。
    计算 $M$ 的面积：$1 - 2 \times \frac{1}{2} \times \frac{3}{4} \times \frac{3}{4} = 1 - \frac{9}{16} = \frac{7}{16}$。
    故 $P(M) = \frac{7}{16}$。

###### （三）模型与现实：Bertrand 悖论
*   **【易错点】** 在连续模型中，**不同的等可能假设会导致完全不同的概率结果**。
*   **【例题】**（来源：Lecture01_handout.pdf P17-21）半径为1的圆内任取一条弦，求弦长大于等于3的概率（注：此处课件写3，实际圆内接等边三角形边长为 $\sqrt{3}$，此处遵循课件数值）。
    *   **假设1**（端点等可能）：固定一端点，另一端点等可能落圆周。$P(A) = 1/3$。
    *   **假设2**（中点在直径上等可能）：弦中点等可能落在垂直直径上。$P(A) = 1/2$。
    *   **假设3**（中点在圆内等可能）：弦中点等可能落在圆内。$P(A) = 1/4$。
*   **【记忆】** 几何概型解题核心：明确“等可能”发生的空间维度（线、面、体）。

##### 1.3 概率的性质与推导

###### （一）基本性质与推导
*   **【重点】性质1：不可能事件概率为0**
    *   **结论**：$P(\emptyset) = 0$
    *   **推导过程**：因为 $\Omega = \Omega \cup \emptyset \cup \emptyset \cup \dots$，且交集为空。由可列可加性（公理3）得：
        $$P(\Omega) = P(\Omega) + P(\emptyset) + P(\emptyset) + \dots$$
        由归一化（公理2）$P(\Omega)=1$，得 $1 = 1 + P(\emptyset) + P(\emptyset) + \dots$，即 $P(\emptyset) + P(\emptyset) + \dots = 0$。由非负性（公理1）得 $P(\emptyset) = 0$。

*   **【重点】性质2：有限可加性**
    *   **结论**：若 $A_i \cap A_j = \emptyset (i \neq j)$，则 $P(\bigcup_{i=1}^n A_i) = \sum_{i=1}^n P(A_i)$
    *   **推导过程**：令 $A_{n+1} = A_{n+2} = \dots = \emptyset$，则 $\bigcup_{i=1}^n A_i = \bigcup_{i=1}^\infty A_i$。由可列可加性与性质1：
        $$P(\bigcup_{i=1}^n A_i) = P(\bigcup_{i=1}^\infty A_i) = \sum_{i=1}^\infty P(A_i) = \sum_{i=1}^n P(A_i)$$

*   **【必背】性质3：对立事件概率**
    *   **结论**：$P(A^c) = 1 - P(A)$
    *   **推导过程**：因为 $A \cap A^c = \emptyset$ 且 $A \cup A^c = \Omega$，由有限可加性与归一化公理：
        $$1 = P(\Omega) = P(A \cup A^c) = P(A) + P(A^c)$$

*   **【必背】性质4：差事件概率与单调性**
    *   **结论**：若 $A \subset B$，则 $P(B \setminus A) = P(B) - P(A)$，且 $P(A) \le P(B)$。
    *   **推导过程**：因为 $B = A \cup (B \setminus A)$ 且 $A \cap (B \setminus A) = \emptyset$，由有限可加性：
        $$P(B) = P(A) + P(B \setminus A) \implies P(B \setminus A) = P(B) - P(A)$$
        由非负性知 $P(B \setminus A) \ge 0$，故 $P(A) \le P(B)$（概率的单调性）。
    *   **推论**：对任意事件 $A$，有 $0 \le P(A) \le 1$。

###### （二）容斥原理
*   **【高频】两事件并的概率**
    $$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$
    *   **推论（次可加性）**：$P(A \cup B) \le P(A) + P(B)$
*   **【重点】三事件并的概率（不交分解法）**
    $$P(A \cup B \cup C) = P(A) + P(A^c \cap B) + P(A^c \cap B^c \cap C)$$
    *   *解析*：将并集分解为三个互不相交的部分，然后利用有限可加性。

##### 1.4 本章总结与速查

###### （一）本章核心结论
1. 概率模型的建立依赖于样本空间的选择和概率公理的满足。
2. 古典概型算“数”，几何概型算“度量”（长/面/体）。
3. 概率的计算本质是公理及由其推导出的性质的灵活应用，特别是对立事件和容斥原理。

###### （二）必背公式汇总
1. **可列可加性**：$P(\bigcup_{i=1}^\infty A_i) = \sum_{i=1}^\infty P(A_i)$ （$A_i$ 互不相交）
2. **对立事件**：$P(A^c) = 1 - P(A)$
3. **差事件**：$P(B \setminus A) = P(B) - P(A)$ （$A \subset B$）
4. **两事件并**：$P(A \cup B) = P(A) + P(B) - P(A \cap B)$
5. **三事件并（互斥分解）**：$P(A \cup B \cup C) = P(A) + P(A^c \cap B) + P(A^c \cap B^c \cap C)$
6. **古典概型**：$P(A) = \frac{|A|}{|\Omega|}$
7. **几何概型**：$P(A) = \frac{m(A)}{m(\Omega)}$

###### （三）考前速查表

| 考点 | 核心要点 | 易错预警 |
| :--- | :--- | :--- |
| **样本空间** | 互斥、完整 | 连续多次操作是**一次**试验；选择模型要匹配关心的问题 |
| **概率公理** | 非负、归一、可列可加 | 公理3是可列可加，不是有限可加；$P(A)\le 1$ 是推论不是公理 |
| **几何概型** | 体积/面积比 | “等可能”是在线段上、面积上还是体积上？(Bertrand悖论) |
| **性质推导** | $P(\emptyset)=0$ 的证明 | 利用 $\Omega = \Omega \cup \emptyset \cup \dots$ 构造可列可加 |
| **概率计算** | 容斥原理、对立事件 | 求“至少”发生的概率，常先求对立事件 $P(A^c) = 1-P(A)$ |
# 1.3 概率的性质与推导

## （一）基本性质与推导

#### 性质4：差事件概率
【定义】如果有事件 $A, B$，且 $A \subset B$，则 $P(B \setminus A) = P(B) - P(A)$。
【重点】推导逻辑：因为 $B = A \cup (B \setminus A)$ 且 $A \cap (B \setminus A) = \emptyset$，由概率的可加性（性质2）得 $P(B) = P(A) + P(B \setminus A)$，移项即得结论。
【推论】
1. （概率的单调性）如果 $A \subset B$，则 $P(A) \le P(B)$。
2. 对任意事件 $A$，有 $0 \le P(A) \le 1$。

#### 性质5：加法公式
【必背】如果有事件 $A, B$，则：
$$P(A \cup B) = P(A) + P(B) - P(A \cap B) \quad (1-1)$$
【重点】推导逻辑：因为 $A \cup B = A \cup (B \setminus A)$，且 $A \cap (B \setminus A) = \emptyset$，由性质2和4得：
$P(A \cup B) = P(A) + P(B \setminus A) = P(A) + P(B \setminus (A \cap B)) = P(A) + P(B) - P(A \cap B)$。
【推论】（有限次可加性 / Boole's inequality）：如果有事件 $A_i, i=1, \dots, n$，则：
$$P\left(\bigcup_{i=1}^n A_i\right) \le \sum_{i=1}^n P(A_i) \quad (1-2)$$

#### 性质8：可列次可加性
【定义】如果有事件 $A_i, i=1,2, \dots$，则：
$$P\left(\bigcup_{i=1}^\infty A_i\right) \le \sum_{i=1}^\infty P(A_i) \quad (1-3)$$
【重点】推导逻辑（构造互斥事件法）：令 $B_1 = A_1, B_n = A_n \setminus \bigcup_{i=1}^{n-1} A_i$，则 $\bigcup_{i=1}^\infty A_i = \bigcup_{i=1}^\infty B_i$ 且 $B_i \cap B_j = \emptyset (i \neq j)$，同时 $B_n \subset A_n$。由概率公理（3）和单调性得：
$P(\bigcup A_i) = P(\bigcup B_i) = \sum P(B_i) \le \sum P(A_i)$。

#### 性质9：交集概率下界
【定义】如果有事件 $A_i, i=1,2, \dots$，则：
$$P\left(\bigcap_{i=1}^\infty A_i\right) \ge 1 - \sum_{i=1}^\infty P(A_i^c) \quad (1-4)$$
【推论】特别地，对于两个事件：$P(A_1 \cap A_2) \ge 1 - P(A_1^c) - P(A_2^c)$。
【重点】推导逻辑：利用德摩根定律求补，再结合性质8。
$P(\bigcap A_i) = 1 - P((\bigcap A_i)^c) = 1 - P(\bigcup A_i^c) \ge 1 - \sum P(A_i^c)$。

## （二）容斥原理

#### 性质6：容斥恒等式
【必背】如果有事件 $A_i, i=1, \dots, n$，则：
$$P\left(\bigcup_{i=1}^n A_i\right) = \sum_{1 \le i \le n} P(A_i) - \sum_{1 \le i < j \le n} P(A_i \cap A_j) + \sum_{1 \le i < j < k \le n} P(A_i \cap A_j \cap A_k) - \dots + (-1)^{n-1} P(A_1 \cap \dots \cap A_n) \quad (1-5)$$
【解题步骤】应用容斥原理计算“至少发生一个”的概率时：
1. 明确所有需要求并集的事件集合 $\{A_i\}$。
2. 分别计算单个事件概率之和。
3. 依次减去两两交集概率之和，加上三三交集概率之和，交替进行。
4. 注意符号规律：奇数个交集取正，偶数个交集取负。

#### 性质7：Bonferroni 不等式与 Kounias 不等式
【重点】Bonferroni 不等式给出了并集概率的下界：
$$P\left(\bigcup_{i=1}^n A_i\right) \ge \sum_{1 \le i \le n} P(A_i) - \sum_{1 \le i < j \le n} P(A_i \cap A_j) \quad (1-6)$$
【重点】Kounias 不等式：
$$P\left(\bigcup_{i=1}^n A_i\right) \le \min_k \left\{ \sum_{1 \le i \le n} P(A_i) - \sum_{i: i \neq k} P(A_i \cap A_k) \right\} \quad (1-7)$$
【易错点】容斥原理是等式，Bonferroni 和 Boole 不等式是不等式。在估计概率范围时，常将 Boole 不等式（上界）与 Bonferroni 不等式（下界）结合使用。
【例题】（自编）设事件 $A, B, C$ 满足 $P(A)=0.4, P(B)=0.5, P(C)=0.6, P(A \cap B)=0.2, P(A \cap C)=0.3, P(B \cap C)=0.35$，且 $P(A \cap B \cap C)=0$。
(1) 利用容斥原理计算 $P(A \cup B \cup C)$ 的精确值；
(2) 利用 Boole 不等式与 Bonferroni 不等式给出 $P(A \cup B \cup C)$ 的上下界估计。
*解答*：
(1) 由容斥原理（公式1-5）：
$P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(A \cap B) - P(A \cap C) - P(B \cap C) + P(A \cap B \cap C)$
$= 0.4 + 0.5 + 0.6 - 0.2 - 0.3 - 0.35 + 0 = 1.5 - 0.85 = 0.65$。
(2) 由 Boole 不等式（公式1-2），并集上界为：
$P(A \cup B \cup C) \le P(A) + P(B) + P(C) = 0.4 + 0.5 + 0.6 = 1.5$。
由 Bonferroni 不等式（公式1-6），并集下界为：
$P(A \cup B \cup C) \ge P(A) + P(B) + P(C) - P(A \cap B) - P(A \cap C) - P(B \cap C) = 0.4 + 0.5 + 0.6 - 0.2 - 0.3 - 0.35 = 0.65$。
可见，精确值 $0.65$ 落在 $[0.65, 1.5]$ 区间内。

# 1.4 随机抽样与随机分配

## （一）排列与组合基础
【必背】
1. $n$ 个对象的排列数：$n!$
2. $n$ 个对象中取 $k$ 个对象的排列数：$P_n^k = \frac{n!}{(n-k)!}$
3. $n$ 个对象中取 $k$ 个对象的组合数：$\binom{n}{k} = \frac{n!}{k!(n-k)!}$

## （二）四种随机抽样模型
【高频】从含有 $M$ 个不同球的箱子中抽取 $n$ 个球，样本空间大小（基本事件总数）取决于两个维度：是否放回、是否有序。

| 抽样方式 | 样本表示 | 样本空间大小 ($\# \Omega$) | 适用场景 |
| :--- | :--- | :--- | :--- |
| **放回、有序** | $(a_1, \dots, a_n)$ | $M^n$ | 骰子点数序列、有放回抽取并记录顺序 |
| **放回、无序** | $[a_1, \dots, a_n]$ | $\binom{M+n-1}{n}$ | 不记顺序的有放回抽样（多重组合） |
| **不放回、有序** | $(a_1, \dots, a_n)$ | $P_M^n = M(M-1)\dots(M-n+1)$ | 选班长副班长、排队问题 |
| **不放回、无序** | $[a_1, \dots, a_n]$ | $\binom{M}{n}$ | 买彩票、组合选人、雨课堂点名 |

【易错点】放回无序抽样的公式 $\binom{M+n-1}{n}$ 是最容易记错的，常被误认为 $\binom{M}{n}$。可通过“隔板法”进行情景证明记忆。
【记忆】抽样模型分类口诀：“放回看序，无序看回”。即先分放回/不放回，再分有序/无序。有序用圆括号，无序用方括号。

## （三）质点入箱模型（随机分配）
【重点】将 $n$ 个质点分配到 $M$ 个箱子中，与抽样模型存在一一对应关系：
- **有序抽样 $\iff$ 质点可以辨别**（两两不同）
- **无序抽样 $\iff$ 质点不可辨**（完全相同）
- **放回抽样 $\iff$ 箱中可容纳任意多质点**（无抑制箱子）
- **不放回抽样 $\iff$ 箱中最多可容纳一个质点**（有抑制箱子）

【解题步骤】遇到“分配”或“粒子状态”问题：
1. 判断粒子是否可辨（可辨 $\to$ 有序，不可辨 $\to$ 无序）。
2. 判断箱子容量限制（无限容量 $\to$ 放回，单容量 $\to$ 不放回）。
3. 对应到上表的四种抽样模型中，直接套用公式计算样本空间大小。

## （四）古典概型解题技巧
【重点】
1. **不要失去常识、也不要过分依赖**：结合实际物理意义验证结果。
2. **用最简单情形验证**：代入 $n=1$ 或 $n=2$ 检验公式正确性。
3. **将物体编号**：使不可辨对象转化为可辨对象，简化计算。
4. **情景证明**：通过构造实际情景证明代数恒等式。
【例题】（自编）利用情景证明法证明组合恒等式：$\binom{m}{n} = \binom{m-1}{n-1} + \binom{m-1}{n}$。
*解答*：
构造情景：从 $m$ 个不同的球中取出 $n$ 个球。
方法1：直接由组合数定义，取法总数为 $\binom{m}{n}$。
方法2：固定其中一个特定的球（记为球A），所有的取法可以分为两类互斥的情况：
1. 取出的 $n$ 个球中包含球A：则还需从剩下的 $m-1$ 个球中取出 $n-1$ 个，取法数为 $\binom{m-1}{n-1}$。
2. 取出的 $n$ 个球中不包含球A：则需从剩下的 $m-1$ 个球中取出 $n$ 个，取法数为 $\binom{m-1}{n}$。
由加法原理，总取法数为 $\binom{m-1}{n-1} + \binom{m-1}{n}$。
由于两种方法计算的是同一个问题的结果，故 $\binom{m}{n} = \binom{m-1}{n-1} + \binom{m-1}{n}$ 成立。

# 1.5 本章总结与速查

## （一）本章核心结论
1. **概率公理是基石**：所有概率性质均由非负性、规范性和可列可加性推导得出。
2. **求并集概率的两大工具**：容斥原理（精确等式）与 Boole/Bonferroni 不等式（估计界限）。
3. **求交集概率的方法**：通过德摩根定律转化为求并集的补集，再利用不等式估计。
4. **古典概型计数核心**：明确“放回/不放回”与“有序/无序”，准确选择四种抽样公式。

## （二）必背公式汇总
1. 加法公式：$P(A \cup B) = P(A) + P(B) - P(A \cap B)$
2. 差事件公式：$P(B \setminus A) = P(B) - P(A)$ （条件 $A \subset B$）
3. Boole 不等式（并集上界）：$P(\bigcup A_i) \le \sum P(A_i)$
4. Bonferroni 不等式（并集下界）：$P(\bigcup A_i) \ge \sum P(A_i) - \sum P(A_i \cap A_j)$
5. 交集下界：$P(\bigcap A_i) \ge 1 - \sum P(A_i^c)$
6. 容斥原理：$P(\bigcup A_i) = \sum P(A_i) - \sum P(A_i \cap A_j) + \dots$
7. 排列数：$P_n^k = \frac{n!}{(n-k)!}$
8. 组合数：$\binom{n}{k} = \frac{n!}{k!(n-k)!}$

## （三）考前速查表

| 考点 | 核心公式/结论 | 易错警示 |
| :--- | :--- | :--- |
| **加法公式** | $P(A \cup B) = P(A) + P(B) - P(A \cap B)$ | 三事件加法需加上三交集 $P(A \cap B \cap C)$ |
| **容斥原理** | 奇数个交集取正，偶数个交集取负 | 漏项或符号写反；交集指标不能重复（如 $i<j$） |
| **概率不等式** | 并集 $\le \sum P(A_i)$；交集 $\ge 1 - \sum P(A_i^c)$ | 不等式方向混淆；求交集下界时忘记取补集 |
| **放回有序** | $M^n$ | 误用排列数 |
| **放回无序** | $\binom{M+n-1}{n}$ | 误用 $\binom{M}{n}$；忘记减 1 |
| **不放回有序** | $P_M^n$ | $n > M$ 时无意义（概率为0） |
| **不放回无序** | $\binom{M}{n}$ | 忽略组合数定义中 $n \le M$ 的限制 |
| **质点入箱** | 可辨 $\leftrightarrow$ 有序；无抑制 $\leftrightarrow$ 放回 | 物理模型与数学模型对应错误 |
