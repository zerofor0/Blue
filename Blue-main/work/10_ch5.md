# 第5讲 概率分布函数、随机变量函数的分布与随机向量的分布

## 5.1 概率分布函数 (CDF)

### （一）分布函数的定义与功能
**【定义】** 对随机变量 $X$，称 $x$ 的函数 $F(x) = \mathbb{P}(X \le x), x \in \mathbb{R}$ 为 $X$ 的概率分布函数或累积分布函数，简称为分布函数 (CDF)。
**【重点】** 分布函数的核心功能是**统一**了离散型、连续型甚至混合型随机变量的描述工具。无论什么类型的随机变量，都可以用 CDF 完整刻画其信息。

**CDF 与 PMF/PDF 的转换关系：**
1. **与 PMF 的关系（离散型）**：
   若 $X$ 为离散型随机变量，概率分布列为 $p_k = \mathbb{P}(X = x_k)$，则其分布函数为：
   $$F(x) = \mathbb{P}(X \le x) = \sum_{x_j \le x} p_j \quad (5\text{-}1)$$
   **【记忆】** 离散型 CDF 是阶梯函数，在每个 $x_k$ 处有跳跃高度 $p_k$。若 $x_k$ 单增，则 $p_k = F(x_k) - F(x_{k-1})$。

2. **与 PDF 的关系（连续型）**：
   若 $X$ 为连续型随机变量，概率密度为 $f(x)$，则其分布函数为：
   $$F(x) = \mathbb{P}(X \le x) = \int_{-\infty}^{x} f(t) dt \quad (5\text{-}2)$$
   且在 $f(x)$ 的连续点 $x$ 处，有 $f(x) = F'(x)$。

**【高频】正态分布的分布函数与 $3\sigma$ 原则**
- 标准正态分布 $X \sim \mathcal{N}(0,1)$ 的 CDF 记为 $\Phi(x)$，密度记为 $\varphi(x)$。利用对称性有：
  $$\Phi(-x) = 1 - \Phi(x) \quad (5\text{-}3)$$
- 一般正态分布 $X \sim \mathcal{N}(\mu, \sigma^2)$ 的概率计算可通过标准化转化为标准正态：
  $$\mathbb{P}(X \le a) = \Phi\left(\frac{a-\mu}{\sigma}\right) \quad (5\text{-}4)$$
- **$3\sigma$ 原则**（产品质量管理常考）：
  - $\mathbb{P}(|X-\mu| \le \sigma) \approx 68.27\%$
  - $\mathbb{P}(|X-\mu| \le 2\sigma) \approx 95.45\%$
  - $\mathbb{P}(|X-\mu| \le 3\sigma) \approx 99.73\%$

### （二）分布函数的性质
**【必背】定理 1.1：分布函数 $F(x)$ 的三大性质**
1. **单调非降**：对任意 $x < y$，有 $F(x) \le F(y)$。（因为 $F(y) - F(x) = \mathbb{P}(x < X \le y) \ge 0$）
2. **边界极限**：$F(-\infty) = \lim_{x \to -\infty} F(x) = 0$，$F(\infty) = \lim_{x \to \infty} F(x) = 1$。
3. **右连续**：$F(x) = F(x+)$。

**【易错点】** 
- CDF 是**右连续**的，而非左连续！离散型 CDF 在跳跃点 $x_k$ 处的值等于该点的高度（包含该点的概率）。
- 连续型随机变量的 CDF 既是右连续也是左连续的（即处处连续）。

**【重点】定理 1.2：连续性判定**
如果对 $\forall x \in \mathbb{R}$ 都有 $\mathbb{P}(X = x) = 0$，则 $F(x) = \mathbb{P}(X \le x)$ 是连续的。
**推论**：若 $F(x)$ 不连续，则 $X$ 必非连续型随机变量。

**定理 1.3：由 CDF 求密度的条件**
设 $F(x)$ 连续，数集 $A$ 中任何两点距离大于某正数 $\delta$。如果在 $A^c$ 上导数 $F'(x)$ 存在且连续，则 $f(x) = F'(x)$ (当 $x \notin A$) 或 $0$ (当 $x \in A$) 是 $X$ 的密度函数。
**【易错点】** $F(x)$ 连续**不能**保证 $X$ 是连续型随机变量（如 Cantor 分布），但 $F(x)$ 不连续**能**保证 $X$ 不是连续型随机变量。

## 5.2 随机变量函数的分布

### （一）离散型随机变量函数的分布
**【解题步骤】** 
已知 $X$ 的分布列，求 $Y = g(X)$ 的分布列：
1. 写出 $Y$ 的所有可能取值 $y_k = g(x_k)$。
2. 若 $y_k$ 有重复（即不同的 $x$ 对应相同的 $y$），将对应概率相加。即 $\mathbb{P}(Y = y_k) = \sum_{g(x_i) = y_k} \mathbb{P}(X = x_i)$。

**【例题】** [Lecture05_handout.pdf P24] 
设 $X$ 有概率分布：$\mathbb{P}(X=-2)=0.1, \mathbb{P}(X=-1)=0.2, \mathbb{P}(X=0)=0.3, \mathbb{P}(X=1)=0.2, \mathbb{P}(X=3)=0.2$。求 $Y=X^2$ 的分布。
**解**：$Y$ 可能取值为 $0, 1, 4, 9$。
- $\mathbb{P}(Y=0) = \mathbb{P}(X=0) = 0.3$
- $\mathbb{P}(Y=1) = \mathbb{P}(X=-1) + \mathbb{P}(X=1) = 0.4$ （注意合并！）
- $\mathbb{P}(Y=4) = \mathbb{P}(X=-2) = 0.1$
- $\mathbb{P}(Y=9) = \mathbb{P}(X=3) = 0.2$

### （二）连续型随机变量函数的分布（分布函数法）
**【解题步骤】** 
已知 $X$ 的密度 $f_X(x)$，求 $Y = g(X)$ 的密度 $f_Y(y)$：
1. **求 CDF**：写出 $Y$ 的分布函数 $F_Y(y) = \mathbb{P}(Y \le y) = \mathbb{P}(g(X) \le y)$。
2. **用 $X$ 表示**：将 $\mathbb{P}(g(X) \le y)$ 转化为关于 $X$ 的概率（通常表现为对 $f_X(x)$ 在某区间积分），求出 $F_Y(y)$ 的表达式。
3. **求导得 PDF**：对 $F_Y(y)$ 求导，$f_Y(y) = \frac{dF_Y(y)}{dy}$。

**【例题】** [Lecture05_handout.pdf P26] 
设 $X \sim \mathcal{N}(\mu, \sigma^2)$，证明 $Y = \frac{X-\mu}{\sigma} \sim \mathcal{N}(0,1)$。
**解**：
$$F_Y(y) = \mathbb{P}(Y \le y) = \mathbb{P}\left(\frac{X-\mu}{\sigma} \le y\right) = \mathbb{P}(X \le \mu + \sigma y) = \int_{-\infty}^{\mu+\sigma y} \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}} dx$$
作换元 $t = \frac{x-\mu}{\sigma}$，得 $F_Y(y) = \int_{-\infty}^y \frac{1}{\sqrt{2\pi}} e^{-\frac{t^2}{2}} dt = \Phi(y)$，故 $Y \sim \mathcal{N}(0,1)$。

**【例题】** [Lecture05_handout.pdf P27] 
设 $X \sim \mathcal{N}(0,1)$，求 $Y = X^2$ 的概率密度。
**解**：
由于 $Y \ge 0$，当 $y > 0$ 时：
$$F_Y(y) = \mathbb{P}(Y \le y) = \mathbb{P}(X^2 \le y) = \mathbb{P}(-\sqrt{y} \le X \le \sqrt{y}) = \Phi(\sqrt{y}) - \Phi(-\sqrt{y}) = 2\Phi(\sqrt{y}) - 1$$
求导得：
$$f_Y(y) = F_Y'(y) = 2\varphi(\sqrt{y}) \cdot \frac{1}{2\sqrt{y}} = \frac{1}{\sqrt{2\pi y}} e^{-y/2}, \quad y > 0$$
即 $Y \sim \Gamma(\frac{1}{2}, \frac{1}{2}) = \chi^2(1)$。

## 5.3 本节总结与速查

### （一）核心结论
1. CDF 是描述任意随机变量的通用工具，统一了离散与连续。
2. CDF 必满足：单调非降、极限为 0 和 1、**右连续**。
3. 连续型 CDF 处处连续，且 $\mathbb{P}(X=x)=0$；离散型 CDF 为阶梯函数。
4. 求随机变量函数的分布，连续型用“分布函数法”（先求 CDF 再求导），离散型用“列表合并法”。

### （二）必背公式汇总
1. CDF 定义：$F(x) = \mathbb{P}(X \le x)$
2. 离散转 CDF：$F(x) = \sum_{x_j \le x} p_j$
3. 连续转 CDF：$F(x) = \int_{-\infty}^x f(t)dt$
4. 连续求密度：$f(x) = F'(x)$（在连续点）
5. 正态标准化：$\mathbb{P}(X \le a) = \Phi\left(\frac{a-\mu}{\sigma}\right)$
6. 标准正态对称性：$\Phi(-x) = 1 - \Phi(x)$

### （三）考前速查表

| 考点 | 核心公式/结论 | 易错提醒 |
| :--- | :--- | :--- |
| **CDF 性质** | 单调非降；$F(-\infty)=0, F(\infty)=1$；**右连续** | 离散型在跳跃点取右极限；连续型左右均连续 |
| **正态分布** | $\Phi(-x) = 1 - \Phi(x)$ | $3\sigma$ 原则对应概率 0.6827, 0.9545, 0.9973 |
| **离散函数分布** | $Y=g(X)$，相同 $Y$ 值对应概率相加 | 漏掉相同取值的概率合并 |
| **连续函数分布** | $F_Y(y) = \mathbb{P}(g(X) \le y)$，再求导 | 积分上下限反解错误；忘记对 $F_Y(y)$ 求导 |
# 5.2 连续型随机变量函数的分布（续）

## （一）Gamma 分布
【定义】设 $\alpha, \lambda$ 是正常数，如果 $X$ 的密度函数为：
$$f(x) = \frac{\lambda^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\lambda x}, \quad x \ge 0 \quad (5-1)$$
则称 $X$ 服从参数为 $(\alpha, \lambda)$ 的 Gamma 分布，记作 $X \sim \Gamma(\alpha, \lambda)$。
其中 $\Gamma(\alpha)$ 为 Gamma 函数：
$$\Gamma(\alpha) = \int_0^\infty x^{\alpha-1} e^{-x} dx \quad (5-2)$$

【必背】Gamma 函数的性质：
- (a) $\Gamma(n) = (n-1)!$
- (b) $\Gamma(\alpha+1) = \alpha\Gamma(\alpha)$
- (c) $\Gamma(\frac{1}{2}) = \sqrt{\pi}$

【重点】参数意义与特例：
- $\alpha$ 为形状参数，$\lambda$ 为尺度参数。
- 当 $\alpha = 1$ 时，退化为指数分布。
- 当 $\alpha$ 为正整数时，称为 Erlang 分布（排队论常用）。

## （二）连续型随机变量函数的求法

### 1. 分布函数法（CDF法）
【解题步骤】适用于任何函数 $Y = g(X)$。
1. 确定 $Y$ 的取值范围。
2. 求分布函数：$F_Y(y) = \mathbb{P}(Y \le y) = \mathbb{P}(g(X) \le y) = \int_{g(x) \le y} f_X(x) dx$。
3. 求导得密度函数：$f_Y(y) = F'_Y(y)$。

【真题】[Lecture05_handout.pdf P27] 例2.3：设 $X \sim \mathcal{N}(0,1)$，求 $Y = X^2$ 的概率密度。
解：由于 $Y \ge 0$，对于 $y > 0$，
$F_Y(y) = \mathbb{P}(Y \le y) = \mathbb{P}(-\sqrt{y} \le X \le \sqrt{y}) = \Phi(\sqrt{y}) - \Phi(-\sqrt{y}) = 2\Phi(\sqrt{y}) - 1$
求导得：$f_Y(y) = F'_Y(y) = 2 \varphi(\sqrt{y}) \cdot \frac{1}{2\sqrt{y}} = \frac{1}{\sqrt{2\pi y}} e^{-y/2}, \quad y > 0$
即 $Y \sim \Gamma(\frac{1}{2}, \frac{1}{2})$（也称为自由度为1的卡方分布 $\chi^2_1$）。

### 2. 公式法（单调函数情形）
【重点】若 $Y = aX + b$，则可直接由 $f_X(x)$ 得到 $f_Y(y)$：
$$f_Y(y) = \frac{1}{|a|} f_X\left(\frac{y-b}{a}\right) \quad (5-3)$$
【记忆】正态分布的线性变换仍是正态分布。

【高频】若 $Y = g(X)$ 且 $g(x)$ 为严格单调函数，$h(y)$ 为 $g(x)$ 的逆映射，则：
$$f_Y(y) = f_X(h(y)) |h'(y)| \quad (5-4)$$

### 3. 一般定理（非单调函数情形）
【重点】定理 2.1：设 $X$ 的概率密度为 $f(x)$，$Y = g(X)$。如果 $Y=y$ 可分解为 $n$ 个互不相交的事件 $\{X = h_i(y)\}$，且每个 $h_i(y)$ 在其值域内可导，则 $Y$ 的密度函数为：
$$f_Y(y) = \sum_{i=1}^n f(h_i(y)) |h'_i(y)|, \quad y \in D \quad (5-5)$$
（其中 $D$ 为 $Y$ 的取值范围，在 $D$ 外 $f_Y(y) = 0$）

【解题步骤】对于非单调函数（如 $Y = \cos X$）：
1. 确定 $Y$ 的值域 $D$。
2. 在 $D$ 内给定 $y$，求解方程 $g(x) = y$，得到所有解 $x_i = h_i(y)$。
3. 验证这些解所在的区间互不相交，且 $h_i(y)$ 可导。
4. 代入公式 (5-5) 求和。

【真题】[Lecture05_handout.pdf P36-38] 例2.4：设 $r$ 是正常数，$X \sim \mathcal{U}(0, 2\pi)$，求 $Y = r\cos X$ 的概率密度。
解：$Y \in (-r, r)$。对于 $y \in (-r, r)$，由 $r\cos X = y$ 解得两个解：
$h_1(y) = \arccos(\frac{y}{r}) \in (0, \pi)$
$h_2(y) = 2\pi - \arccos(\frac{y}{r}) \in (\pi, 2\pi)$
由于 $f_X(x) = \frac{1}{2\pi}$，代入公式：
$f_Y(y) = \frac{1}{2\pi} |h'_1(y)| + \frac{1}{2\pi} |h'_2(y)| = \frac{1}{2\pi} \frac{1}{\sqrt{r^2-y^2}} + \frac{1}{2\pi} \frac{1}{\sqrt{r^2-y^2}} = \frac{1}{\pi\sqrt{r^2-y^2}}$

【易错点】
- 使用公式法时，**必须取绝对值** $|h'(y)|$。
- 非单调函数求解时，**容易遗漏解**（如 $\cos X = y$ 在一个周期内有两个解），必须找全所有满足条件的逆映射。

# 5.3 随机向量及其分布

## （一）随机向量的联合分布
【定义】如果 $X_1, \dots, X_n$ 都是概率空间 $(\Omega, \mathcal{F}, \mathbb{P})$ 上的随机变量，称 $\mathbf{X} = (X_1, \dots, X_n)$ 为 $n$ 维随机向量。

【定义】设 $\mathbf{X} = (X_1, \dots, X_n)$ 是随机向量，称 $\mathbb{R}^n$ 上的 $n$ 元函数：
$$F(x_1, x_2, \dots, x_n) = \mathbb{P}(X_1 \le x_1, X_2 \le x_2, \dots, X_n \le x_n) \quad (5-6)$$
为 $\mathbf{X}$ 的联合概率分布函数（简称联合分布）。

【性质】联合分布函数 $F(x_1, \dots, x_n)$ 是 $\mathbb{R}^n$ 上的右连续函数，且关于每个自变元 $x_j$ 单调非降。

## （二）离散型随机向量
【定义】如果 $X_1, \dots, X_n$ 都是离散型随机变量，则称 $\mathbf{X}$ 为离散型随机向量。其所有可能取值的概率称为联合分布列：
$$p_{j_1, j_2, \dots, j_n} = \mathbb{P}(\mathbf{X} = \mathbf{x}_{j_1, j_2, \dots, j_n})$$
性质：$p_{j_1, \dots, j_n} \ge 0$ 且 $\sum p_{j_1, \dots, j_n} = 1$。

【重点】二维离散型随机向量通常用表格表示。表格的行和为 $X$ 的边缘分布列 $p_i$，列和为 $Y$ 的边缘分布列 $q_j$。

## （三）随机向量CDF的功能与边缘分布
【解题步骤】利用联合 CDF 计算矩形区域概率：
设 $F(x, y)$ 是 $(X, Y)$ 的联合分布，对矩形 $D = \{a < x \le b, c < y \le d\}$，有：
$$\mathbb{P}(a < X \le b, c < Y \le d) = F(b, d) - F(b, c) - F(a, d) + F(a, c) \quad (5-7)$$
【记忆】“右上角减去同行的右下角，减去同列的左上角，加上左下角”（容斥原理）。

【定义】边缘分布函数：对于 $n$ 维随机向量，任意 $k$ 个分量构成的向量的联合分布称为原分布的 $k$-维边缘分布。
二维情形下：
$$F_X(x) = \mathbb{P}(X \le x, Y \le \infty) = F(x, \infty) \quad (5-8)$$
$$F_Y(y) = \mathbb{P}(X \le \infty, Y \le y) = F(\infty, y)$$

【易错点】
- **边缘分布由联合分布唯一决定**，但**联合分布不能由边缘分布唯一决定**（除非独立）。
- 计算矩形概率时，边界开闭区间影响结果，公式 (5-7) 严格对应 $a < x \le b, c < y \le d$。

## （四）随机向量的独立性
【定义】设 $X_1, \dots, X_n$ 是随机变量，若对任意实数 $x_1, \dots, x_n$，都有：
$$\mathbb{P}(X_1 \le x_1, \dots, X_n \le x_n) = \prod_{i=1}^n \mathbb{P}(X_i \le x_i) \quad (5-9)$$
则称 $X_1, \dots, X_n$ 相互独立。

【重点】等价表达：联合分布函数等于各边缘分布函数的乘积，即：
$$F(x_1, x_2, \dots, x_n) = F_1(x_1) F_2(x_2) \dots F_n(x_n)$$

# 5.4 本节总结与速查

## （一）核心结论
1. 求连续型随机变量函数的分布，最稳妥的方法是**分布函数法（CDF法）**；若函数严格单调或分段单调，可使用**公式法**。
2. Gamma 分布是指数分布的推广，也是正态分布变量平方的分布（卡方分布）的基础。
3. 联合分布完全决定了边缘分布，但反之不然；独立性是连接两者的桥梁（独立时联合等于边缘乘积）。
4. 二维矩形区域概率计算遵循容斥原理：$F(b,d) - F(a,d) - F(b,c) + F(a,c)$。

## （二）必背公式汇总
| 公式名称 | 表达式 | 适用条件 |
| :--- | :--- | :--- |
| Gamma 函数递推 | $\Gamma(\alpha+1) = \alpha\Gamma(\alpha)$ | $\alpha > 0$ |
| 线性变换密度 | $f_Y(y) = \frac{1}{\|a\|} f_X\left(\frac{y-b}{a}\right)$ | $Y = aX+b, a \ne 0$ |
| 单调变换密度 | $f_Y(y) = f_X(h(y)) \|h'(y)\|$ | $Y=g(X)$ 严格单调, $h=g^{-1}$ |
| 一般变换密度 | $f_Y(y) = \sum f(h_i(y)) \|h'_i(y)\|$ | $Y=g(X)$ 分段单调 |
| 矩形区域概率 | $P(a < X \le b, c < Y \le d) = F(b,d) - F(b,c) - F(a,d) + F(a,c)$ | 二维随机向量 |
| 独立性判定 | $F(x,y) = F_X(x)F_Y(y)$ | $X, Y$ 相互独立 |

## （三）考前速查表
| 考点 | 考法提示 | 易错警告 |
| :--- | :--- | :--- |
| **CDF法求密度** | 给定 $X \sim f(x)$，求 $Y=g(X)$ 的密度。先求 $F_Y(y)$ 再求导。 | 积分上下限容易弄反；求导时漏掉链式法则的系数。 |
| **公式法求密度** | $Y = g(X)$，直接套用逆函数公式。 | **绝对值！绝对值！绝对值！** $h'(y)$ 必须带绝对值。 |
| **非单调函数** | 如 $Y=X^2$ 或 $Y=\cos X$，需分段求逆函数后相加。 | 遗漏某些区间内的逆解，导致概率不完整。 |
| **联合与边缘** | 给定联合分布列/函数，求边缘分布。 | 误以为边缘能决定联合（除非已知独立）。 |
| **矩形概率** | 给定联合 CDF，求二维矩形区域概率。 | 加减号记错，应为“右上 - 右下 - 左上 + 左下”。 |
#### 5.3 随机向量及其分布（续）

##### （一）随机向量的独立性

###### 1. 独立性定义回顾
【定义】随机变量的相互独立性
设 $X_1, \dots, X_n$ 是 $\Omega, \mathcal{F}$ 上的随机变量，如果对任意的实数 $x_1, \dots, x_n$ 都有
$$ \mathbb{P}(X_1 \le x_1, \dots, X_n \le x_n) = \mathbb{P}(X_1 \le x_1) \cdots \mathbb{P}(X_n \le x_n) $$
则称随机变量 $X_1, \dots, X_n$ 相互独立。

【重点】等价表达
设 $(X_1, X_2, \dots, X_n)$ 有联合分布 $F(x_1, x_2, \dots, x_n)$，$X_i$ 有边缘分布 $F_i(x_i)$。则 $X_1, \dots, X_n$ 相互独立当且仅当：
$$ F(x_1, x_2, \dots, x_n) = F_1(x_1) F_2(x_2) \dots F_n(x_n) $$
【记忆】独立性本质：联合分布等于边缘分布的乘积（无论是CDF、PMF还是PDF）。

###### 2. 离散型随机向量的独立性
【定理 3.1】
设离散型随机向量 $\mathbf{X} = (X_1, X_2, \dots, X_n)$ 有概率分布 $p_{j_1, j_2, \dots, j_n} = \mathbb{P}(\mathbf{X} = \mathbf{x}_{j_1, j_2, \dots, j_n})$，则 $X_1, \dots, X_n$ 相互独立的充分必要条件是：对任意的 $x_1^{j_1}, x_2^{j_2}, \dots, x_n^{j_n}$，有
$$ \mathbb{P}(X_1 = x_1^{j_1}, \dots, X_n = x_n^{j_n}) = \prod_{i=1}^n \mathbb{P}(X_i = x_i^{j_i}) $$

【推论 3.1】
设离散型随机向量 $(X, Y)$ 的所有不同取值是 $x_i, y_j, i, j \ge 1$。则 $X, Y$ 相互独立的充分必要条件是：对任意的 $(x_i, y_j)$，有
$$ \mathbb{P}(X = x_i, Y = y_j) = \mathbb{P}(X = x_i)\mathbb{P}(Y = y_j) $$

【易错点】独立性要求对所有取值点均成立联合概率等于边缘概率乘积。若存在某一对 $(x_i, y_j)$ 使得等式不成立，则 $X, Y$ 不独立。
【解题步骤】证明离散型随机变量独立性：写出联合分布律表与边缘分布律表，逐一检验 $p_{ij} = p_{i\cdot} p_{\cdot j}$ 是否对所有 $i, j$ 成立。

##### （二）经典离散型随机向量例

###### 1. 多项分布
【定义】多项分布
设 $A_1, A_2, \dots, A_r$ 是试验 $S$ 的完备事件组。对试验 $S$ 进行 $n$ 次独立重复试验时，用 $X_i$ 表示 $A_i$ 发生的次数，则 $(X_1, \dots, X_r)$ 的概率分布为：
$$ \mathbb{P}(X_1 = k_1, \dots, X_r = k_r) = \frac{n!}{k_1! \dots k_r!} p_1^{k_1} \dots p_r^{k_r} $$
其中 $k_i \ge 0, p_i = \mathbb{P}(A_i), \sum_{i=1}^r k_i = n$。

【重点】多项分布的边缘分布
对任意 $1 \le m < n$，$(X_{j_1}, X_{j_2}, \dots, X_{j_m})$ 的边缘分布仍为多项分布：
$$ \mathbb{P}(X_{j_1} = k_1, \dots, X_{j_m} = k_m) = \frac{n!}{k_1! \dots k_m! k!} p_{j_1}^{k_1} \dots p_{j_m}^{k_m} q^k $$
其中 $k = n - \sum_{i=1}^m k_i \ge 0, q = 1 - \sum_{i=1}^m p_{j_i}$。
特别地，单个变量 $X_j$ 的边缘分布为二项分布 $B(n, p_j)$：
$$ \mathbb{P}(X_j = k) = \binom{n}{k} p_j^k (1-p_j)^{n-k}, \quad k \ge 0 $$
【记忆】多项分布是二项分布在多类结果（完备事件组）下的推广；其边缘分布要么是二项分布，要么是更低维的多项分布（把未选中的事件合并为一类）。

##### （三）本章总结与速查

###### 1. 核心结论
- **分布函数（CDF）的统一性**：CDF统一描述了离散型、连续型及混合型随机变量的分布，是连接PMF和PDF的桥梁。
- **随机变量函数的分布求解**：
  - **曲线救国法（CDF法）**：先求 $Y = g(X)$ 的 CDF $F_Y(y) = \mathbb{P}(g(X) \le y)$，再通过求导（连续型）或差分（离散型）得到 PMF/PDF。
  - **直接求解法**：根据 $X$ 的 PMF/PDF 直接推导 $g(X)$ 的分布。
- **随机向量的核心知识**：联合分布、边缘分布、独立性的判定是核心。离散型经典例子为多项分布，连续型为多元正态分布。

###### 2. 必背公式汇总
| 名称 | 公式 | 适用条件 |
| :--- | :--- | :--- |
| 随机变量独立性(CDF) | $F(x_1, \dots, x_n) = \prod_{i=1}^n F_i(x_i)$ | 任意随机变量 |
| 离散型独立性(PMF) | $\mathbb{P}(X_1=x_1, \dots, X_n=x_n) = \prod_{i=1}^n \mathbb{P}(X_i=x_i)$ | 离散型随机向量 |
| 多项分布PMF | $\mathbb{P}(X_1=k_1, \dots, X_r=k_r) = \frac{n!}{\prod k_i!} \prod p_i^{k_i}$ | 完备事件组 $n$ 次独立试验 |
| 多项分布边缘分布 | $X_j \sim B(n, p_j)$ | 多项分布的单个分量 |

###### 3. 考前速查表
| 考点 | 考察方式 | 易错点 |
| :--- | :--- | :--- |
| 随机变量独立性 | 判断给定联合分布的变量是否独立；由独立性求未知参数 | 必须对所有取值检验，不能只看部分点 |
| 多项分布计算 | 计算多项分布下特定事件组合的概率；求边缘分布 | 忽略完备事件组条件 $\sum k_i = n$；边缘分布误认为还是同维多项分布 |
| 函数分布求解 | 给定 $X$ 分布，求 $Y=g(X)$ 的 PMF/PDF | CDF法求导时未注意 $g(x)$ 的单调性导致分段错误；忽略归一化常数 |

##### （四）附录（选修拓展）

###### 1. 奇异分布
【定义】奇异函数
一个函数 $F$ 被称为奇异的，当且仅当它连续、不恒为零、导数 $F'$ 几乎处处 存在，且 $F'(t) = 0$ a.e.

###### 2. 康托尔集上的均匀分布
- **康托尔集构造**：从 $[0, 1]$ 中挖去中间的三分之一 $(1/3, 2/3)$，然后对剩余区间不断重复挖去中间三分之一的过程。
- **康托尔分布函数**：在挖去的区间上，函数值取为该区间两端点函数值的平均（如 $x \in (1/3, 2/3)$ 时 $F(x) = 1/2$）。
- **性质**：该函数 $F$ 非降、连续，且 $F(-\infty)=0, F(\infty)=1$，是一个合法的 CDF。除了康托尔集上的点外，其导数 $F'(x) = 0$。这被称为“勒贝格奇异函数”。
