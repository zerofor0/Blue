# 第5讲 概率分布函数、随机变量函数的分布与随机向量的分布

## 5.1 概率分布函数 (CDF)

### （一）分布函数的定义与功能
**【定义】** 对随机变量 $X$，称 $x$ 的函数 $F(x) = \mathbb{P}(X \le x), x \in \mathbb{R}$ 为 $X$ 的概率分布函数或累积分布函数，简称为分布函数 (CDF)。
**【重点】** 分布函数的核心功能是**统一**了离散型、连续型甚至混合型随机变量的描述工具。无论什么类型的随机变量，都可以用 CDF 完整刻画其信息。

**CDF 与 PMF/PDF 的转换关系：**
1. **与 PMF 的关系（离散型）**：
   若 $X$ 为离散型随机变量，概率分布列为 $p_k = \mathbb{P}(X = x_k)$，则其分布函数为：
   $$F(x) = \mathbb{P}(X \le x) = \sum_{x_j \le x} p_j \quad (5\text{-}1)$$
   **【记忆】** 离散型 CDF 是阶梯函数，在每个 $x_k$ 处有跳跃高度 $p_k$。若 $x_k$ 单增，则 $p_k = F(x_k) - F(x_{k-1})$。

2. **与 PDF 的关系（连续型）**：
   若 $X$ 为连续型随机变量，概率密度为 $f(x)$，则其分布函数为：
   $$F(x) = \mathbb{P}(X \le x) = \int_{-\infty}^{x} f(t) dt \quad (5\text{-}2)$$
   且在 $f(x)$ 的连续点 $x$ 处，有 $f(x) = F'(x)$。

**【高频】正态分布的分布函数与 $3\sigma$ 原则**
- 标准正态分布 $X \sim \mathcal{N}(0,1)$ 的 CDF 记为 $\Phi(x)$，密度记为 $\varphi(x)$。利用对称性有：
  $$\Phi(-x) = 1 - \Phi(x) \quad (5\text{-}3)$$
- 一般正态分布 $X \sim \mathcal{N}(\mu, \sigma^2)$ 的概率计算可通过标准化转化为标准正态：
  $$\mathbb{P}(X \le a) = \Phi\left(\frac{a-\mu}{\sigma}\right) \quad (5\text{-}4)$$
- **$3\sigma$ 原则**（产品质量管理常考）：
  - $\mathbb{P}(|X-\mu| \le \sigma) \approx 68.27\%$
  - $\mathbb{P}(|X-\mu| \le 2\sigma) \approx 95.45\%$
  - $\mathbb{P}(|X-\mu| \le 3\sigma) \approx 99.73\%$

### （二）分布函数的性质
**【必背】定理 1.1：分布函数 $F(x)$ 的三大性质**
1. **单调非降**：对任意 $x < y$，有 $F(x) \le F(y)$。（因为 $F(y) - F(x) = \mathbb{P}(x < X \le y) \ge 0$）
2. **边界极限**：$F(-\infty) = \lim_{x \to -\infty} F(x) = 0$，$F(\infty) = \lim_{x \to \infty} F(x) = 1$。
3. **右连续**：$F(x) = F(x+)$。

**【易错点】** 
- CDF 是**右连续**的，而非左连续！离散型 CDF 在跳跃点 $x_k$ 处的值等于该点的高度（包含该点的概率）。
- 连续型随机变量的 CDF 既是右连续也是左连续的（即处处连续）。

**【重点】定理 1.2：连续性判定**
如果对 $\forall x \in \mathbb{R}$ 都有 $\mathbb{P}(X = x) = 0$，则 $F(x) = \mathbb{P}(X \le x)$ 是连续的。
**推论**：若 $F(x)$ 不连续，则 $X$ 必非连续型随机变量。

**定理 1.3：由 CDF 求密度的条件**
设 $F(x)$ 连续，数集 $A$ 中任何两点距离大于某正数 $\delta$。如果在 $A^c$ 上导数 $F'(x)$ 存在且连续，则 $f(x) = F'(x)$ (当 $x \notin A$) 或 $0$ (当 $x \in A$) 是 $X$ 的密度函数。
**【易错点】** $F(x)$ 连续**不能**保证 $X$ 是连续型随机变量（如 Cantor 分布），但 $F(x)$ 不连续**能**保证 $X$ 不是连续型随机变量。

## 5.2 随机变量函数的分布

### （一）离散型随机变量函数的分布
**【解题步骤】** 
已知 $X$ 的分布列，求 $Y = g(X)$ 的分布列：
1. 写出 $Y$ 的所有可能取值 $y_k = g(x_k)$。
2. 若 $y_k$ 有重复（即不同的 $x$ 对应相同的 $y$），将对应概率相加。即 $\mathbb{P}(Y = y_k) = \sum_{g(x_i) = y_k} \mathbb{P}(X = x_i)$。

**【例题】** [Lecture05_handout.pdf P24] 
设 $X$ 有概率分布：$\mathbb{P}(X=-2)=0.1, \mathbb{P}(X=-1)=0.2, \mathbb{P}(X=0)=0.3, \mathbb{P}(X=1)=0.2, \mathbb{P}(X=3)=0.2$。求 $Y=X^2$ 的分布。
**解**：$Y$ 可能取值为 $0, 1, 4, 9$。
- $\mathbb{P}(Y=0) = \mathbb{P}(X=0) = 0.3$
- $\mathbb{P}(Y=1) = \mathbb{P}(X=-1) + \mathbb{P}(X=1) = 0.4$ （注意合并！）
- $\mathbb{P}(Y=4) = \mathbb{P}(X=-2) = 0.1$
- $\mathbb{P}(Y=9) = \mathbb{P}(X=3) = 0.2$

### （二）连续型随机变量函数的分布（分布函数法）
**【解题步骤】** 
已知 $X$ 的密度 $f_X(x)$，求 $Y = g(X)$ 的密度 $f_Y(y)$：
1. **求 CDF**：写出 $Y$ 的分布函数 $F_Y(y) = \mathbb{P}(Y \le y) = \mathbb{P}(g(X) \le y)$。
2. **用 $X$ 表示**：将 $\mathbb{P}(g(X) \le y)$ 转化为关于 $X$ 的概率（通常表现为对 $f_X(x)$ 在某区间积分），求出 $F_Y(y)$ 的表达式。
3. **求导得 PDF**：对 $F_Y(y)$ 求导，$f_Y(y) = \frac{dF_Y(y)}{dy}$。

**【例题】** [Lecture05_handout.pdf P26] 
设 $X \sim \mathcal{N}(\mu, \sigma^2)$，证明 $Y = \frac{X-\mu}{\sigma} \sim \mathcal{N}(0,1)$。
**解**：
$$F_Y(y) = \mathbb{P}(Y \le y) = \mathbb{P}\left(\frac{X-\mu}{\sigma} \le y\right) = \mathbb{P}(X \le \mu + \sigma y) = \int_{-\infty}^{\mu+\sigma y} \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}} dx$$
作换元 $t = \frac{x-\mu}{\sigma}$，得 $F_Y(y) = \int_{-\infty}^y \frac{1}{\sqrt{2\pi}} e^{-\frac{t^2}{2}} dt = \Phi(y)$，故 $Y \sim \mathcal{N}(0,1)$。

**【例题】** [Lecture05_handout.pdf P27] 
设 $X \sim \mathcal{N}(0,1)$，求 $Y = X^2$ 的概率密度。
**解**：
由于 $Y \ge 0$，当 $y > 0$ 时：
$$F_Y(y) = \mathbb{P}(Y \le y) = \mathbb{P}(X^2 \le y) = \mathbb{P}(-\sqrt{y} \le X \le \sqrt{y}) = \Phi(\sqrt{y}) - \Phi(-\sqrt{y}) = 2\Phi(\sqrt{y}) - 1$$
求导得：
$$f_Y(y) = F_Y'(y) = 2\varphi(\sqrt{y}) \cdot \frac{1}{2\sqrt{y}} = \frac{1}{\sqrt{2\pi y}} e^{-y/2}, \quad y > 0$$
即 $Y \sim \Gamma(\frac{1}{2}, \frac{1}{2}) = \chi^2(1)$。

## 5.3 本节总结与速查

### （一）核心结论
1. CDF 是描述任意随机变量的通用工具，统一了离散与连续。
2. CDF 必满足：单调非降、极限为 0 和 1、**右连续**。
3. 连续型 CDF 处处连续，且 $\mathbb{P}(X=x)=0$；离散型 CDF 为阶梯函数。
4. 求随机变量函数的分布，连续型用“分布函数法”（先求 CDF 再求导），离散型用“列表合并法”。

### （二）必背公式汇总
1. CDF 定义：$F(x) = \mathbb{P}(X \le x)$
2. 离散转 CDF：$F(x) = \sum_{x_j \le x} p_j$
3. 连续转 CDF：$F(x) = \int_{-\infty}^x f(t)dt$
4. 连续求密度：$f(x) = F'(x)$（在连续点）
5. 正态标准化：$\mathbb{P}(X \le a) = \Phi\left(\frac{a-\mu}{\sigma}\right)$
6. 标准正态对称性：$\Phi(-x) = 1 - \Phi(x)$

### （三）考前速查表

| 考点 | 核心公式/结论 | 易错提醒 |
| :--- | :--- | :--- |
| **CDF 性质** | 单调非降；$F(-\infty)=0, F(\infty)=1$；**右连续** | 离散型在跳跃点取右极限；连续型左右均连续 |
| **正态分布** | $\Phi(-x) = 1 - \Phi(x)$ | $3\sigma$ 原则对应概率 0.6827, 0.9545, 0.9973 |
| **离散函数分布** | $Y=g(X)$，相同 $Y$ 值对应概率相加 | 漏掉相同取值的概率合并 |
| **连续函数分布** | $F_Y(y) = \mathbb{P}(g(X) \le y)$，再求导 | 积分上下限反解错误；忘记对 $F_Y(y)$ 求导 |
