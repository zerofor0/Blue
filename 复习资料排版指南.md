# 复习资料排版指南（基于 5 份模板分析）

分析样本：oop_exam_reference_examples.pdf、OOP期末复习资料.pdf、微积分A_1__期中复习_黄骏斌.pdf、微积分A_1_讲义_黄骏斌 (1).pdf、概率论讲义（终版）.pdf。

---

## 一、各模板格式特征

### 1. oop_exam_reference_examples.pdf
- 定位与来源：考场黑白速查资料，Markdown 导出 PDF（Noto Serif/Sans CJK 字体族），37 页，A4 纵向。
- 字体：正文 Noto Serif CJK 10pt；标题 Noto Sans CJK；代码与注释 8.4-8.5pt；标题字号 12/14.3/17.2pt。
- 版式：页眉"OOP 期末纸质资料"，页码居中；目录带点引导线；数字三级编号（1 / 1.1 / 1.1.1）。
- 特殊格式：大量代码块、对比表格（如编译/链接阶段表）、"易错/注意/考点"提示行、"考场做法"引导句。
- 知识点呈现：先按题型与高频点排序，再给最小错例库，最后按知识点配例题（例题化）。

### 2. OOP期末复习资料.pdf
- 定位与来源：系统复习讲义，LaTeX 排版（Latin Modern + ctex），33 页，A4 纵向。
- 字体：西文/正文 Latin Modern Roman 10.9pt；中文宋体 SimSun、标题黑体 SimHei、仿宋 FangSong；代码 Consolas。
- 版式：封面（课程名、作者、学期、覆盖范围："涵盖14讲课件·OJ 6次作业·3套往年题"）→ 目录 → 正文；标题层级 12/17.2/24.8pt。
- 特殊格式：代码块带行号、节中"注意"提示、节末"模块 N 知识关系"衔接段、编号列表。
- 知识点呈现：按章系统整理，每个知识点配代码与注意事项，章末给出知识关联与考点提示。

### 3. 微积分A_1__期中复习_黄骏斌.pdf
- 定位与来源：期中复习讲义，LaTeX + 大量数学公式（Computer Modern 数学字体 + Fandol 宋体），33 页，A4 纵向，正文 12pt。
- 字体：中文 FandolSong；数学 CMR/CMMI/CMSY。
- 版式：页眉（讲义名、作者、日期）；标题 14.3/17.2/20.7pt。
- 特殊格式：首页"方法总览"清单；统一解题模板，分 题目 / 思路(Idea) / 解答(Solution) / 技巧(Tips) / 模式(Patterns) 五段；展示公式独占行、多行对齐。
- 知识点呈现：先给方法总览，每个考点用"题→思路→解→技巧"模板走一道例题。

### 4. 微积分A_1_讲义_黄骏斌 (1).pdf
- 定位与来源：完整课程讲义，与期中复习同源（LaTeX + 数学，同作者），第 N 讲结构，N.N 编号。
- 特征：体例规整，覆盖全课程，数学排版与期中复习一致。

### 5. 概率论讲义（终版）.pdf
- 定位与来源：期末复习讲义，LaTeX + 数学，207 页大长篇，A4 纵向，正文 12pt。
- 字体：Latin Modern + SimSun/SimHei + Computer Modern 数学字体 + MSBM（黑板粗体）。
- 版式：封面 → "复习方式"说明 → 按 Lecture 分章；标题 15/18/22pt。
- 特殊格式：每讲固定三段——知识整理 + 作业题解答 + 额外例题；展示公式密集；概率、期望、分布符号规范统一。
- 知识点呈现：讲次对齐课件，三段式结构（整理/作业/例题）贯穿全篇。

## 二、复习资料排版指南（共性提炼）

页面与版心：
- A4 纵向（210×297mm），四周留白；正文 10-12pt，行距 1.2-1.4 倍。
- 页眉放资料名/作者/日期，页码居中或居右。

字体（四件套）：
- 中文正文：宋体（SimSun / FandolSong / Noto Serif CJK）。
- 中文标题：黑体（SimHei / Noto Sans CJK）。
- 西文与数学：衬线体（Latin Modern / Computer Modern），数学符号用专门字体。
- 代码：等宽体（Consolas / Courier）。

标题与编号：
- 数字三级编号：章（1）/ 节（1.1）/ 小节（1.1.1），层级用字号区分（如 20pt / 15pt / 12pt）。
- 必备目录：章/节标题 + 页码 + 点引导线；长资料按讲次（Lecture / 第 N 讲）分章。

结构化元素：
- 代码块：等宽字体、带行号、必要时语法高亮；代码与中文说明分行。
- 表格：用于对比（阶段、语法差异、概念辨析），表头加粗。
- 提示框：用"注意 / 易错 / 考点 / 考场做法"等引导词标注重点与陷阱。
- 展示公式：独占一行、居中、编号；行内公式随文。
- 列表：要点用编号列表或项目符号。

内容组织：
- 开头给"总览/框架/高频点排序"，让读者快速建立全局认识。
- 每个知识点：定义 → 原理/方法 → 代码或公式 → 易错提示 → 例题。
- 章末给"知识关系/小结"，串联前后内容。
- 例题结构化（题面 → 思路 → 解答 → 技巧），理工科尤其重要。

## 三、不同学科的排版特殊要求

理工科（数学、物理、概率统计）：
- 正文略大（12pt），为公式留空间。
- 展示公式独占行、统一编号；多步推导用对齐排版。
- 采用结构化解题模板：题目 / 思路 / 解答 / 技巧 / 适用条件。
- 定理、公式必须标注前提条件与适用域；符号体例前后一致。
- 重视方法总览页（一类问题对应一套方法）。

计算机类（程序设计、数据结构、算法）：
- 代码块为第一公民：等宽字体、行号、语法高亮、关键行加注释。
- 用对比表格区分易混淆语法（声明/定义、编译期/运行期、值/引用）。
- 用"易错/考点/考场做法"提示框强调陷阱与高频题。
- 代码与中文说明分行，避免行内混排降低可读性。
- 复杂度、流程用列表或伪代码呈现。

经管类（经济学、金融、管理）：
- 用模型对比表（前提、结论、适用场景）。
- 因果链与传导机制用箭头链或编号步骤。
- 图形解释配文字说明；公式相对较少但需标注变量含义。

文史哲社科类：
- 以时间线、人物与著作、核心观点为主线。
- 简答题给答题框架，论述题给模板与得分点。
- 避免大段原文摘抄，用要点与逻辑关系呈现。
