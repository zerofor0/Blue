## 1.1 程序文本结构

### （一）代码文本与行合并
【定义】物理行：ASCII/Unicode 文本行，行尾有换行标志。逻辑行：单独物理行或按照显式/隐式规则合并的连续物理行。[P8]
- **显式合并**：在行尾使用 `\` 结束本物理行。`\` 必须为换行符前最后一个字符，其后不得有任何内容（包括空格和注释）。[P9]
- **隐式合并**：小括号 `()`、中括号 `[]` 和大括号 `{}` 中的表达式可自由折行，无语法限制。[P9]

【易错点】使用显式合并 `\` 时，其后紧跟空格或注释会导致语法错误。隐式合并时，括号内各物理行可随意添加注释。

### （二）注释与缩进
【定义】注释：以 `#` 开始至本物理行结束的文本序列。[P10]
【重点】缩进：Python 通过行首缩进确定代码块或语句组的层次。缩进与缩退必须匹配，不符合缩进逻辑的代码无法执行。[P11]
【易错点】不建议在单一脚本文件中混合使用空格和 Tab 键，否则极易引发 `IndentationError`。建议统一使用 4 个空格。

### （三）标识符与命名规范
【定义】标识符：以下划线或字母开头，后跟下划线、字母和数字；长度无限制，区分大小写。不得与系统关键字相同。[P12]
- **命名规范** [P13-P14]：
  - 函数名：全小写，单词间使用下划线，尽量使用动词。
  - 类名：单词首字母大写（驼峰命名），不加下划线。
  - 对象名：大小写均可，短程量可简写。

### （四）关键字与特定形式标识符
【定义】关键字：系统预定义并有特殊意义的标识符，全程序有效。软关键字：仅在特定场合有效的关键字（Python 3.10 引入）。[P15]
【重点】特定形式标识符 [P16]：
- `_*`：模块内部使用，不可通过 `from module import *` 导入。
- `_`：交互模式下表示上一个表达式的结果。
- `__*__`：系统定义的名称（前后双下划线）。
- `__*`：类私有的名称（仅前双下划线）。

## 1.2 对象与型式

### （一）对象的三大基本特征
【必背】Python 中凡物皆对象，对象具有三大基本特征 [P22]：
1. **本征值**：用于区分不同对象的本质信息。在 CPython 中为对象在内存中的存储地址。具有唯一性和有常性（一经创设即无法变更）。使用 `id()` 函数获取。[P23]
2. **型式**：所有型式均为类。型式决定对象的属性和行为。使用 `type()` 函数获取。一般不可改变。[P26-P27]
3. **值**：对象在程序运行期间保存的数据。[P30]

【高频】小整数单子模式：CPython 对 `[-5, 256]` 范围内的整数使用单子模式，即在整个程序运行期间只为该类保留唯一一份对象副本。[P25]
> 【模拟题】在 Python 交互模式下执行 `a = 256; b = 256; print(a is b)` 和 `c = 257; d = 257; print(c is d)`，输出结果分别是什么？
> **答案**：`True` 和 `False`（或受特定运行环境影响，但考点在于 `[-5, 256]` 内的整数 `id` 相同）。

### （二）可变与不可变对象
【定义】[P30]
- **有常对象（不可变对象，immutable）**：值不能变更的对象。如：数值、字符串、元组。
- **无常对象（可变对象，mutable）**：值可能变更的对象。如：字典、列表。
【易错点】有常容器（如元组）不可变指的是其包含的引用不可变，但如果它引用了无常对象（如列表），该无常对象的值依然可以改变。

## 1.2.2-1.2.3 数值型式与文字

### （一）数值型式
【重点】三种数值类型 [P31]：
1. **整型**：正负无穷大，仅受限于内存。
2. **实型**：受限于计算机内部表示，存在取值范围和误差。
3. **复型**：实部 `c.real` 与虚部 `c.imag` 均为浮点数。

### （二）文字（字面量）
【定义】文字：内置型式的有常值，类似常数。文字也是对象。[P32]

【解题步骤】进制表示与前缀识别 [P33]：
- 二进制：`0b` 或 `0B` 起始（如 `0b0101`）
- 八进制：`0o` 或 `0O` 起始（如 `0o2017`）
- 十六进制：`0x` 或 `0X` 起始（如 `0xABADCAD`）
- 十进制：普通整数

【易错点】实型表示误差：实型有效数字 16 位，精确数字 15 位。浮点数运算会产生误差。[P35]
> 示例：`1.1 + 2.2` 结果为 `3.3000000000000003`。需精确计算应使用 `decimal` 模块。

【重点】字符串型文字前缀 [P38-P39]：
- 普通串：单引号、双引号或三引号。Python 无字符类型，单个中英文字符均计数为 1。
- 字节串：加 `b` 前缀（如 `b"Python"`）。
- 原生串：加 `r` 前缀，表示字符串中无转义字符序列（如 `r"C:\name"`）。
- 格式化串：加 `f` 前缀，带有格式化信息（如 `f"{name}"`）。

【记忆】数字分节：Python 3.6+ 允许使用下划线对数字分节增强可读性（如 `1_000_000`）。[P37]

## 1.2.4 量

### （一）量的本质
【定义】量：引用特定对象的标识符，类似代数。量不是对象，仅是对象的引用。量无类型，只有对象才有型式。[P41]
【重点】同一性判定：使用 `is` 与 `is not` 操作符判定两个量是否引用同一对象（即比较本征值/内存地址）。[P41]
【易错点】`is` 与 `==` 的区别：`is` 比较对象的 `id()`（内存地址），`==` 比较对象的值。对于小整数池 `[-5, 256]`，`a is b` 为 True；对于超出此范围或可变对象，即使值相同，`is` 也可能为 False。

## 本章核心结论与速查（1/3片）

### 核心结论
1. Python 通过缩进而非大括号控制代码块，缩进必须严格匹配，禁混 Tab 和空格。
2. 万物皆对象，对象有三大特征：本征值（`id()`）、型式（`type()`）、值。
3. CPython 缓存了 `[-5, 256]` 的整数对象，此范围内的整数赋值引用同一对象。
4. 不可变对象（int, str, tuple）值不可变；可变对象（list, dict）值可变。
5. 变量只是对象的引用（类似指针），没有类型之分，对象才有类型。

## 1.3 表达式

### （一）表达式原子
【定义】表达式原子：构成表达式的最基本单位。最简单的表达式原子为标识符与文字。[P46]
- **标识符**：必须束定至具体对象上，否则引用时将引发 `NameError` 异常。
- **文字（字面量）**：对文字的求值导致对象构造。所有文字均为**有常对象**（不可变对象）。
- 【易错点】同一逻辑行中的相同文字将作为单一文字对象（如 `1.1 is 1.1` 结果为 `True`）；但如果分为两个逻辑行，结果为 `False`。[P46]

### （二）操作符与操作数
【定义】操作符由数学符号、关键字等构成计算逻辑；操作数由文字、量、括号、函数调用等构成计算对象。[P47]
- **分类**：一元（如 `-x`）、二元（如 `a + b`）、三元（条件表达式）。
- 【重点】表达式求值特殊约定（优先级与结合律）：[P52]
  - 示例：`a + -b ** c`，计算顺序为：先计算 `b ** c`，再计算取负 `-`，最后是加法 `+`。
  - 示例：`b ** -c`，计算顺序为：先计算取负 `-c`，再计算 `b ** (-c)`。
- 【易错点】交互模式下，`_` 表示上一表达式的结果。

### （三）括号表达式
【必背】括号表达式的分类及用法：[P53]
| 括号类型 | 用法说明 |
| :--- | :--- |
| **大括号 `{}`** | 1. 集合分列式；2. 字典分列式 |
| **中括号 `[]`** | 1. 下标操作符；2. 切片操作符；3. 列表分列式 |
| **小括号 `()`** | 1. 改变优先级；2. 函数调用；3. 构造空元组；4. 元组定界符；5. 生成器表达式；6. 推导式；7. 类继承语法 |

## 1.4 语句

### （一）简单语句与复合语句
- 【定义】简单语句：表达单一任务，全部代码位于同一逻辑行。多条可用分号 `;` 合并，但**强烈不建议**（降低可读性）。[P57]
- 【定义】复合语句：为复杂任务构造层次结构，位于多条逻辑行中。由子句头和子句体组成，使用冒号 `:` 分隔。[P58]

### （二）空语句
【定义】空语句：使用关键字 `pass` 或省略号型式 `...`（`Ellipsis`），表示空操作，什么都不做。[P59]
- 【应用场景】1. 满足特定语法规则要求（该处需要子句体但暂无指令）；2. 作为未来代码的占位标记。

### （三）赋值语句与表达式
【重点】赋值即定义：未赋值量无定义。赋值的目的是将名称束定到值上。[P60]

1. **赋值表达式（海象运算符）** [P61-62]
   - 【定义】Python 3.8 引入，使用 `:=`，也称具名表达式。
   - 格式：`identifier := expression`
   - 原理：计算表达式值，将名称束定到该对象上，并**返回表达式的值**。
   - 【易错点】赋值表达式优先级最低，一般需要加括号。如：`if (a := 1.1) > (b := 2.2):`

2. **赋值语句** [P63]
   - 格式：`identifier = expression`
   - 连续赋值：`a = b = 1`（合法有效）。

3. **多变量赋值** [P64]
   - 格式：`a, b, c = 1, 2, 3`（从左到右依次进行，实际原理是构造元组对象）。
   - 【应用方法】值互换：`a, b = b, a`（省略临时变量）。

4. **同一性测试** [P65-67]
   - 【易错点】赋值与量的同一性依赖 Python 具体实现：
     - `a = 1` 与 `b = 1`：是否引用同一有常对象存疑（小整数池机制，通常 `a is b` 为 True）。
     - `a = []` 与 `b = []`：总是引用**不同**无常对象。
     - `a = b = []`：总是引用**同一**无常对象（修改 a 会影响 b）。
   - 建议使用 `id()` 函数查看本征值，使用 `is` 操作符查看是否引用同一对象。

### （四）含参赋值与导入语句
1. **含参赋值语句** [P68]
   - 符号：`+=`, `-=`, `*=`, `/=`, `//=`, `%=`, `**=` 等。
   - 【解题步骤】`a += b` 类似于 `a = a + b`。
   - 【易错点】实施此类操作前，右侧子表达式需计算完毕。如 `a *= b + c` 类似于 `a = a * (b + c)`，而非 `a = a * b + c`。

2. **导入语句** [P69]
   - `import math`：导入模块，使用 `math.pi` 访问。
   - `from math import pi`：仅导入模块中的特定名称，直接使用 `pi`。
   - `as` 子句：重命名，防止名称冲突或简化长名称。重命名后原名不可用。

## 1.5 数值与数学计算

### （一）数值运算操作符
【必背】四则运算及相关操作符：[P73]
- `+` / `-` / `*` / `/`（除法）
- `//`：整数除法（向下取整）
- `%`：模运算（取余）
- `**`：幂运算

【高频】移位运算与位运算（仅适用于整数）：[P74-75]
| 操作符 | 名称 | 说明与示例 |
| :--- | :--- | :--- |
| `<<` | 左移 | 左移一位相当于倍乘。`2049 << 10` 等价于 `2049 * 2**10` |
| `>>` | 右移 | 右移一位相当于折半取整。`2049 >> 10` 等价于 `2049 // 2**10` |
| `\|` | 位或 | 全 0 为 0，其他为 1 |
| `^` | 位异或 | 相同为 0，相异为 1 |
| `&` | 位与 | 全 1 为 1，其他为 0 |
| `~` | 位反 | 1 变 0，0 变 1。规律：`n + ~n = -1`，即 `~n = -n - 1` |

### （二）内置数值函数
【重点】内置数值函数分类与注意事项：[P76-80]
1. **数值型式转换函数**：
   - `int(x)`：浮点数或字符串转整数。**【易错点】转换时不四舍五入，小数部分直接舍弃**。
   - `float(x)`：转浮点数。
   - `complex(re[, im])`：合成复数。复数不可直接转整/浮，需用 `.real` 和 `.imag` 提取后转换。
2. **数值表示转换函数**：
   - `bin(x)`：转二进制字符串，前缀 `0b`。
   - `hex(x)`：转十六进制字符串，前缀 `0x`。
   - `oct(x)`：转八进制字符串，前缀 `0o`（Python 3.0 后）。
3. **数值运算函数**：
   - `abs(x)`：绝对值。
   - `divmod(x, y)`：返回二元组 `(x // y, x % y)`。
   - `max()` / `min()`：求最大/最小值。
   - `pow(x, y[, z])`：返回 `x**y`；若提供 `z`，求 `x**y % z`。
   - `round(x[, n])`：四舍五入，可保留 n 位小数。
   - 【高频考点】`pow(x, y, z)` 用于超大整数幂与模混合运算时，**结果精确且速度极快**。如 `pow(2, 2017, 10000)`。若分开计算 `pow(2, 2017) % 10000` 效率极低。

### （三）数学模块与常用模块
【记忆】常用数值计算模块：[P81]
- `math`：C 数学库函数封装。
- `decimal`：十进制定点数，用于精确表示浮点数。
- `fractions`：有理数算术。
- `random`：伪随机数。

【必背】`math` 模块常数：[P82]
- `math.e`：自然对数底数（2.718...）
- `math.pi`：圆周率（3.14159...）
- `math.tau`：倍圆周率（6.283...，Python 3.6 引入）
- `math.inf` / `-math.inf`：正/负无穷大
- `math.nan`：非数标志

## 本章核心结论与速查

1. 表达式原子包括标识符和文字，文字均为有常对象。同一逻辑行内相同文字是同一对象。
2. 赋值的本质是将名称束定到对象上。多变量赋值 `a, b = b, a` 底层依赖元组封装与解包。
3. `a = b = []` 会导致两个变量引用同一个无常对象，修改其中一个会影响另一个。
4. `int()` 转浮点数时直接截断小数部分，不进行四舍五入。
5. `pow(x, y, z)` 在计算大数模幂时具有极高的效率优势，必须使用三参数形式。

### 考前速查表：易错运算与函数
| 考点 | 易错描述 | 正确示例/结论 |
| :--- | :--- | :--- |
| 幂与负号优先级 | `-b ** c` 容易误认为 `(-b) ** c` | 先算 `**` 后算 `-`，即 `-(b ** c)` |
| 含参赋值右侧计算 | `a *= b + c` 容易误算为 `a = a * b + c` | 右侧整体计算：`a = a * (b + c)` |
| `int()` 取整 | 误以为四舍五入 | 直接舍弃小数部分 |
| 位反运算 `~` | 难以快速计算结果 | `~n = -n - 1` |
| 幂模运算 `pow` | 误以为 `pow(x,y) % z` 效率相同 | `pow(x, y, z)` 速度极快，必须使用此形式 |

## 1.5 数学模块与常用模块（续）

### （四）数学模块函数

【重点】`math` 模块提供了对浮点数的底层 C 库数学函数的访问。使用前必须导入：`import math` [P82-89]。

#### 1. 常用数学常数
| 常数 | 含义 | 备注 |
| :--- | :--- | :--- |
| `math.pi` | 圆周率 | 3.141592... |
| `math.e` | 自然对数 | 2.718281... |
| `math.inf` | 正无穷大 | 等价于 `float("inf")`，Python 3.5引入 |
| `math.nan` | 非数标志 | 等价于 `float("NaN")`，Python 3.5引入 |
| `math.tau` | 倍圆周率 | 6.283185...，Python 3.6引入 |

#### 2. 高频数论与表示函数
【高频】以下函数在选择题和填空题中极易出现：
- `math.ceil(x)`：向上取整（返回不小于 x 的最小整数）。
- `math.floor(x)`：向下取整（返回不大于 x 的最大整数）。
- `math.trunc(x)`：返回 x 的整数部分（直接截断小数部分，向零取整）。
- `math.fabs(x)`：返回绝对值（返回浮点数）。
- `math.factorial(x)`：求阶乘。
- `math.gcd(x, y)`：求最大公约数。
- `math.fsum([x1, x2, ...])`：精确求和（避免浮点数精度丢失）。
- `math.isclose(x, y)`：比较是否近似相等。

【易错点】 `math.factorial(x)` 要求 x 为非负整数，若 x 为负数或小数，会抛出 `ValueError` 异常 [P83]。

#### 3. 指数与对数函数
- `math.sqrt(x)`：返回平方根。
- `math.log(x[, base])`：返回对数，默认基底为 `math.e`，可指定 base。
- `math.log2(x)` / `math.log10(x)`：基底为 2 / 10 的对数。
- `math.pow(x, y)`：返回 x 的 y 次方。

【易错点】 `math.pow(x, y)` 返回浮点数，且参数需转换为实数，**不如内置操作符 `**` 和内置函数 `pow()` 精确** [P86]。考试中若求整数幂，优先使用 `**`。

#### 4. 三角函数
- `math.sin(x)` / `math.cos(x)` / `math.tan(x)`：参数 x 必须是**弧度**。
- `math.degrees(x)`：弧度转角度。
- `math.radians(x)`：角度转弧度。
- `math.hypot(x, y)`：返回平面向量的模长 $\sqrt{x^2 + y^2}$。

【记忆】 三角函数参数全为弧度，遇到角度先 `radians()` 转换。

## 1.6 基本输入输出

【定义】Python 输入输出以**字符串**为媒介。无论整数、实数，还是其他对象，均以字符串型式输入输出，由程序员手动或由程序自动转型为其他型式对象 [P94]。

### （一）输出函数 print()

【必背】 `print()` 完整格式：
```python
print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
```
[P95-96]

**参数解析：**
| 参数 | 功能 | 默认值 | 考点说明 |
| :--- | :--- | :--- | :--- |
| `*objects` | 输出对象序列 | 无 | 多个对象自动调用 `str()` 转换为字符串 |
| `sep` | 分隔符 | `' '` (空格) | 多个对象输出时之间的连接符 |
| `end` | 结束符 | `'\n'` (换行) | 输出完毕后的结尾字符，常用于不换行输出 |
| `file` | 输出目的地 | `sys.stdout` | 重定向输出到文件 |
| `flush` | 流刷新 | `False` | 是否强制刷新缓冲区 |

【解题步骤】 常见输出格式控制题：
1. 若要求同行输出并用特定字符分隔：设置 `sep` 参数（如 `print(a, b, sep=',')`）。
2. 若要求输出不换行：设置 `end` 参数（如 `print(a, end=' ')`）。

### （二）输入函数 input()

【定义】 `input([prompt])`：读取用户的原生输入数据，**返回值为字符串类型** [P98]。
- 参数 `prompt` 为提示信息，会在获取输入前输出。
- 可以接受字符串、数字和表达式等多种型式的数据输入，但**一律作为字符串返回**。

【易错点】 
1. 纯数字输入时，`input()` 结果是字符串（如 `'1'`），不能直接参与算术运算，必须经过转型或求值 [P99]。
2. 正常字符串输入不需要输入引号；若输入引号，则引号被作为普通字符保留在返回的字符串中 [P99]。

### （三）求值函数 eval()

【定义】 `eval(expression, globals=None, locals=None)`：将字符串 `expression` 作为 Python 表达式进行分析和求值，并返回其结果 [P101]。

【高频】 `eval()` 常与 `input()` 结合使用，用于获取用户输入的数值或表达式：
```python
# 接收一个数字或数学表达式并计算
n = eval(input('A number: ')) 
```
[P102]

【易错点】 `eval()` 会执行字符串内的任何代码，存在安全隐患；但在基础文法阶段，它是将输入的字符串 `"1+2"` 转换为计算结果 `3` 的核心方法。

### （四）表示函数 repr()

【定义】 `repr(object)`：返回对象的“正式”字符串表示 [P104]。
- 大多数情况下，将 `repr()` 的输出回传给 `eval()` 能推导出具有同样值的对象。

【解题步骤】 `repr()` 与 `eval()` 的互逆特性：
```python
>>> repr(1 + 2)   # 表达式求值后转为正式字符串
'3'
>>> eval('3')     # 字符串转为整型
3

>>> repr('1 + 2') # 字符串的正式表示（带引号）
"'1 + 2'"
>>> eval("'1 + 2'") # 还原为字符串
'1 + 2'
```

## 核心结论
1. `math` 模块中取整函数的区别：`ceil` 向上、`floor` 向下、`trunc` 向零截断。
2. `math.pow()` 返回浮点数且精度不如内置 `**`。
3. `input()` 函数的返回值永远是字符串，必须用 `eval()` 或类型转换函数（如 `int()`）处理才能进行算术运算。
4. `print()` 函数通过 `sep` 和 `end` 参数控制输出格式，默认分隔符为空格，默认结束符为换行。
5. `repr()` 和 `eval()` 在多数情况下互为逆运算。

## 考前速查表：易错运算与函数

| 考点 | 易错描述 | 正确示例/结论 |
| :--- | :--- | :--- |
| `input()` 返回类型 | 误以为输入数字返回数字 | `input()` 永远返回 `str` |
| `eval(input())` | 忘记配合 `eval` 导致字符串相加 | `eval("1+2")` -> `3`，`"1"+"2"` -> `"12"` |
| `math.factorial()` | 传入小数或负数 | 抛出 `ValueError` |
| `math.pow()` vs `**` | 混淆两者精度和类型 | `2**3` -> `int 8`，`math.pow(2,3)` -> `float 8.0` |
| `math.sqrt()` | 传入负数 | 抛出 `ValueError`（复数需用 `cmath`） |
| `print()` 默认换行 | 忘记修改 `end` 参数导致自动换行 | 不换行需设 `end=''` |
| 三角函数参数 | 误传角度值给 `math.sin()` | 必须传弧度，角度用 `radians()` 转换 |

## 模拟题
**【模拟题】** 阅读以下 Python 程序片段：
```python
import math
a = input("请输入一个数字: ")
b = eval(a)
print(math.floor(b), math.ceil(b), end=";")
print(math.trunc(b))
```
若用户输入 `-3.14`，写出程序输出结果。

**【解析】** 
1. `input()` 获取字符串 `"-3.14"`。
2. `eval()` 将其转为浮点数 `-3.14`。
3. `math.floor(-3.14)` 向下取整，结果为 `-4`。
4. `math.ceil(-3.14)` 向上取整，结果为 `-3`。
5. 第一个 `print` 结束符设为 `";"`，不换行。
6. `math.trunc(-3.14)` 截断小数部分，向零取整，结果为 `-3`。
**输出结果：** `-4 -3;-3`

## 五、全课程易错点汇总

- 使用显式合并 `\` 时，其后紧跟空格或注释会导致语法错误。隐式合并时，括号内各物理行可随意添加注释。
- 不建议在单一脚本文件中混合使用空格和 Tab 键，否则极易引发 `IndentationError`。建议统一使用 4 个空格。
- 有常容器（如元组）不可变指的是其包含的引用不可变，但如果它引用了无常对象（如列表），该无常对象的值依然可以改变。
- 实型表示误差：实型有效数字 16 位，精确数字 15 位。浮点数运算会产生误差。[P35]
- `is` 与 `==` 的区别：`is` 比较对象的 `id()`（内存地址），`==` 比较对象的值。对于小整数池 `[-5, 256]`，`a is b` 为 True；对于超出此范围或可变对象，即使值相同，`is` 也可能为 False。
- `math.factorial(x)` 要求 x 为非负整数，若 x 为负数或小数，会抛出 `ValueError` 异常 [P83]。
- `math.pow(x, y)` 返回浮点数，且参数需转换为实数，**不如内置操作符 `**` 和内置函数 `pow()` 精确** [P86]。考试中若求整数幂，优先使用 `**`。
- 纯数字输入时，`input()` 结果是字符串（如 `'1'`），不能直接参与算术运算，必须经过转型或求值 [P99]。
- 正常字符串输入不需要输入引号；若输入引号，则引号被作为普通字符保留在返回的字符串中 [P99]。
- `eval()` 会执行字符串内的任何代码，存在安全隐患；但在基础文法阶段，它是将输入的字符串 `"1+2"` 转换为计算结果 `3` 的核心方法。
