#### 本章总览
本章引入一类研究随机变量分布与数字特征的有力工具：**概率母函数（PGF）** 与**矩母函数（MGF）**。核心问题是：如何利用一个函数“打包”随机变量的所有概率信息或矩信息？高频考点包括：PGF的定义与收敛域、利用PGF求概率和数字特征（期望、方差）、独立随机变量和的PGF、以及常见分布（二项、泊松、几何）的PGF推导与应用。核心方法为：通过求导获取概率/矩，以及利用独立性和PGF判定和分布的类型。

---

#### 一）概率母函数（PGF）的定义与计算 【必背】

**【定义】**
设 $X$ 是取非负整数值的随机变量，其分布列为 $p_j = \mathbb{P}(X = j)$，$j=0,1,2,\dots$。称
$$g(s) = \mathbb{E}[s^X] = \sum_{j=0}^{\infty} s^j \mathbb{P}(X = j), \quad s \in [-1, 1]$$
为 $X$ 的概率母函数（Probability Generating Function, PGF），约定 $0^0 = 1$。
[Lec12_2026_0520.pdf P9]

**原理与推导**
PGF可视为序列 $\{p_j\}$ 的一个幂级数表示。由于 $\sum |p_j| = 1$，该级数在 $|s| \le 1$ 时绝对收敛，因此定义在 $[-1,1]$ 上是良好的。PGF 唯一决定了随机变量的分布列。

**【解题步骤】**
计算离散随机变量 PGF 的一般步骤：
1. **确定分布列**：写出 $p_j = \mathbb{P}(X=j)$ 的表达式。
2. **代入定义**：计算 $g(s) = \sum_{j} p_j s^j$。
3. **化简级数**：利用等比数列求和、二项式定理或指数函数的幂级数展开等技巧，将求和式化为封闭形式（初等函数）。

**【易错点】**
- **定义域混淆**：PGF的定义域是 $[-1, 1]$，级数在此区间内绝对收敛。忘记此前提可能导致误用性质。
- **非负整值要求**：PGF只对取非负整数值的随机变量定义。对于可取负数或非整数的随机变量，需引入矩母函数或特征函数。
- **$0^0$ 约定**：计算 $g(0)$ 时，$g(0) = \sum s^j \mathbb{P}(X=j) \big|_{s=0} = \mathbb{P}(X=0)$，这与 $0^0=1$ 的约定一致。

| 易混概念对比 | 概率母函数 (PGF) | 矩母函数 (MGF) |
| :--- | :--- | :--- |
| **定义** | $\mathbb{E}[s^X]$ | $\mathbb{E}[e^{tX}]$ |
| **适用变量** | 仅适用于**非负整数值**随机变量 | 适用于**一般**随机变量 |
| **主要作用** | 唯一决定分布列，便于处理独立和 | 唯一决定分布，便于计算各阶矩 |

**例题**
**例题**：
**题目**：设 $X$ 是取非负整值的随机变量，$g(s)$ 是其概率母函数，求 $g(1)$ 和 $g'(1)$ 的值。[Lec12_2026_0520.pdf P9-P10]
**思路**：直接从PGF定义和绝对收敛性质出发，通过代入 $s=1$ 和逐项求导解决。
**解答**：
1. 代入定义：$g(s) = \sum_{j=0}^{\infty} s^j \mathbb{P}(X=j)$。
2. 当 $s=1$ 时，$g(1) = \sum_{j=0}^{\infty} 1^j \mathbb{P}(X=j) = \sum \mathbb{P}(X=j) = 1$。
3. 对定义逐项求导：$g'(s) = \sum_{j=1}^{\infty} j s^{j-1} \mathbb{P}(X=j)$。
4. 代入 $s=1$：$g'(1) = \sum_{j=1}^{\infty} j \cdot 1^{j-1} \mathbb{P}(X=j) = \sum j \mathbb{P}(X=j) = \mathbb{E}[X]$。
**技巧**：务必记住PGF在1处的函数值和一阶导数值对应着概率总和与期望，这是串联PGF与数字特征的桥梁。

#### 二）概率母函数的基本性质 【高频】

**【定义】**
设 $g(s)$ 为 $X$ 的PGF，$g^{(k)}(s)$ 是其 $k$ 阶导数。

**原理与推导**
根据幂级数的性质，PGF在 $(-1,1)$ 内无限次可微，并且可以逐项求导。利用此解析性质，可以方便地提取概率和矩信息。
1. **求概率与期望**：
   - 由 $g(s) = \sum p_j s^j$，令 $s=1$ 得 $g(1)=1$。
   - 逐项求导得 $g'(s) = \sum j p_j s^{j-1}$，令 $s=1$ 得 $g'(1) = \mathbb{E}[X]$。[Lec12_2026_0520.pdf P11]
2. **提取分布列**：
   - 将 $g(s)$ 泰勒展开，$s^j$ 的系数即 $\mathbb{P}(X=j)$，即 $\mathbb{P}(X=k) = \frac{g^{(k)}(0)}{k!}$。这保证了PGF与分布列是相互唯一决定的。[Lec12_2026_0520.pdf P12]
3. **独立随机变量和**：
   - 若 $X_1,\dots,X_n$ 相互独立，其PGF分别为 $g_1(s),\dots,g_n(s)$，则其和 $Y = X_1+\dots+X_n$ 的PGF为 $g_Y(s) = \prod_{i=1}^n g_i(s)$。这是因为 $\mathbb{E}[s^{\sum X_i}] = \mathbb{E}[\prod s^{X_i}] \stackrel{ind.}{=} \prod \mathbb{E}[s^{X_i}]$。[Lec12_2026_0520.pdf P13]

4. **计算方差**：
   - 对 $g(s)$ 求二阶导：$g''(s) = \sum j(j-1) p_j s^{j-2}$。
   - 令 $s=1$：$g''(1) = \mathbb{E}[X(X-1)] = \mathbb{E}[X^2] - \mathbb{E}[X]$。
   - 因此，方差可表示为：
   $$\mathrm{Var}(X) = \mathbb{E}[X^2] - (\mathbb{E}[X])^2 = g''(1) + g'(1) - [g'(1)]^2$$
   [Lec12_2026_0520.pdf P15]

**【解题步骤】**
- **步骤 1：求期望**：对PGF求导，代入 $s=1$。
- **步骤 2：求方差**：先求 $g'(1)$ 和 $g''(1)$，代入公式 $\mathrm{Var}(X) = g''(1) + g'(1) - [g'(1)]^2$。
- **步骤 3：求独立和的分布**：分别写出各分量的PGF，相乘，再通过泰勒展开或幂级数识别法确定和变量的分布类型及参数。

**【易错点】**
- **求导不彻底**：计算方差时，必须正确使用 $g''(1) = \mathbb{E}[X(X-1)]$，而非直接得到 $\mathbb{E}[X^2]$。忘记加回 $\mathbb{E}[X]$ 项是常见错误。
- **独立性前提**：独立随机变量和的PGF等于各自PGF的乘积，此性质**必须**以独立性为前提，对非独立变量不成立。
- **泰勒展开与幂级数**：从乘积PGF还原分布列时，可利用泰勒展开公式 $p_k = g^{(k)}(0)/k!$，或直接尝试将乘积累化为已知分布的标准幂级数形式。后者通常更快捷。

**例题**
**例题**：
**题目**：设 $X_1, X_2, \dots, X_n$ 是相互独立的随机变量，$X_i \sim \mathcal{P}(\lambda_i)$。求 $Y = X_1 + X_2 + \dots + X_n$ 的分布。[Lec12_2026_0520.pdf P18, P19]
**思路**：利用独立和的PGF性质和泊松分布的PGF，计算出 $Y$ 的PGF，再将其与标准分布的PGF对比，反推出分布类型。
**解答**：
1. 已知 $X_i \sim \mathcal{P}(\lambda_i)$，其PGF为 $g_{X_i}(s) = e^{\lambda_i(s-1)}$。（见下方例1.2）
2. 由于各 $X_i$ 相互独立，应用性质3，$Y$ 的PGF为各PGF之积：
   $$g_Y(s) = \prod_{i=1}^n g_{X_i}(s) = \prod_{i=1}^n e^{\lambda_i(s-1)} = e^{(\sum_{i=1}^n \lambda_i)(s-1)}$$
3. 观察 $g_Y(s)$ 的形式，它恰为参数 $\lambda = \sum \lambda_i$ 的泊松分布的PGF。
4. 因此，由PGF与分布的唯一确定性，知 $Y \sim \mathcal{P}(\sum_{i=1}^n \lambda_i)$。
**技巧**：独立泊松分布之和仍为泊松分布，参数为各参数之和。记住“PGF相乘，参数相加”。

#### 三）常见分布的概率母函数 【必背】

##### 1. 二项分布 $B(n, p)$
- **PGF推导**：
  $$g(s) = \mathbb{E}[s^X] = \sum_{j=0}^n s^j \binom{n}{j} p^j q^{n-j} = \sum_{j=0}^n \binom{n}{j} (sp)^j q^{n-j} = (q + sp)^n$$
  其中 $q = 1-p$。[Lec12_2026_0520.pdf P16]
- **应用**：若 $X_i \sim B(n_i, p)$ 且相互独立，则 $\sum X_i \sim B(\sum n_i, p)$。

##### 2. 泊松分布 $\mathcal{P}(\lambda)$
- **PGF推导**：
  $$g(s) = \sum_{k=0}^{\infty} s^k \frac{\lambda^k}{k!} e^{-\lambda} = e^{-\lambda} \sum_{k=0}^{\infty} \frac{(s\lambda)^k}{k!} = e^{-\lambda} \cdot e^{s\lambda} = e^{\lambda(s-1)}$$
  [Lec12_2026_0520.pdf P17]
- **应用**：多个独立泊松变量之和仍为泊松变量，参数相加。

##### 3. 几何分布 $G(p)$
- **PGF推导**：
  $$g(s) = \sum_{k=1}^{\infty} s^k p q^{k-1} = \frac{sp}{1 - sq}$$
  其中 $q = 1-p$，收敛域 $|sq| < 1$。[Lec12_2026_0520.pdf P20]
- **应用**：$m$ 个独立同分布的几何分布变量之和（$S_m$）服从帕斯卡分布。其PGF为 $g_{S_m}(s) = \left(\frac{sp}{1 - sq}\right)^m$。将PGF泰勒展开或求导，可得其分布列为：
  $$\mathbb{P}(S_m=k) = \binom{k-1}{m-1} p^m q^{k-m}, \quad k = m, m+1, \dots $$
  [Lec12_2026_0520.pdf P20-P21]

#### 四）随机向量的概率母函数 【重点】

**【定义】**
设 $X, Y$ 是二维取非负整数值的随机向量，其联合分布列为 $p_{ik} = \mathbb{P}(X=i, Y=k), i,k=0,1,2,\dots$。称
$$g(s, t) = \mathbb{E}[s^X t^Y] = \sum_{i=0}^{\infty} \sum_{k=0}^{\infty} p_{ik} s^i t^k, \quad s,t \in [-1, 1]$$
为 $(X, Y)$ 的概率母函数。[Lec12_2026_0520.pdf P22]

**原理与推导**
二维PGF是一维情形的直接推广，其核心价值在于刻画两变量间的关系。
- **性质**：
  1. 有界性：$|g(s,t)| \le g(1,1) = 1$，当 $|s| \le 1, |t| \le 1$。
  2. 边际性：$g_X(s) = g(s, 1)$，$g_Y(t) = g(1, t)$。
  3. **独立性判据**：$X$ 与 $Y$ 相互独立，当且仅当 $g(s, t) = g_X(s) \cdot g_Y(t)$ 对所有 $s, t$ 成立。
  4. 联合分布列提取：$p_{ik} = \frac{1}{i! k!} \cdot \frac{\partial^{i+k} g(s,t)}{\partial s^i \partial t^k} \Big|_{s=0, t=0}$。[Lec12_2026_0520.pdf P23]

**【解题步骤】**
- **判独立**：计算 $g(s, t)$，再分别计算 $g(s,1)$ 和 $g(1,t)$。验证 $g(s, t) = g(s,1) \cdot g(1,t)$ 是否成立。
- **求边际分布列**：对联合PGF分别令 $t=1$ 或 $s=1$，得到边际PGF，再按一维方法提取概率。

**【易错点】**
- **独立性的充分必要性**：PGF的可乘性$g(s,t)=g_X(s)g_Y(t)$是独立的**充要条件**，比相关系数为零等条件强得多，是判断独立的强有力工具。
- **与协方差/相关系数的区别**：两个随机变量的PGF满足乘积分解等价于独立，但协方差为0只是不相关的必要条件，不能推出独立。PGF是判定非负整值变量独立性的终极标准之一。

#### 五）从PGF到MGF的过渡 【重点】

**原理与推导**
概率母函数 $g(s) = \mathbb{E}[s^X]$ 只适用于非负整值变量。若随机变量可取负数或非整数（如正态分布），$\mathbb{E}[s^X]$ 可能无法良好定义。
为了解决这个问题，我们将基底 $s$ 替换为 $e^t$，得到：
$$\mathbb{E}[e^{tX}]$$
当 $t$ 使期望存在时，该函数称为**矩母函数**（MGF）。MGF 适用于任何实值随机变量，并且其各阶导数在 $t=0$ 处的值直接给出各阶原点矩：$\mathbb{E}[X^k] = M^{(k)}(0)$。[Lec12_2026_0520.pdf P24]

**【易错点】**
- PGF与MGF的联系：形式上，MGF $M(t) = \mathbb{E}[e^{tX}]$，PGF为 $\mathbb{E}[s^X]$。因此，当 $X$ 取值非负整数时，有 $M(t) = g(e^t)$ 或 $g(s) = M(\ln s)$。
- PGF的局限性是引入MGF的直接动机。

#### 本章小结

- **知识关系**：本章介绍了研究非负整数值随机变量的核心工具——概率母函数（PGF）。PGF的解析性质（可逐项求导）搭建了从“生成函数”到“概率/数字特征”的桥梁。独立性假设下，PGF的乘法性质极大简化了随机变量和的分布研究。PGF的局限性自然地引出了适用范围更广的矩母函数（MGF）。
- **核心结论**：
  1. PGF可唯一确定非负整数值随机变量的分布列。
  2. 利用PGF的导数可轻松计算期望和方差：$\mathbb{E}[X] = g'(1)$, $\mathrm{Var}(X) = g''(1) + g'(1) - [g'(1)]^2$。
  3. 独立随机变量和的PGF等于它们各自PGF的乘积。
- **必背公式**：
  - **PGF定义**：$g(s) = \sum_{j=0}^\infty s^j \mathbb{P}(X=j)$
  - **求期望**：$\mathbb{E}[X] = g'(1)$
  - **求方差**：$\mathrm{Var}(X) = g''(1) + g'(1) - [g'(1)]^2$
  - **二项分布 PGF**：$B(n,p): g(s) = (q+sp)^n$
  - **泊松分布 PGF**：$\mathcal{P}(\lambda): g(s) = e^{\lambda(s-1)}$
  - **几何分布 PGF**：$G(p): g(s) = \dfrac{sp}{1-sq}$
- **【记忆】一句话记忆**：PGF是概率的幂级数“生成器”，在1处求导得期望，独立场合乘积化。
