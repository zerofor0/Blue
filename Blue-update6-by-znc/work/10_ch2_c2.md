#### 矩母函数 【高频】
##### 【定义】
设 $X$ 是一个随机变量，若对于某个 $h > 0$，期望 $E[e^{sX}]$ 对所有 $|s| < h$ 均存在，则称函数
$$M_X(s) = E[e^{sX}]$$
为 $X$ 的**矩母函数**(Moment Generating Function, MGF)  [Lec12_2026_0520.pdf P26]
- 离散变量：$M_X(s) = \sum_{k} e^{s x_k} \mathbb{P}(X = x_k)$
- 连续变量：$M_X(s) = \int_{-\infty}^{+\infty} e^{s x} f_X(x) dx$

##### 原理与推导
矩母函数因其能“生成”各阶矩而得名。将指数函数 $e^{sX}$ 在 $s=0$ 处进行 Taylor 展开，并对两边取期望得到：
$$M_X(s) = E\left[ \sum_{k=0}^{\infty} \frac{(sX)^k}{k!} \right] = \sum_{k=0}^{\infty} \frac{E[X^k]}{k!} s^k$$
这意味着 $k$ 阶矩 $E[X^k]$ 正是 $M_X(s)$ 在 $0$ 点 Taylor 展开中 $\frac{s^k}{k!}$ 项的系数。因此，对矩母函数求 $k$ 次导数并在 $s=0$ 处求值，便得到 $k$ 阶矩：
$$E[X^k] = M_X^{(k)}(0), \quad k=1,2,\ldots$$
[Lec12_2026_0520.pdf P27]

##### 矩母函数的性质 【必背】
设随机变量 $X$ 的矩母函数为 $M_X(s)$。
1.  **生成矩**：$E[X^k] = M_X^{(k)}(0)$
2.  **唯一性**：若 $M_X(s)$ 在 $s=0$ 附近存在，则它唯一地决定了 $X$ 的分布函数。(不要求证明)
3.  **线性变换**：若 $Y = aX + b$，则 $Y$ 的矩母函数为：
    $$M_Y(s) = e^{sb} M_X(as)$$
4.  **独立和**：若 $X_1, X_2, \dots, X_n$ 相互独立，其矩母函数分别为 $M_{X_i}(s)$，则它们的和 $S_n = X_1 + \dots + X_n$ 的矩母函数为：
    $$M_{S_n}(s) = \prod_{i=1}^{n} M_{X_i}(s)$$
[Lec12_2026_0520.pdf P27]

##### 常见分布的矩母函数 【必背】
| 分布 | PGF | MGF | 备注 [Lec12_2026_0520.pdf P30, P32, P34] |
| :--- | :--- | :--- | :--- |
| **二项分布** $B(n,p)$ | $(q+ps)^n$ | $(q+pe^s)^n$ | $q=1-p$ |
| **泊松分布** $\mathcal{P}(\lambda)$ | $e^{\lambda(s-1)}$ | $e^{\lambda(e^s-1)}$ | |
| **指数分布** $Exp(\lambda)$ | - | $\frac{\lambda}{\lambda - s}$ | 存在域为 $s < \lambda$ |
| **标准正态分布** $N(0,1)$ | - | $e^{\frac{s^2}{2}}$ | |
| **一般正态分布** $N(\mu, \sigma^2)$ | - | $e^{\mu s + \frac{\sigma^2 s^2}{2}}$ | |
| **均匀分布** $U(a,b)$ | - | $\frac{e^{sb} - e^{sa}}{s(b-a)}$ | 特殊情况 $s=0$ 时取极限值为 1 |

##### 【解题步骤】
1.  **求矩**：给定分布，可先求 MGF，再求导并令 $s=0$ 得各阶矩。导数的计算往往比直接积分/求和简便。
2.  **求独立和的分布**：当难以直接计算独立随机变量和的分布时，可利用性质4。将各独立成分的MGF相乘，观察所得MGF形式，由唯一性识别和的分布。
3.  **判断分布类型**：熟记常见分布的MGF形式。当遇到一个形式复杂但实际是常见分布MGF乘积的函数时，可直接用唯一性判断分布，无需从pdf/pmf重新推导。

##### 【易错点】
- **MGF可能不存在**：并非所有分布都有矩母函数。例如，**柯西分布**(Cauchy Distribution) 的矩母函数不存在，因为积分 $\int e^{sx} \frac{1}{\pi(1+x^2)} dx$ 对任何 $s \neq 0$ 都发散至无穷。 [Lec12_2026_0520.pdf P37]
- **定义域的重要性**：MGF是在 $s=0$ 附近的邻域定义的。如指数分布的MGF只对 $s < \lambda$ 有定义。讨论MGF时必须注意其定义域。
- **不要混淆独立和与非独立和**：性质4（MGF的乘积）**仅对独立随机变量之和**成立。
- **PGF与MGF的混淆**：
  | 特征 | 概率母函数 (PGF) | 矩母函数 (MGF) |
  | :--- | :--- | :--- |
  | **定义** | $g(s) = E[s^X]$ | $M(s) = E[e^{sX}]$ |
  | **主要应用** | 非负整数值随机变量 | 更广泛的连续/离散随机变量 |
  | **与矩的关系** | $E[X] = g'(1)$ | $E[X^k] = M^{(k)}(0)$ |
  | **关系** | $M_X(s) = g_X(e^s)$ | $g_X(s) = M_X(\ln s)$ |

##### 例2.1 （均匀分布的矩母函数）[Lec12_2026_0520.pdf P31]
**题目**：设 $X \sim U(a, b)$，求其矩母函数 $M_X(s)$。
**思路**：直接应用连续随机变量矩母函数的定义进行计算。
**解答**：
1.  **代入定义**：$M_X(s) = \int_{a}^{b} e^{sx} \cdot \frac{1}{b-a} dx$
2.  **积分求解**：$M_X(s) = \frac{1}{s(b-a)} [e^{sx}]_{a}^{b}$
3.  **代入上下限**：$M_X(s) = \frac{e^{sb} - e^{sa}}{s(b-a)}$ （当 $s \neq 0$ 时；$M(0)=1$ 为其极限）
**技巧**：此处的积分非常基础，关键点在于最后结果形式，可用于结合性质2（唯一性）判断某复杂形式是否来自均匀分布。

##### 例2.2 （指数分布的矩母函数与矩）[Lec12_2026_0520.pdf P32]
**题目**：设 $X \sim Exp(\lambda)$，密度函数为 $f(x)=\lambda e^{-\lambda x}I_{(x>0)}$。求 (1) 矩母函数 $M_X(s)$；(2) 期望 $E[X]$ 和二阶矩 $E[X^2]$。
**思路**：先利用连续随机变量定义计算MGF，再通过对MGF求导得到各阶矩。
**解答**：
1.  **求MGF**：对 $s < \lambda$，有
    $$M_X(s) = \int_{0}^{\infty} e^{sx} \cdot \lambda e^{-\lambda x} dx = \lambda \int_{0}^{\infty} e^{(s-\lambda)x} dx = \lambda \left[ \frac{e^{(s-\lambda)x}}{s-\lambda} \right]_{0}^{\infty} = \frac{\lambda}{\lambda - s}$$
2.  **求一阶矩 $E[X]$**：对MGF求一阶导：$M'(s) = \frac{\lambda}{(\lambda - s)^2}$。令 $s=0$，得 $E[X] = M'(0) = \frac{1}{\lambda}$。
3.  **求二阶矩 $E[X^2]$**：对MGF求二阶导：$M''(s) = \frac{2\lambda}{(\lambda - s)^3}$。令 $s=0$，得 $E[X^2] = M''(0) = \frac{2}{\lambda^2}$。
**技巧**：求矩母函数后再求导，比直接积分 $\int x^n \lambda e^{-\lambda x} dx$ 每次都要分部积分要更快且不易出错。

##### 例2.3 （独立正态分布之和的分布）[Lec12_2026_0520.pdf P34]
**题目**：设 $X \sim N(\mu_1, \sigma_1^2)$ 与 $Y \sim N(\mu_2, \sigma_2^2)$ 相互独立。使用矩母函数证明 $Z = X + Y$ 服从 $N(\mu_1 + \mu_2, \sigma_1^2 + \sigma_2^2)$。
**思路**：利用 “线性变换性质” 先求出 $X, Y$ 的MGF，再利用 “独立和性质” 将它们相乘，最后观察乘积结果的形式，由“唯一性”得出结论。
**解答**：
1.  **建立基础**：已知标准正态分布 $V \sim N(0, 1)$ 的MGF为 $M_V(s) = e^{s^2/2}$。
2.  **求一般正态分布的MGF**：由 $W = \mu + \sigma V \sim N(\mu, \sigma^2)$，利用性质3得 $M_W(s) = e^{\mu s} M_V(\sigma s) = e^{\mu s + \frac{\sigma^2 s^2}{2}}$。
3.  **求 $Z$ 的MGF**：
    $$\begin{aligned} M_Z(s) &= M_X(s) \cdot M_Y(s) \\ &= \exp\left(\mu_1 s + \frac{\sigma_1^2 s^2}{2}\right) \cdot \exp\left(\mu_2 s + \frac{\sigma_2^2 s^2}{2}\right) \\ &= \exp\left((\mu_1+\mu_2)s + \frac{(\sigma_1^2+\sigma_2^2) s^2}{2}\right) \end{aligned}$$
4.  **识别分布**：将 $M_Z(s)$ 的表达式与一般正态分布 $N(\mu, \sigma^2)$ 的MGF $e^{\mu s + \frac{\sigma^2 s^2}{2}}$ 对比，知 $Z \sim N(\mu_1+\mu_2, \sigma_1^2+\sigma_2^2)$。证毕。
**技巧**：矩母函数的最大价值体现在这类问题上：**求独立随机变量和的分布**。它将复杂的卷积运算转化为简单的函数相乘，结合唯一性，威力巨大。

#### 特征函数 【重点】
##### 【定义】
对于任意随机变量 $X$，其特征函数 (Characteristic Function, C.F.) 定义为
$$\phi_X(t) = E[e^{itX}] = E[\cos(tX) + i\sin(tX)], \quad t \in \mathbb{R}$$
其中 $i = \sqrt{-1}$ 为虚数单位。 [Lec12_2026_0520.pdf P39]

##### 原理与推导
特征函数与矩母函数的定义区别在于用 $it$ 替代了 $s$。由于欧拉公式 $e^{i\varphi} = \cos\varphi + i\sin\varphi$，使得 $|e^{itX}| \equiv 1$，积分/求和总是绝对收敛。这克服了矩母函数可能不存在的缺陷（如柯西分布），使得每个随机变量的特征函数都存在，成为一种应用更广泛的分布刻画工具。 [Lec12_2026_0520.pdf P37, P39]

##### 与矩母函数的关系
形式上，$\phi_X(t) = M_X(it)$。很大程度上，矩母函数的性质（线性变换、独立和的乘法等）可以平移到特征函数上。求矩时，则有 $E[X^k] = \frac{1}{i^k} \phi^{(k)}(0)$。

##### 随机向量的矩母函数
对于随机向量 $\vec{X} = (X_1, \dots, X_n)^T$，定义多元矩母函数:
$$M_{\vec{X}}(\vec{s}) = E[e^{\vec{s}^T \vec{X}}] = E[e^{s_1 X_1 + \dots + s_n X_n}], \quad \vec{s} \in \mathbb{R}^n$$
[Lec12_2026_0520.pdf P36]

### 本章小结

本章系统介绍了刻画随机变量分布的两类解析工具：概率母函数(PGF)、矩母函数(MGF)，并简要引入特征函数(C.F.)。其核心路径是从“分布列/密度函数”过渡到“函数”，利用函数的代数运算简化概率问题。

**本章知识关系**：本章从PGF扩展到MGF，再推广至适用范围更广的C.F.。PGF擅长处理非负整数离散随机变量及其和；MGF作为核心工具，将求矩、求独立和分布等问题函数化；C.F.作为MGF的复数域推广，提供了理论上的完美解决方案。

**核心结论**:
- **PGF**：$g_X(s) = E[s^X]$，$X$ 为非负整数。主要性质：$P(X=k) = g^{(k)}(0)/k!$，$E[X] = g'(1)$。
- **MGF**：$M_X(s) = E[e^{sX}]$，性质更强。核心在于 $E[X^k] = M^{(k)}(0)$ 和独立随机变量和的MGF等于各自MGF之积。
- **C.F.**：$\phi_X(t) = E[e^{itX}]$，始终存在，是理论分析利器。

**必背公式**:
- 关系：$M_X(s) = g_X(e^s)$
- MGF求矩：$E[X^k] = M_X^{(k)}(0)$
- 线性变换：$M_{aX+b}(s) = e^{sb}M_X(as)$
- 独立和：$M_{X_1+\dots+X_n}(s) = \prod_{i=1}^{n} M_{X_i}(s)$
- 二项分布 $B(n,p)$ 的MGF: $(q+pe^s)^n$
- 泊松分布 $\mathcal{P}(\lambda)$ 的MGF: $e^{\lambda(e^s-1)}$
- 指数分布 $Exp(\lambda)$ 的MGF: $\frac{\lambda}{\lambda - s}$（$s < \lambda$）
- 正态分布 $N(\mu, \sigma^2)$ 的MGF: $e^{\mu s + \frac{\sigma^2 s^2}{2}}$

【记忆】MGF像幂级数版PGF，指数项换底 $s \to e^s$，核心是求矩导0和独立和相乘。
