#### 本章总览
- **核心问题**：如何求多维随机变量函数的分布？三大抽样分布（卡方、t、F）是如何由正态分布构造出来的？
- **高频考点排序**：1. 卷积公式求和的分布；2. 雅可比变换法求联合密度；3. 三大抽样分布的构造与性质；4. CDF法求一般函数的分布。
- **方法清单**：分布函数法（CDF法）、卷积公式法（和/差）、变量代换法（雅可比行列式）。

### （一）三大抽样分布

#### 卡方分布 $\chi^2_v$ 【必背】
1. 【定义】：设 $v > 0$，若随机变量 $X$ 的密度函数为
$$f(x) = \frac{1}{2^{v/2}\Gamma(v/2)} x^{v/2-1} e^{-x/2}, \quad x \ge 0$$
则称 $X$ 服从参数为 $v$ 的卡方分布，记作 $X \sim \chi^2_v$。
2. 原理与推导：标准正态变量的平方和服从卡方分布。设 $Z_1, \dots, Z_n$ 独立同分布且 $Z_j \sim \mathcal{N}(0,1)$，令 $X = Z_1^2 + \dots + Z_n^2$，则 $X \sim \chi^2_n$。此外，$\chi^2_n$ 本质上是 Gamma 分布，即 $\chi^2_n = \Gamma(n/2, 1/2)$。
3. 【易错点】：卡方分布的取值域为 $[0, +\infty)$，不是整个实数轴；自由度 $n$ 是正整数。

#### Student's t 分布 $t_v$ 【必背】
1. 【定义】：设 $v > 0$，若 $X$ 的密度函数为
$$f(x) = \frac{\Gamma(\frac{v+1}{2})}{\sqrt{v\pi}\Gamma(\frac{v}{2})} \left(1 + \frac{x^2}{v}\right)^{-\frac{v+1}{2}}, \quad x \in \mathcal{R}$$
则称 $X$ 服从参数为 $v$ 的 t 分布，记作 $X \sim t_v$。
2. 原理与推导：设 $Z \sim \mathcal{N}(0,1)$，$X \sim \chi^2_n$，且 $Z$ 与 $X$ 独立，令 $T = \frac{Z}{\sqrt{X/n}}$，则 $T \sim t_n$。
3. 【易错点】：t 分布比标准正态分布**重尾**（Heavy-tailed）；当 $v=1$ 时退化为柯西分布（Cauchy 分布）；当 $v$ 很大时，$t_v$ 近似于标准正态分布 $\mathcal{N}(0,1)$。注意构造时分母是 $\sqrt{X/n}$ 而不是 $X/n$。

#### F 分布 $F(m, n)$ 【必背】
1. 【定义】：设 $m, n$ 为正整数，若 $X$ 的密度函数为
$$f(x) = \frac{\Gamma(\frac{m+n}{2})}{\Gamma(\frac{m}{2})\Gamma(\frac{n}{2})} \left(\frac{m}{n}\right)^{m/2} x^{m/2-1} \left(1 + \frac{mx}{n}\right)^{-\frac{m+n}{2}}, \quad x \ge 0$$
则称 $X$ 服从参数为 $(m, n)$ 的 F 分布，记作 $X \sim F(m, n)$。
2. 原理与推导：设两组独立的标准正态变量 $X_1, \dots, X_m \sim \mathcal{N}(0,1)$ 和 $Y_1, \dots, Y_n \sim \mathcal{N}(0,1)$ 相互独立。令
$$F = \frac{\sum_{i=1}^m X_i^2 / m}{\sum_{i=1}^n Y_i^2 / n}$$
则 $F \sim F(m, n)$。
3. 【易错点】：F 分布的自由度有先后顺序，$F(m, n)$ 和 $F(n, m)$ 是不同的分布。取值域为 $[0, +\infty)$。

### （二）随机向量函数的分布（一般方法）

#### 分布函数法（CDF法） 【重点】
1. 【定义】：对于随机向量 $\vec{X} = (X_1, \dots, X_n)$ 的函数 $Z = g(\vec{X})$，先求其累积分布函数 $F_Z(z) = \mathbb{P}(g(\vec{X}) \le z)$，再通过求导得到概率密度函数 $f_Z(z) = F_Z'(z)$ 的方法。
2. 【解题步骤】：
   1. 写出目标变量 $Z$ 的 CDF 定义式：$F_Z(z) = \mathbb{P}(g(\vec{X}) \le z)$。
   2. 将概率转化为联合密度在对应区域上的多重积分。
   3. 计算积分，得到 $F_Z(z)$ 的显式表达式（注意分段讨论）。
   4. 对 $z$ 求导，得到 PDF $f_Z(z)$。
3. 【易错点】：积分区域的边界确定极易出错，必须根据 $z$ 的不同取值范围画出积分区域并分段讨论。

**例题**：[Lecture 07_handout.pdf P10]
**题目**：设 $(X, Y)$ 为单位正方形上的均匀分布，有联合密度 $f(x, y)$。可验证 $X, Y$ 独立。求 $Z = g(X, Y) = Y/X$ 的概率密度函数。
**思路**：使用 CDF 法。先求 $F_Z(z) = \mathbb{P}(Y/X \le z)$，由于 $X, Y \in (0,1)$，需根据 $z$ 与 0, 1 的关系分段讨论积分区域。
**解答**：
当 $z < 0$ 时，$F_Z(z) = 0$。
当 $0 \le z \le 1$ 时，积分区域为 $0 < x < 1, 0 < y < zx$，面积为 $\frac{z}{2}$，故 $F_Z(z) = \frac{z}{2}$。
当 $z > 1$ 时，积分区域为 $0 < x < 1, 0 < y < zx$，但受限于 $y < 1$。此时区域面积为 $1 - \frac{1}{2z}$，故 $F_Z(z) = 1 - \frac{1}{2z}$。
对 $F_Z(z)$ 求导得：
$$f_Z(z) = \begin{cases} \frac{1}{2}, & 0 \le z \le 1 \\ \frac{1}{2z^2}, & z > 1 \\ 0, & \text{其他} \end{cases}$$
**技巧**：对于均匀分布，求 CDF 往往转化为求几何区域的面积（或体积），画图辅助确定边界是关键。

**例题**：[Lecture 07_handout.pdf P11-12]
**题目**：设 $X, Y$ 独立且都服从 $\mathcal{N}(0,1)$。求 $Z = \sqrt{X^2 + Y^2}$ 的分布。
**思路**：使用 CDF 法。由于涉及 $x^2+y^2$，采用极坐标变换计算二重积分。
**解答**：
$(X, Y)$ 联合密度为 $f(x, y) = \frac{1}{2\pi} \exp\left(-\frac{x^2+y^2}{2}\right)$。
对 $z \ge 0$，有：
$$F_Z(z) = \mathbb{P}(X^2 + Y^2 \le z^2) = \iint_{\{x^2+y^2 \le z^2\}} \frac{1}{2\pi} \exp\left(-\frac{x^2+y^2}{2}\right) dx dy$$
作极坐标变换 $x = r\cos\theta, y = r\sin\theta$：
$$F_Z(z) = \int_0^{2\pi} d\theta \int_0^z \frac{1}{2\pi} e^{-r^2/2} r dr = \int_0^z r e^{-r^2/2} dr$$
对 $z$ 求导得密度函数：
$$f_Z(z) = z e^{-z^2/2}, \quad z \ge 0$$
此分布称为 Rayleigh 分布。
**技巧**：遇到含有 $x^2+y^2$ 的正态变量积分，极坐标变换是标准套路，注意雅可比行列式 $r$ 不要漏掉。

### （三）和与差的分布（卷积公式）

#### 卷积公式 【高频】
1. 【定义】：设 $(X, Y)$ 有联合密度 $f(x, y)$，则 $Z = X + Y$ 的密度函数为：
$$f_Z(z) = \int_{-\infty}^{\infty} f(x, z-x) dx$$
特别地，当 $X, Y$ 独立时：
$$f_Z(z) = \int_{-\infty}^{\infty} f_X(x) f_Y(z-x) dx = \int_{-\infty}^{\infty} f_X(z-y) f_Y(y) dy$$
对于差 $V = X - Y$，其密度函数为：
$$f_V(v) = \int_{-\infty}^{\infty} f(x, x-v) dx$$
独立时为 $f_V(v) = \int_{-\infty}^{\infty} f_X(x) f_Y(x-v) dx$。
2. 原理与推导：由 CDF 法 $F_Z(z) = \mathbb{P}(X+Y \le z) = \iint_{x+y \le z} f(x,y) dx dy$，交换积分次序并求导即可得证。离散型场合对应为分布列的卷积：$P_Z(z) = \sum_x P_X(x) P_Y(z-x)$。
3. 【解题步骤】：
   1. 确定被积函数非零的区间，以此确定积分上下限。
   2. 代入卷积公式进行积分。
   3. 检查结果是否分段，并验证 $\int f_Z(z) dz = 1$。
4. 【易错点】：积分上下限的确定。必须让 $f_X(x)$ 和 $f_Y(z-x)$ 同时非零，解不等式组求出 $x$ 的范围。

**例题**：[Lecture 07_handout.pdf P16]
**题目**：设 $X, Y$ 独立，且服从 $\mathcal{U}(0,1)$。求 $Z = X + Y$ 的概率密度。
**思路**：使用独立情况和的卷积公式。注意 $X \sim \mathcal{U}(0,1)$ 的密度用示性函数 $I_{\{0<x<1\}}$ 表示，便于确定积分限。
**解答**：
$f_X(x) = I_{\{0<x<1\}}, f_Y(y) = I_{\{0<y<1\}}$。
$$f_Z(z) = \int_{-\infty}^{\infty} f_X(x) f_Y(z-x) dx = \int_0^1 I_{\{0<z-x<1\}} dx$$
需要满足 $0 < x < 1$ 且 $z-1 < x < z$。
当 $0 \le z \le 1$ 时，$x \in (0, z)$，积分得 $z$。
当 $1 < z \le 2$ 时，$x \in (z-1, 1)$，积分得 $2-z$。
其他情况为 0。即：
$$f_Z(z) = \begin{cases} z, & 0 \le z \le 1 \\ 2-z, & 1 < z \le 2 \\ 0, & \text{其他} \end{cases}$$
**技巧**：均匀分布卷积必产生三角分布。用示性函数写密度函数能极大简化解不等式的过程。

### （四）多维随机变量函数的联合密度

#### 雅可比变换法 【高频】
1. 【定义】：设 $(X, Y)$ 有联合密度 $f(x, y)$，$U = u(X,Y), V = v(X,Y)$。若存在 $n$ 个逆映射 $(x_i, y_i) = (x_i(u,v), y_i(u,v))$，且雅可比行列式绝对值 $J_i = \left| \frac{\partial(x_i, y_i)}{\partial(u, v)} \right| \neq 0$，则 $(U, V)$ 的联合密度为：
$$g(u, v) = \sum_{i=1}^n f(x_i(u,v), y_i(u,v)) \left| \frac{\partial(x_i, y_i)}{\partial(u, v)} \right|$$
2. 原理与推导：基于多元微积分中的变量代换公式。将 $(u,v)$ 空间下的概率微元通过逆映射转换回 $(x,y)$ 空间，乘以雅可比行列式绝对值以保持微元面积相等。
3. 【解题步骤】：
   1. 求反函数：解方程组 $U = u(X,Y), V = v(X,Y)$，得到所有的解 $(x_i, y_i)$。
   2. 求雅可比行列式：计算 $J_i = \left| \frac{\partial(x_i, y_i)}{\partial(u, v)} \right|$。
   3. 代入公式：将各解代入原密度函数并乘以对应的 $|J_i|$，最后求和。
4. 【易错点】：
   - 遗漏多解情况（如平方关系会有正负两个解），必须将所有解的密度相加。
   - 雅可比行列式写反。公式中用的是 $\frac{\partial(x,y)}{\partial(u,v)}$，如果先求出了 $\frac{\partial(u,v)}{\partial(x,y)}$，需取倒数，或直接求逆映射的偏导。

### 本章小结
- **本章知识关系**：本章从一维函数过渡到多维随机向量函数的分布求解。CDF法是根本通法，卷积公式是 CDF 法在求和/差时的特例，雅可比变换法是 CDF 法在求多维联合分布时的推广。三大抽样分布是这些变换技巧在正态分布上的经典应用。
- **本章核心结论**：正态变量的平方和为卡方；正态与卡方之比为 t；两个卡方之比为 F。求和用卷积，求多维联合用雅可比。
- **必背公式**：
  - 卷积公式：$f_Z(z) = \int_{-\infty}^{\infty} f_X(x) f_Y(z-x) dx$
  - 雅可比变换：$g(u, v) = \sum f(x_i, y_i) \left| \frac{\partial(x_i, y_i)}{\partial(u, v)} \right|$
  - 三大分布构造：$X = \sum Z_i^2 \sim \chi^2_n$；$T = \frac{Z}{\sqrt{X/n}} \sim t_n$；$F = \frac{X_1/m}{X_2/n} \sim F(m,n)$
- **【记忆】**：CDF求导通天下，卷积相乘定上下，雅可比换防漏解，正态平方卡方加。
