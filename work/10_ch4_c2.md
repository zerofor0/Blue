#### 4.2 常见离散型随机变量（续）

##### （五）超几何分布
【定义】如果随机变量 $X$ 的概率分布为：
$$P(X = k) = \frac{\binom{M}{k}\binom{N-M}{n-k}}{\binom{N}{n}}, \quad k = 0, 1, \dots, \min(n, M) \quad (4-1)$$
则称 $X$ 服从超几何分布，记作 $X \sim H(n, M, N)$。
- **实际背景**：在包含 $N$ 个元素的总体中，$M$ 个是红的，$N-M$ 个是黑的。任意选取 $n$ 个元素组成一组，取出的红元素个数 $X$ 即服从超几何分布。
- **应用场景**：质量检测（如产品次品率抽样）、捕获-再捕获模型。

【重点】超几何分布与二项分布的关系（近似）
当总体容量 $N$ 充分大时，无放回抽取与有放回抽取的差异变小。若 $\lim_{N \to \infty} \frac{M}{N} = p$，则有：
$$\lim_{N \to \infty} \frac{\binom{M}{k}\binom{N-M}{n-k}}{\binom{N}{n}} = \binom{n}{k} p^k (1-p)^{n-k} \quad (4-2)$$
即当 $N$ 很大而 $n$ 相对较小时，超几何分布 $H(n, M, N)$ 可用二项分布 $B(n, p_N)$ 近似，其中 $p_N = \frac{M}{N}$。

【易错点】
- 混淆超几何分布（无放回抽样）与二项分布（有放回抽样）的应用前提。
- 计算时忽略 $k$ 的取值上限 $\min(n, M)$。

【真题/例题】
【例题：待补充】（对应课件 P26 例5：$N$ 件产品中恰有 $M$ 件次品，从中任取 $n$ 件，求次品数分布。来源：Lecture04_handout.pdf P26）

##### （六）负超几何分布
【定义】如果随机变量 $X$ 的概率分布为：
$$P(X = k) = \frac{\binom{k+r-1}{k}\binom{N-k-r}{M-r}}{\binom{N}{M}}, \quad k = 0, 1, \dots, N-M \quad (4-3)$$
则称 $X$ 服从负超几何分布，记作 $X \sim NH(r, M, N)$。
- **实际背景**：在包含 $N$ 个元素（$M$ 红，$N-M$ 黑）的总体中，无放回抽取直到抽到 $r$ 个红元素为止，此时共抽到的黑元素个数 $X$ 服从该分布。

##### （七）泊松分布
【定义】如果随机变量 $X$ 的概率分布为：
$$P(X = k) = \frac{\lambda^k}{k!} e^{-\lambda}, \quad k = 0, 1, 2, \dots \quad (4-4)$$
则称 $X$ 服从参数为 $\lambda$ 的泊松分布，记作 $X \sim \mathcal{P}(\lambda)$，其中 $\lambda > 0$。
- **应用场景**：常用于描述单位时间或空间内稀有事件发生的次数。如：一段时间内交通事故数、一天中到达的顾客数、办公室收到的电话数等。

【重点】二项分布的泊松近似（泊松定理）
当 $n$ 很大，$p$ 很小，且 $np \approx \lambda$（$\lambda$ 为常数）时，二项分布 $B(n, p)$ 可用泊松分布 $\mathcal{P}(\lambda)$ 近似。
- **推导来源**：将时间 $t$ 等分为 $n$ 段，假设每段内最多发生一次事件且概率独立为 $p_n = \frac{\mu t}{n}$。则发生次数 $X \sim B(n, p_n)$。当 $n \to \infty$ 时：
$$\lim_{n \to \infty} \binom{n}{k} p_n^k (1-p_n)^{n-k} = \frac{(\mu t)^k}{k!} e^{-\mu t} \quad (4-5)$$
令 $\lambda = \mu t$，即得泊松分布。

【易错点】
- 泊松分布的 $k$ 取值是无限的（$k = 0, 1, 2, \dots$），不要像超几何分布那样设置上限。
- 近似计算时，必须满足“$n$ 大 $p$ 小”的条件，否则近似误差较大。

【真题/例题】
【例题：待补充】（对应课件 P31 例6：Rutherford 和 Geiger 观察放射性物质钋放射 $\alpha$ 粒子，验证泊松分布。来源：Lecture04_handout.pdf P31-34）

#### 4.3 连续型随机变量

##### （一）连续型随机变量的定义与密度函数
【定义】设随机变量 $X$，如果存在非负函数 $f(x)$，使得对任意的 $a < b$，有：
$$P(a < X \le b) = \int_a^b f(x) dx \quad (4-6)$$
则称 $X$ 为连续型随机变量，称 $f(x)$ 为 $X$ 的概率密度函数（PDF）。

【重点】概率密度的性质
1. **规范性**：$\int_{-\infty}^{\infty} f(x) dx = 1$。
2. **Borel集概率**：对任意 Borel 集 $A$，有 $P(X \in A) = \int_A f(x) dx$。
3. **单点概率为0**：$P(X = a) = 0$。
   - 推论：$P(a < X \le b) = P(a \le X \le b) = P(a < X < b) = \int_a^b f(x) dx$。

【易错点】
- 密度函数 $f(x)$ 在某点上的值不代表概率，且 $f(x)$ **未必有界**。例如 $f(x) = \frac{1}{2\sqrt{x}}$ 在 $[0,1]$ 上是合法密度函数，但在 $x \to 0$ 时无界。
- “连续型随机变量”是指其分布函数绝对连续，而不是说 $X(\omega)$ 作为样本空间上的函数具有连续性（样本空间无拓扑结构）。

##### （二）均匀分布
【定义】对 $a < b$，若 $X$ 的概率密度为：
$$f(x) = \begin{cases} \frac{1}{b-a}, & x \in (a, b) \\ 0, & x \notin (a, b) \end{cases} \quad (4-7)$$
称 $X$ 服从区间 $(a, b)$ 上的均匀分布，记作 $X \sim \mathcal{U}(a, b)$。
- **扩展**：对于测度有限的 Borel 集 $A$（$m(A) < \infty$），可定义 $A$ 上的均匀分布，其密度为 $f(x) = \frac{1}{m(A)} I_A(x)$。

##### （三）指数分布
【定义】对正常数 $\lambda$，若 $X$ 的概率密度为：
$$f(x) = \begin{cases} \lambda e^{-\lambda x}, & x \ge 0 \\ 0, & x < 0 \end{cases} \quad (4-8)$$
称 $X$ 服从参数为 $\lambda$ 的指数分布，记作 $X \sim \mathcal{E}(\lambda)$。
- **应用场景**：到发生某事件为止所用的时间、仪器使用寿命等。
- **尾部概率**：$P(X \ge a) = e^{-\lambda a}, \forall a \ge 0$。

【必背】无记忆性（定理 2.1）
设 $X$ 是连续型非负随机变量，则 $X$ 服从指数分布的**充要条件**是 $X$ 具有无记忆性，即对任意 $s, t \ge 0$，有：
$$P(X > s+t \mid X > s) = P(X > t) \quad (4-9)$$
- **推导逻辑**：
  必要性：若 $X \sim \mathcal{E}(\lambda)$，则 $P(X > x) = e^{-\lambda x}$，代入条件概率公式即证。
  充分性：令 $G(x) = P(X > x)$，由无记忆性得 $G(s+t) = G(s)G(t)$。由高等数学知识知 $G(x) = e^{-\lambda x}$，进而求导得密度函数 $f(x) = \lambda e^{-\lambda x}$。
- **失效率**：指数分布的失效率为常数 $\lambda$（尺度参数），表示元件无老化。实际中“无老化”是理想近似，常用于描述寿命较长元件的初期阶段。

【易错点】
- 指数分布与几何分布具有平行的性质（几何分布是离散型的“无记忆性”分布，指数分布是连续型的）。
- 指数分布另一种参数化方式为 $f(x) = \frac{1}{\theta} e^{-x/\theta}$，注意 $\lambda$ 与 $\theta$ 的关系（$\theta = 1/\lambda$）。

#### 4.4 本节总结与速查

##### （一）核心结论
1. 超几何分布用于无放回抽样，当总体极大时可近似为二项分布。
2. 泊松分布是二项分布在 $n \to \infty, p \to 0, np \to \lambda$ 时的极限分布，常用于稀有事件计数。
3. 连续型随机变量单点概率必为0，密度函数值非概率，且可以无界。
4. 指数分布是唯一具有无记忆性的连续型分布，常描述寿命或等待时间。

##### （二）必背公式汇总
| 分布名称 | 记号 | 概率分布 / 密度函数 | 适用条件 / 背景 |
| :--- | :--- | :--- | :--- |
| 超几何分布 | $H(n, M, N)$ | $P(X=k) = \frac{\binom{M}{k}\binom{N-M}{n-k}}{\binom{N}{n}}$ | 无放回抽样 |
| 泊松分布 | $\mathcal{P}(\lambda)$ | $P(X=k) = \frac{\lambda^k}{k!} e^{-\lambda}$ | 稀有事件计数，二项分布近似 |
| 均匀分布 | $\mathcal{U}(a, b)$ | $f(x) = \frac{1}{b-a} I_{(a,b)}(x)$ | 区间内等可能取值 |
| 指数分布 | $\mathcal{E}(\lambda)$ | $f(x) = \lambda e^{-\lambda x} I_{[0, \infty)}(x)$ | 寿命、等待时间，无记忆性 |

##### （三）考前速查表
- **超几何 vs 二项**：无放回用超几何，有放回用二项；$N$ 极大时超几何 $\to$ 二项。
- **二项 vs 泊松**：$n$ 大 $p$ 小 $np=\lambda$ 时，二项 $\to$ 泊松。
- **连续型 r.v.**：$P(X=a)=0$，积分求概率，$f(x)$ 可无界。
- **指数分布核心**：无记忆性 $P(X>s+t \mid X>s) = P(X>t)$，充要条件。
