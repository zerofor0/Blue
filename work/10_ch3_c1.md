### 本章总览
- **核心问题**：如何度量随机变量的“中心位置”（期望）与“波动大小”（方差）？如何计算随机变量函数的期望？
- **高频考点排序**：1. 随机变量函数的期望计算；2. 常见分布的期望公式；3. 期望的线性性质应用；4. 对称分布的期望技巧。
- **方法清单**：定义法（求和/积分）、函数期望公式法（免求分布直接积分）、对称化技巧（奇函数积分为零）。

### 一、期望的定义与计算

#### 期望的定义 【必背】
1. 【定义】：随机变量的数学期望（或均值）是随机变量取值的加权平均，权重为对应取值的概率。
   - **离散型**：设 $X$ 有概率分布 $P(X=x_j)$ ($j=1,2,\dots$)，若 $\sum_{j=1}^{\infty} |x_j| P(X=x_j) < +\infty$，则
     $$E(X) = \sum_{j=1}^{\infty} x_j P(X=x_j) \tag{8-1}$$
   - **连续型**：设 $X$ 有概率密度 $f(x)$，若 $\int_{-\infty}^{\infty} |x| f(x) dx < +\infty$，则
     $$E(X) = \int_{-\infty}^{\infty} x f(x) dx \tag{8-2}$$
2. 原理与推导：期望的物理直观是“质心”或“重心”。离散型是质点系的质心，连续型是连续物体的重心。前提条件是绝对收敛（即积分或求和必须绝对可积），否则期望不存在。只取有限个值的随机变量期望总是存在的。
3. 【解题步骤】：
   1. 判断随机变量类型（离散型或连续型）。
   2. 写出对应的概率分布（PMF）或概率密度（PDF）。
   3. 代入公式 (8-1) 或 (8-2) 进行求和或积分计算。
4. 【易错点】：
   - 计算前不验证绝对收敛条件，对条件发散的级数强行求和。
   - 混淆随机变量的取值与概率权重。
5. 例题：

**例题**：赌金分配问题
**题目**：甲、乙对赌，水平相当，先赢满5局的人将获得全部赌金100个金币。当甲赢了4局，乙赢了3局时，被迫中断赌局。该如何分配赌金？ [Lecture08_handout.pdf P3]
**思路**：计算甲乙各自预期收益（期望）。甲赢只需1局，乙赢需要2局。列举后续可能情况及概率，按期望定义计算。
**解答**：甲的收益 $X_{甲}$ 取值为 100 和 0。甲赢下一局的概率为 1/2，乙赢为 1/2。
若甲赢下一局（概率 1/2），甲获得 100。
若乙赢下一局（概率 1/2），比分 4:4。此时再下一局甲赢概率 1/2（获 100），乙赢概率 1/2（甲获 0）。
所以甲最终获胜概率为 $1/2 + 1/2 \times 1/2 = 3/4$。
$$E(X_{甲}) = 100 \times \frac{3}{4} + 0 \times \frac{1}{4} = 75$$
乙的期望收益为 $100 - 75 = 25$。故应按 3:1 分配赌金。
**技巧**：期望本质是加权平均，解决未定博弈问题时，计算各方的预期收益是最公平的分配方式。

#### 常见分布的数学期望 【必背】
1. 【定义】：各类常见离散与连续分布的期望公式。
2. 原理与推导：均由期望的定义直接推导，常利用归一化性质（即总概率为1）或变量代换简化计算。
3. 【解题步骤】：直接套用公式，或在使用公式前先确认分布的参数。
4. 【易错点】：指数分布与 Gamma 分布的参数混淆（率参数 vs 尺度参数）。
5. 常见分布期望公式表：

| 分布名称 | 记号 | 概率分布 / 密度函数 | 期望 $E(X)$ |
| :--- | :--- | :--- | :--- |
| 两点分布 | $B(1, p)$ | $P(X=1)=p, P(X=0)=1-p$ | $p$ |
| 二项分布 | $B(n, p)$ | $P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$ | $np$ |
| 泊松分布 | $\mathcal{P}(\lambda)$ | $P(X=k) = \frac{\lambda^k}{k!} e^{-\lambda}$ | $\lambda$ |
| 几何分布 | $G(p)$ | $P(X=k) = p(1-p)^{k-1}$ | $1/p$ |
| 均匀分布 | $U(a, b)$ | $f(x) = \frac{1}{b-a} I_{(a,b)}(x)$ | $\frac{a+b}{2}$ |
| 指数分布 | $\varepsilon(\lambda)$ | $f(x) = \lambda e^{-\lambda x}, x>0$ | $\frac{1}{\lambda}$ |
| Gamma分布 | $\Gamma(\alpha, \beta)$ | $f(x) = \frac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x}, x>0$ | $\frac{\alpha}{\beta}$ |
| 正态分布 | $\mathcal{N}(\mu, \sigma^2)$ | $f(x) = \frac{1}{\sqrt{2\pi}\sigma} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$ | $\mu$ |

#### 对称分布的期望计算技巧 【重点】
1. 【定义】：若随机变量 $X$ 的概率密度 $f(x)$ 关于 $x=\mu$ 对称，即 $f(\mu+x) = f(\mu-x)$，且期望存在，则 $E(X) = \mu$。
2. 原理与推导：
   $$E(X) = \int_{-\infty}^{\infty} x f(x) dx = \int_{-\infty}^{\infty} \mu f(x) dx + \int_{-\infty}^{\infty} (x-\mu) f(x) dx$$
   令 $t = x-\mu$，第二项积分变为 $\int_{-\infty}^{\infty} t f(t+\mu) dt$。由于 $f(t+\mu)$ 是偶函数，$t f(t+\mu)$ 是奇函数，在对称区间积分为 0。故 $E(X) = \mu \int f(x) dx = \mu$。
3. 【解题步骤】：
   1. 观察密度函数是否具有对称性。
   2. 找到对称中心 $\mu$。
   3. 直接得出期望为 $\mu$。
4. 【易错点】：误认为所有钟形分布期望都是 0，需确认对称中心的位置。

### 二、随机变（向）量函数的数学期望

#### 随机变（向）量函数的数学期望 【重点】
1. 【定义】：设 $\boldsymbol{X} = (X_1, \dots, X_n)$ 是随机向量，若实函数 $g(\boldsymbol{x})$ 使得 $\int_{\mathbb{R}^n} |g(\boldsymbol{x})| dF(\boldsymbol{x}) < \infty$，则 $Y = g(\boldsymbol{X})$ 的数学期望为：
   $$E(Y) = E[g(\boldsymbol{X})] = \int_{\mathbb{R}^n} g(\boldsymbol{x}) dF(\boldsymbol{x}) \tag{8-3}$$
   离散型为 $\sum g(x_j) P(X=x_j)$，连续型为 $\int g(x) f(x) dx$。
2. 原理与推导：该定理的核心在于，计算 $Y=g(X)$ 的期望时，**不需要先求出 $Y$ 的分布**，直接利用 $X$ 的原分布和函数 $g$ 进行积分即可。这大大简化了计算过程。
3. 【解题步骤】：
   1. 明确原随机变量 $X$ 的分布（PMF 或 PDF）。
   2. 写出目标函数 $g(x)$。
   3. 将 $g(x)$ 与 $X$ 的概率或密度相乘后求和或积分。
4. 【易错点】：
   - 画蛇添足：先求 $Y=g(X)$ 的密度，再用定义求 $E(Y)$，导致计算量爆炸甚至算错。
   - 忽略绝对可积条件。
5. 例题：

**例题**：随机变量函数的期望
**题目**：设 $X \sim U(0, \pi/2)$，计算 $E(\cos X)$。 [Lecture08_handout.pdf P22]
**思路**：利用随机变量函数的期望公式，无需先求 $Y=\cos X$ 的密度，直接用 $X$ 的密度乘以 $\cos x$ 积分。
**解答**：$X$ 的密度函数为 $f(x) = \frac{2}{\pi}$ ($0 < x < \pi/2$)。
$$E(\cos X) = \int_{-\infty}^{\infty} \cos x f(x) dx = \int_{0}^{\pi/2} \cos x \frac{2}{\pi} dx = \frac{2}{\pi} [\sin x]_0^{\pi/2} = \frac{2}{\pi}$$
**技巧**：计算 $E(g(X))$ 时，直接对 $g(x)$ 乘以 $X$ 的密度积分，避免求 $g(X)$ 的分布。

- **核心结论**：期望是加权平均；计算 $E(g(X))$ 无需先求 $g(X)$ 的分布，直接用 $X$ 的分布积分即可。
