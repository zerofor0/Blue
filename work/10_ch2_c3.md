#### 本章总览
- 核心问题：如何利用条件信息预测随机变量？如何分解无条件方差？
- 高频考点排序：全方差法则计算 > 最佳预测的证明与理解 > 条件方差的定义与计算 > 条件期望的正交性质。
- 方法清单：全方差公式法（求复杂随机变量的方差）、均方误差最小化法（证明最佳预测）、加减项凑零均值法（化简二次型期望）。

#### 条件期望的性质 【重点】
【定义】
条件期望 $E(X|Y)$ 具有正交性，即残差 $X - E(X|Y)$ 与任意 $Y$ 的函数 $h(Y)$ 的乘积的期望为 0。
原理与推导：
设 $X, Y$ 是随机变量，$h(x)$ 是实函数，若 $E(X^2) < \infty, E(h^2(Y)) < \infty$，则：
$$E[(X - E(X|Y))h(Y)] = 0 \tag{9-1}$$
证明：
1. 由 Cauchy-Schwarz 不等式得 $E|Xh(Y)| \le \sqrt{E(X^2)E(h^2(Y))} < \infty$，保证期望存在。
2. 利用条件期望的平滑性质：
$$E[(X - E(X|Y))h(Y)] = E[Xh(Y)] - E[E(X|Y)h(Y)]$$
因为 $h(Y)$ 在给定 $Y$ 时为常数，可放入条件期望内部：
$$E[E(X|Y)h(Y)] = E[E(Xh(Y)|Y)] = E[Xh(Y)]$$
所以原式 $= E[Xh(Y)] - E[Xh(Y)] = 0$。
【易错点】
容易误认为 $E(X|Y)$ 与 $h(Y)$ 独立，实际上它们只是不相关（正交）。残差 $X - E(X|Y)$ 与 $\sigma(Y)$ 生成的任何元素正交。

**例题**：
**题目**：设 $X, Y$ 是随机变量, $h(x)$ 是实函数, 若 $E(X^2) < \infty, E(h^2(Y)) < \infty$, 证明 $E[(X - E(X|Y))h(Y)] = 0$。[Lecture 09_handout.pdf P39]
**思路**：利用条件期望的平滑性质，将 $h(Y)$ 移入条件期望内部，证明两项相抵消。
**解答**：
$$E[(X - E(X|Y))h(Y)] = E[Xh(Y)] - E[E(X|Y)h(Y)]$$
由于 $h(Y)$ 是 $Y$ 的函数：
$$E[E(X|Y)h(Y)] = E[E(Xh(Y)|Y)] = E[Xh(Y)]$$
代回原式得：
$$E[Xh(Y)] - E[Xh(Y)] = 0$$
**技巧**：在涉及条件期望的乘积时，若其中一项是条件变量的函数，常利用平滑性 $E[E(X|Y)h(Y)] = E[Xh(Y)]$ 进行化简。

#### 条件期望的意义（最佳预测） 【高频】
【定义】
在均方误差意义下，$m(Y) = E(X|Y)$ 是 $X$ 的最佳预测。即对任何实函数 $g(Y)$，$m(Y)$ 使得 $E[(X - g(Y))^2]$ 最小。
原理与推导：
设 $E(X^2) < \infty, m(Y) = E(X|Y)$，对任意实函数 $g(Y)$，有：
$$E[(X - m(Y))^2] \le E[(X - g(Y))^2] \tag{9-2}$$
等号成立当且仅当 $g(Y) = m(Y)$ a.s.
证明：
令 $h(Y) = m(Y) - g(Y)$，则：
$$E[(X - g(Y))^2] = E[(X - m(Y) + h(Y))^2]$$
展开得：
$$E[(X - m(Y))^2] + E[h^2(Y)] + 2E[(X - m(Y))h(Y)]$$
由正交性质式 (9-1) 可知交叉项 $E[(X - m(Y))h(Y)] = 0$，故：
$$E[(X - g(Y))^2] = E[(X - m(Y))^2] + E[h^2(Y)] \ge E[(X - m(Y))^2]$$
等号成立当且仅当 $E[h^2(Y)] = 0$，即 $g(Y) = m(Y)$ a.s.
【解题步骤】
1. 判定预测目标：寻找使均方误差 $E[(X - g(Y))^2]$ 最小的函数 $g(Y)$。
2. 直接写出结论：$g(Y) = E(X|Y)$。
3. 若求最佳线性预测，则需利用协方差和期望计算 $aY+b$ 形式（正态分布下两者等价）。
【易错点】
最佳预测不一定是线性预测，但在 $(X, Y)$ 服从二元正态分布时，$E(X|Y)$ 是 $Y$ 的线性组合，此时最佳预测等于最佳线性预测。

**例题**：
**题目**：设 $X, Y \sim \mathcal{N}(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$，求 $X$ 的最佳预测，并说明其与最佳线性预测的关系。[Lecture 09_handout.pdf P43]
**思路**：利用二元正态分布的条件期望公式，观察其函数形式。
**解答**：
对于二元正态分布，条件期望为：
$$E(X|Y) = \mu_1 + \rho \frac{\sigma_1}{\sigma_2} (Y - \mu_2)$$
该表达式是 $Y$ 的线性组合。因此对正态分布而言，最佳预测等于最佳线性预测。
**技巧**：记住二元正态分布的条件期望是线性的，这是最佳预测等于最佳线性预测的典型特例。

#### 条件方差的定义与性质 【必背】
【定义】
给定 $Y$ 下 $X$ 的条件方差定义为：
$$\text{var}(X|Y) \coloneqq E[(X - E(X|Y))^2 | Y] \tag{9-3}$$
在 $Y=y$ 的条件下：
$$\text{var}(X|Y=y) = E[(X - E(X|Y=y))^2 | Y=y]$$
原理与推导：
条件方差反映了在已知 $Y$ 的条件下，$X$ 的波动程度。类似于无条件方差，它也可以化简为：
$$\text{var}(X|Y) = E(X^2|Y) - [E(X|Y)]^2 \tag{9-4}$$
【解题步骤】
1. 先求条件期望 $E(X|Y=y)$。
2. 再求条件二阶矩 $E(X^2|Y=y)$。
3. 作差得到条件方差 $\text{var}(X|Y=y)$。
4. 若需求随机化的条件方差 $\text{var}(X|Y)$，将步骤3结果中的 $y$ 替换为 $Y$。
【易错点】
条件方差 $\text{var}(X|Y)$ 是随机变量 $Y$ 的函数，而 $\text{var}(X|Y=y)$ 是一个确定的数值。

#### 全方差法则 【高频】
【定义】
全方差法则描述了无条件方差与条件方差、条件期望方差之间的关系：
$$\text{var}(X) = E[\text{var}(X|Y)] + \text{var}(E(X|Y)) \tag{9-5}$$
原理与推导：
证明过程运用了"加减一项"的技巧：
$$\text{var}(X) = E[(X - E(X))^2]$$
将 $E(X|Y)$ 加减进去：
$$E[((X - E(X|Y)) + (E(X|Y) - E(X)))^2]$$
展开为三项：
$$E[(X - E(X|Y))^2] + E[(E(X|Y) - E(X))^2] + 2E[(X - E(X|Y))(E(X|Y) - E(X))]$$
由于 $E(X) = E[E(X|Y)]$，且由正交性知交叉项为 0，故：
$$E[E[(X - E(X|Y))^2 | Y]] + \text{var}(E(X|Y)) = E[\text{var}(X|Y)] + \text{var}(E(X|Y))$$
【解题步骤】
1. 判断随机变量是否为随机个随机变量的和（如 $X = \sum_{i=1}^Y X_i$）。
2. 先求条件期望 $E(X|Y)$ 和条件方差 $\text{var}(X|Y)$。
3. 利用全期望公式求 $E(X) = E[E(X|Y)]$。
4. 利用全方差公式求 $\text{var}(X) = E[\text{var}(X|Y)] + \text{var}(E(X|Y))$。
【易错点】
容易忘记全方差公式中的第二项 $\text{var}(E(X|Y))$，误认为 $\text{var}(X) = E[\text{var}(X|Y)]$。

**例题**：
**题目**：设 $Y$ 是我校明年选修概率论课程的学生人数, $X_i$ 是第 $i$ 位同学在学习后5年内写论文的篇数, $X$ 是该段时间内这些同学写的论文的总共篇数. 假设每位同学的论文数 $X_i$ 是相互独立同分布且与选课人数 $Y$ 也相互独立. 如果 $E(X_1), E(Y), \text{var}(X_1), \text{var}(Y)$ 已知, 求 $X$ 的期望与方差。[Lecture 09_handout.pdf P49]
**思路**：总论文数 $X = X_1 + \dots + X_Y$ 是随机个独立同分布随机变量的和，使用全期望和全方差公式求解。
**解答**：
已知 $X = X_1 + \dots + X_Y$，在给定 $Y=n$ 的条件下：
$$E(X|Y=n) = nE(X_1)$$
$$\text{var}(X|Y=n) = n\text{var}(X_1)$$
将 $n$ 替换为 $Y$：
$$E(X|Y) = YE(X_1)$$
$$\text{var}(X|Y) = Y\text{var}(X_1)$$
由全期望公式：
$$E(X) = E[E(X|Y)] = E[YE(X_1)] = E(Y)E(X_1)$$
由全方差公式：
$$\text{var}(X) = E[\text{var}(X|Y)] + \text{var}(E(X|Y))$$
代入计算：
$$\text{var}(X) = E[Y\text{var}(X_1)] + \text{var}(YE(X_1)) = E(Y)\text{var}(X_1) + (E(X_1))^2\text{var}(Y)$$
**技巧**：处理"随机多个独立随机变量之和"（如复合分布）时，全期望和全方差公式是标准工具，务必先求条件期望和条件方差。

#### 本章小结
- 本章知识关系：本章从条件期望的定义出发，推导了其正交性质，进而证明了条件期望在均方误差意义下的最佳预测性质。随后引入条件方差，并通过全方差法则将条件方差与无条件方差联系起来，形成了从条件分布到无条件数字特征的完整计算闭环。
- 本章核心结论：
  - 条件期望 $E(X|Y)$ 是所有 $Y$ 的函数中对 $X$ 的最佳均方预测。
  - 全方差公式将方差分解为"条件方差的期望"与"条件期望的方差"之和。
- 必背公式：
  - 正交性：$E[(X - E(X|Y))h(Y)] = 0$
  - 条件方差：$\text{var}(X|Y) = E[(X - E(X|Y))^2 | Y] = E(X^2|Y) - [E(X|Y)]^2$
  - 全方差法则：$\text{var}(X) = E[\text{var}(X|Y)] + \text{var}(E(X|Y))$
- 【记忆】期望平滑（全期望），方差分解（全方差），最佳预测是条件期望。
