#### 概率的基本性质 【必背】
【定义】概率的基本性质是由概率的三条公理（非负性、规范性、可列可加性）直接推导出的常用结论，用于复杂事件概率的计算与边界估计。

原理与推导：
- **性质4（差的概率）**：若 $A \subset B$，则 $P(B \setminus A) = P(B) - P(A)$。
  证明：因为 $B = A \cup (B \setminus A)$ 且 $A \cap (B \setminus A) = \emptyset$，由可加性得 $P(B) = P(A) + P(B \setminus A)$。
  推论（单调性）：若 $A \subset B$，则 $P(A) \le P(B)$；且对任意事件 $A$，有 $0 \le P(A) \le 1$。
- **性质5（加法公式）**：对任意事件 $A, B$，有 $P(A \cup B) = P(A) + P(B) - P(A \cap B)$。
  推论（有限次可加性/Boole不等式）：$P(\bigcup_{i=1}^n A_i) \le \sum_{i=1}^n P(A_i)$。
- **性质6（容斥原理）**：对任意 $n$ 个事件 $A_1, \dots, A_n$，有
  $$P\left(\bigcup_{i=1}^n A_i\right) = \sum_{1 \le i \le n} P(A_i) - \sum_{1 \le i < j \le n} P(A_i \cap A_j) + \sum_{1 \le i < j < k \le n} P(A_i \cap A_j \cap A_k) - \dots + (-1)^{n-1} P(A_1 \cap \dots \cap A_n)$$
- **性质7（Bonferroni 不等式）**：$P(\bigcup_{i=1}^n A_i) \ge \sum_{i=1}^n P(A_i) - \sum_{1 \le i < j \le n} P(A_i \cap A_j)$。
  Kounias 不等式：$P(\bigcup_{i=1}^n A_i) \le \min_k \left\{ \sum_{i=1}^n P(A_i) - \sum_{i: i \ne k} P(A_i \cap A_k) \right\}$。
- **性质8（可列次可加性）**：对可列个事件 $A_i, i=1,2,\dots$，有 $P(\bigcup_{i=1}^\infty A_i) \le \sum_{i=1}^\infty P(A_i)$。
  证明思路：令 $B_1 = A_1, B_n = A_n \setminus \bigcup_{i=1}^{n-1} A_i$，则 $B_i$ 互斥且 $\bigcup A_i = \bigcup B_i$，由 $B_n \subset A_n$ 及单调性可证。
- **性质9（交的下界）**：$P(\bigcap_{i=1}^\infty A_i) \ge 1 - \sum_{i=1}^\infty P(A_i^c)$。特别地，$P(A_1 \cap A_2) \ge 1 - P(A_1^c) - P(A_2^c)$。

【解题步骤】
1. 识别事件间的包含、互斥、独立关系。
2. 将复杂事件拆解为互斥事件的并集，或利用对立事件求补。
3. 求多事件并集概率时，优先使用容斥原理；若仅需估计范围，使用 Bonferroni 或 Boole 不等式。

【易错点】
- 容易混淆有限可加性和可列可加性的前提条件（必须互斥）。
- Bonferroni 不等式给出的是并集概率的**下界**，Boole 不等式给出的是**上界**，不要混淆不等号方向。

#### 样本空间的选择 【重点】
【定义】随机抽样是指在含有 $M$ 个不同球的箱子中抽取 $n$ 个球的方法。根据“是否放回”和“是否有序”，分为四种基本抽样方式。

原理与推导：
- **排列与组合基础**：
  - $n$ 个对象的全排列数：$n!$
  - $n$ 个对象中取 $k$ 个的排列数：$P_n^k = \frac{n!}{(n-k)!}$
  - $n$ 个对象中取 $k$ 个的组合数：$\binom{n}{k} = \frac{n!}{k!(n-k)!}$
- **四种抽样方式的样本空间大小**（设从 $M$ 个球中抽 $n$ 次）：
  1. **放回有序**：样本为向量 $(a_1, \dots, a_n)$，$a_i \in \{1,\dots,M\}$，总数 $\#\Omega = M^n$。
  2. **放回无序**：样本为集合 $[a_1, \dots, a_n]$，$a_i \in \{1,\dots,M\}$，总数 $\#\Omega = \binom{M+n-1}{n}$。
  3. **不放回有序**：样本为向量 $(a_1, \dots, a_n)$ 且元素互异，总数 $\#\Omega = P_M^n = M(M-1)\dots(M-n+1)$。
  4. **不放回无序**：样本为集合 $[a_1, \dots, a_n]$ 且元素互异，总数 $\#\Omega = \binom{M}{n}$。
- **与质点分配模型的对应**：
  - 有序抽样 $\Leftrightarrow$ 质点可辨；无序抽样 $\Leftrightarrow$ 质点不可辨。
  - 放回抽样 $\Leftrightarrow$ 箱中可容纳任意多质点（无抑制）；不放回抽样 $\Leftrightarrow$ 箱中最多容纳一个质点（有抑制）。

【解题步骤】
1. 判断抽样是否放回（决定同一元素能否被重复抽取）。
2. 判断抽样是否有序（决定 $(1,2)$ 和 $(2,1)$ 是否视为同一基本事件）。
3. 套用对应公式计算样本空间总数或有利事件数。

【易错点】
易混概念对比表：

| 抽样方式 | 是否放回 | 是否有序 | 样本空间大小 | 质点分配模型对应 |
| :--- | :---: | :---: | :---: | :--- |
| 放回有序 | 是 | 是 | $M^n$ | 质点可辨，箱容量不限 |
| 放回无序 | 是 | 否 | $\binom{M+n-1}{n}$ | 质点不可辨，箱容量不限 |
| 不放回有序 | 否 | 是 | $P_M^n$ | 质点可辨，箱容量为1 |
| 不放回无序 | 否 | 否 | $\binom{M}{n}$ | 质点不可辨，箱容量为1 |

#### 古典概型与几何概型 【高频】
【定义】古典概型解题技巧是指在计算等可能事件概率时，为避免复杂组合计算而采用的直观化、简单化分析方法。

原理与推导：
- **常用小技巧**：
  1. 不要失去常识，也不要过分依赖直觉。
  2. 用最简单情形（如 $n=1,2$）验证公式。
  3. 将物体编号，使事件更加明确。
  4. 情景证明：通过构造实际计数情景来证明组合恒等式。
- **情景证明思考题**：
  - $\binom{m}{n} = \binom{m-1}{n-1} + \binom{m-1}{n}$（从 $m$ 个球中取 $n$ 个，等价于固定某球“取”或“不取”两种情况之和）。
  - $\binom{m}{n} = \sum_{k=n-1}^{m-1} \binom{k}{n-1}$。
  - 多项式系数：有 $m$ 个球，共 $r$ 种颜色，同色球不可辨。这 $m$ 个球的排列数为 $\frac{m!}{m_1! m_2! \dots m_r!}$（等价于把 $m$ 个球放入 $r$ 个盒子，使第 $j$ 个盒子有 $m_j$ 个球）。

【解题步骤】
1. 遇到复杂组合数或概率公式，尝试用具体的小数字代入验证逻辑。
2. 寻找公式两边的实际计数意义，构造对应的“取球”或“分配”情景进行证明。

【易错点】
- 忽略等可能性前提，导致样本空间划分不均。
- 情景证明时逻辑不严密，未完全对应公式两端的所有情况。

#### 本章小结
- **本章知识关系**：由概率公理出发，推导了概率的单调性、加法公式、容斥原理及一系列不等式，为复杂事件概率计算提供理论工具；随机抽样理论将样本空间的构建系统化，是古典概型计算的核心基础。
- **本章核心结论**：概率的容斥原理与四种随机抽样方式的样本空间计算公式。
- **必背公式**：
  - $P(A \cup B) = P(A) + P(B) - P(A \cap B)$
  - $P(\bigcup_{i=1}^n A_i) = \sum P(A_i) - \sum P(A_i \cap A_j) + \dots + (-1)^{n-1} P(\bigcap A_i)$
  - 放回有序：$M^n$；放回无序：$\binom{M+n-1}{n}$；不放回有序：$P_M^n$；不放回无序：$\binom{M}{n}$
- **【记忆】** 抽样看放回（定重复）看有序（定排列），性质靠公理（拆互斥求补）。
