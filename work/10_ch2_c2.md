### 本章总览
- 核心问题：如何定义和计算条件在随机变量上的期望？条件期望有哪些性质和计算法则？
- 高频考点排序：重期望法则及其应用、条件期望的计算与性质、全期望定理。
- 方法清单：定义法计算条件期望、利用重期望法则求期望、利用条件期望性质化简表达式。

#### 全期望定理 【高频】
【定义】设 $A_1, \dots, A_n$ 构成样本空间 $\Omega$ 的一个分割，则全期望定理公式为：
$$E(X) = \sum_{i=1}^n P(A_i)E(X|A_i) \tag{9-1}$$
原理与推导：
对于离散型随机变量，全概率公式给出 $p_X(x) = \sum_{i=1}^n p_{X|A_i}(x)P(A_i)$。将其代入期望定义：
$$E(X) = \sum_x x p_X(x) = \sum_x x \left\{ \sum_{i=1}^n p_{X|A_i}(x)P(A_i) \right\} = \sum_{i=1}^n \left( \sum_x x p_{X|A_i}(x) \right) P(A_i) = \sum_{i=1}^n E(X|A_i)P(A_i)$$
连续型同理，将概率质量函数替换为概率密度函数即可。
【解题步骤】
1. 寻找样本空间的一个分割 $A_1, \dots, A_n$（通常对应不同的情况或阶段）。
2. 计算每个事件发生的概率 $P(A_i)$。
3. 计算在每个事件发生条件下的条件期望 $E(X|A_i)$。
4. 代入全期望公式求和。
【易错点】
- 分割事件必须互斥且并集为全集。
- 在递归问题中（如几何分布、密室逃脱），$E(X|A_i)$ 可能包含 $E(X)$ 本身，需解方程求解。

**题目**：设 $X \sim G(p)$（几何分布），证明 $E(X) = 1/p$。 [Lecture 09_handout.pdf P22]
**思路**：利用全期望定理，将样本空间分割为 $A_1 = \{X=1\}$（首次成功）和 $A_2 = \{X>1\}$（首次失败）。
**解答**：
由全期望公式：
$$E(X) = E(X|A_1)P(A_1) + E(X|A_2)P(A_2)$$
其中 $P(A_1) = p$，$E(X|A_1) = 1$；$P(A_2) = 1-p$。
对于 $A_2$，由于几何分布的无记忆性，$X|X>1$ 相当于 $1 + X'$，其中 $X'$ 与 $X$ 同分布：
$$E(X|X>1) = E(X-1|X-1>0) + 1 = E(X) + 1$$
代入得：
$$E(X) = 1 \cdot p + (E(X) + 1)(1-p) = p + E(X)(1-p) + 1-p = E(X)(1-p) + 1$$
解得 $E(X) = 1/p$。
**技巧**：利用无记忆性构造递推关系是求解几何分布期望的标准方法。

#### 条件期望 $E(X|Y)$ 的定义 【必背】
【定义】设 $X, Y$ 是随机向量，$E|X| < \infty$。给定 $Y=y$ 条件下 $X$ 的期望记为 $m(y) = E(X|Y=y)$。称随机变量 $m(Y)$ 为给定 $Y$ 时 $X$ 的条件期望，记作 $E(X|Y)$。
- 离散型：$E(X|Y=y) = \sum_{i=1}^\infty x_i P(X=x_i|Y=y)$
- 连续型：$E(X|Y=y) = \int_{-\infty}^\infty x f_{X|Y}(x|y)dx$
原理与推导：
条件期望 $E(X|Y=y)$ 是在给定 $Y=y$ 的条件下，用条件分布算出的期望。由于 $y$ 是变量，$m(y) = E(X|Y=y)$ 是关于 $y$ 的函数。将随机变量 $Y$ 代入，得到 $m(Y)$，即 $E(X|Y)$，它是一个随机变量，其随机性完全来源于 $Y$。
【解题步骤】
1. 求出给定 $Y=y$ 时的条件分布（条件概率或条件密度）。
2. 利用期望定义计算 $m(y) = E(X|Y=y)$。
3. 将 $y$ 替换为随机变量 $Y$，得到 $E(X|Y) = m(Y)$。
【易错点】
- $E(X|Y=y)$ 是一个确定的数，而 $E(X|Y)$ 是一个随机变量。
- 计算时不需要 $X, Y$ 同为离散或连续，只要对应的条件期望有合理定义即可。

**题目**：设 $X, Y$ 独立同分布，且 $X \sim P(\lambda)$，求 $E(X+Y|X=x)$ 和 $E(X|X+Y)$。 [Lecture 09_handout.pdf P29]
**思路**：第一问直接利用独立性和线性性质；第二问需要求条件分布或利用对称性。
**解答**：
1. 求 $E(X+Y|X=x)$：
由于 $X, Y$ 独立，给定 $X=x$，$Y$ 的分布不变。
$$E(X+Y|X=x) = E(x+Y|X=x) = x + E(Y) = x + \lambda$$
2. 求 $E(X|X+Y)$：
设 $S = X+Y$，则 $S \sim P(2\lambda)$。
给定 $S=s$，$X$ 的条件分布为二项分布 $B(s, 1/2)$。
因此 $E(X|S=s) = s/2$。
将 $s$ 替换为 $S$，得：
$$E(X|X+Y) = \frac{X+Y}{2}$$
**技巧**：对于独立同分布的随机变量和，条件期望常利用对称性，即 $E(X|X+Y) = E(Y|X+Y) = (X+Y)/2$。

**题目**：用 $X, Y$ 分别表示中国成年人的身高、足长。设 $(X, Y) \sim \mathcal{N}(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$，求 $E(X|Y)$。 [Lecture 09_handout.pdf P30]
**思路**：利用二元正态分布的条件分布性质。
**解答**：
给定 $Y=y$ 下，条件分布为：
$$X|Y=y \sim \mathcal{N}\left(\mu_1 + \rho \frac{\sigma_1}{\sigma_2}(y-\mu_2), (1-\rho^2)\sigma_1^2\right)$$
于是条件期望为：
$$E(X|Y=y) = \mu_1 + \rho \frac{\sigma_1}{\sigma_2}(y-\mu_2)$$
将 $y$ 替换为 $Y$，得：
$$E(X|Y) = \mu_1 + \rho \frac{\sigma_1}{\sigma_2}(Y-\mu_2)$$
**技巧**：二元正态分布的条件期望是线性函数，这是统计学中线性回归模型的理论基础。

#### 重期望法则 【高频】
【定义】设 $X, Y$ 是随机变量，则：
$$E(E(X|Y)) = E(X) \tag{9-2}$$
原理与推导（离散情形）：
$$E(E(X|Y)) = E(m(Y)) = \sum_y m(y)P(Y=y) = \sum_y E(X|Y=y)P(Y=y)$$
$$= \sum_y \left( \sum_x x P(X=x|Y=y) \right) P(Y=y) = \sum_x x \sum_y P(X=x|Y=y)P(Y=y)$$
$$= \sum_x x P(X=x) = E(X)$$
【解题步骤】
1. 当直接求 $E(X)$ 困难时，寻找一个与 $X$ 相关的随机变量 $Y$。
2. 先求条件期望 $E(X|Y=y)$，得到 $E(X|Y)$。
3. 再对 $E(X|Y)$ 求期望，利用 $E(E(X|Y)) = E(X)$ 求解。
【易错点】
- 内层期望是对 $X$ 求，外层期望是对 $Y$ 求。
- 常用于具有层次结构或分阶段决策的问题（如贝叶斯网络）。

**题目**：密室逃脱问题。玩家有三个门可选：第一个门走3小时到出口；第二个门走5小时回原地；第三个门走7小时回原地。每次等可能选一个门。求到达出口的平均时间。 [Lecture 09_handout.pdf P33]
**思路**：设 $X$ 为到达出口时间，$Y$ 为首次选的门。利用全期望公式（重期望法则离散形式）。
**解答**：
$P(Y=1)=P(Y=2)=P(Y=3)=1/3$。
$$E(X) = \sum_{i=1}^3 E(X|Y=i)P(Y=i) = \frac{1}{3} \left[ E(X|Y=1) + E(X|Y=2) + E(X|Y=3) \right]$$
分析条件期望：
- $Y=1$：直接出去，$E(X|Y=1) = 3$。
- $Y=2$：走5小时回原地，相当于重新开始，$E(X|Y=2) = 5 + E(X)$。
- $Y=3$：走7小时回原地，相当于重新开始，$E(X|Y=3) = 7 + E(X)$。
代入得：
$$E(X) = \frac{1}{3} \left[ 3 + (5+E(X)) + (7+E(X)) \right] = \frac{15 + 2E(X)}{3}$$
解方程得 $E(X) = 15$。
**技巧**：对于“回到原点重新开始”的递归问题，设总期望为 $E(X)$，在条件期望中加上 $E(X)$ 构成方程。

**题目**：设计算机环境指标 $Y \sim \Gamma(\alpha, \beta)$，已知 $Y=y$ 时，软件寿命 $X \sim \mathcal{E}(y)$。求 $E(X|Y), E(Y|X)$ 和 $E(X)$。 [Lecture 09_handout.pdf P35]
**思路**：综合考查条件期望计算、边缘分布求法及重期望法则。
**解答**：
1. 求 $E(X|Y)$：
给定 $Y=y$，$X \sim \mathcal{E}(y)$，其密度为 $f_{X|Y}(x|y) = ye^{-xy}$ ($x>0$)。
$$E(X|Y=y) = \int_0^\infty x ye^{-xy} dx = \frac{1}{y}$$
所以 $E(X|Y) = \frac{1}{Y}$。
2. 求 $E(Y|X)$：
先求联合密度 $f(x,y) = f_{X|Y}(x|y)f_Y(y) = ye^{-xy} \cdot \frac{\beta^\alpha}{\Gamma(\alpha)} y^{\alpha-1} e^{-\beta y} = \frac{\beta^\alpha}{\Gamma(\alpha)} y^\alpha e^{-(x+\beta)y}$ ($x,y>0$)。
再求 $X$ 的边缘密度：
$$f_X(x) = \int_0^\infty f(x,y) dy = \frac{\beta^\alpha}{\Gamma(\alpha)} \int_0^\infty y^\alpha e^{-(x+\beta)y} dy = \frac{\beta^\alpha}{\Gamma(\alpha)} \frac{\Gamma(\alpha+1)}{(x+\beta)^{\alpha+1}} = \frac{\alpha\beta^\alpha}{(x+\beta)^{\alpha+1}}$$
于是条件密度：
$$f_{Y|X}(y|x) = \frac{f(x,y)}{f_X(x)} = \frac{(x+\beta)^{\alpha+1}}{\Gamma(\alpha+1)} y^\alpha e^{-(x+\beta)y}$$
这恰好是 $\Gamma(\alpha+1, x+\beta)$ 分布的密度。
因此 $E(Y|X=x) = \frac{\alpha+1}{x+\beta}$，即 $E(Y|X) = \frac{\alpha+1}{X+\beta}$。
3. 求 $E(X)$：
由重期望法则：
$$E(X) = E(E(X|Y)) = E(1/Y) = \int_0^\infty \frac{1}{y} f_Y(y) dy = \int_0^\infty y^{-1} \frac{\beta^\alpha}{\Gamma(\alpha)} y^{\alpha-1} e^{-\beta y} dy = \frac{\beta^\alpha}{\Gamma(\alpha)} \int_0^\infty y^{\alpha-2} e^{-\beta y} dy$$
利用 $\Gamma$ 函数性质：
$$E(X) = \frac{\beta^\alpha}{\Gamma(\alpha)} \frac{\Gamma(\alpha-1)}{\beta^{\alpha-1}} = \frac{\beta}{\alpha-1} \quad (\alpha > 1)$$
当 $\alpha \in (0, 1]$ 时，积分发散，$E(X) = \infty$。
**技巧**：求条件期望 $E(Y|X)$ 时，若能识别出条件密度的标准分布形式，可直接写出期望，避免积分。

#### 条件期望的性质 【重点】
【定义】设 $X, Y$ 是随机变量，$g(x), h(y)$ 是实函数，期望有限。条件期望具有以下性质：
1. $|E(X|Y)| \le E(|X||Y)$
2. $E^2(X|Y) \le E(X^2|Y)$
3. $E(h(Y)g(X)|Y) = h(Y)E(g(X)|Y)$ （已知 $Y$ 时，$h(Y)$ 视为常数）
4. $E(E(g(X)|Y)) = E(g(X))$ （重期望法则的推广）
5. 当 $X, Y$ 独立时，$E(g(X)|Y) = E(g(X))$
6. $E(c + \sum_{i=1}^n c_i X_i | Y) = c + \sum_{i=1}^n c_i E(X_i|Y)$ （线性性质）
7. 若 $X_1 \le X_2$，则 $E(X_1|Y) \le E(X_2|Y)$ （单调性）
原理与推导：
性质3推导：给定 $Y=y$，$h(y)$ 是常数，可提取到期望符号外面。
性质4推导：由重期望法则直接推广，将 $X$ 替换为 $g(X)$ 即可。
【解题步骤】
1. 识别表达式中可作为常数提取的项（关于条件变量的函数）。
2. 利用线性性质拆分或合并条件期望。
3. 利用重期望法则去掉外层期望。
【易错点】
- 只有关于条件变量 $Y$ 的函数 $h(Y)$ 才能在给定 $Y$ 时提出期望符号，关于 $X$ 的函数不能直接提出。
- $E(h(Y)X|Y) = h(Y)E(X|Y)$ 是正确的，但 $E(h(X)Y|Y) \neq h(X)E(Y|Y)$。

**题目**：设 $X, Y$ 是随机变量，$h(x)$ 是实函数，若 $E(X^2) < \infty, E(h^2(Y)) < \infty$，证明 $E[(X - E(X|Y))h(Y)] = 0$。 [Lecture 09_handout.pdf P39]
**思路**：利用条件期望的性质，将乘积的期望转化为条件期望的期望。
**解答**：
由 Cauchy-Schwarz 不等式知 $E|Xh(Y)| < \infty$，期望存在。
利用性质3和性质4：
$$E[(X - E(X|Y))h(Y)] = E[Xh(Y)] - E[E(X|Y)h(Y)]$$
对于第二项，由于 $h(Y)$ 在给定 $Y$ 时是常数：
$$E[E(X|Y)h(Y)] = E[E(Xh(Y)|Y)]$$
再由重期望法则：
$$E[E(Xh(Y)|Y)] = E[Xh(Y)]$$
因此：
$$E[(X - E(X|Y))h(Y)] = E[Xh(Y)] - E[Xh(Y)] = 0$$
**技巧**：证明含条件期望的等式时，常将 $h(Y)$ 移入条件期望内部，再利用重期望法则转化为无条件期望。

### 本章小结（本片部分）
- 本章知识关系：全期望定理是重期望法则在事件分割下的特例；条件期望 $E(X|Y)$ 是联系条件分布与无条件期望的桥梁。
- 本章核心结论：重期望法则 $E(E(X|Y))=E(X)$ 是计算复杂期望的核心工具；条件期望在给定条件下具有与普通期望相同的线性、单调等性质。
- 必背公式：
  - $E(X) = \sum P(A_i)E(X|A_i)$
  - $E(E(X|Y)) = E(X)$
  - $E(h(Y)g(X)|Y) = h(Y)E(g(X)|Y)$
- 【记忆】条件期望是随机变量，重期望法则是求期望的“降维打击”工具。
