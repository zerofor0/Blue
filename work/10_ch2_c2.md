#### 本章总览
- **核心问题**：如何利用已发生的事件更新概率认知？复杂交事件概率如何分解计算？
- **高频考点排序**：条件概率定义与性质 > 乘法公式与全概率公式应用 > Borel-Cantelli引理 > 配对问题。
- **方法清单**：缩减样本空间法、序贯树形图法、事件分割法。

### （一）概率的连续性及其应用

#### 概率的连续性 【高频】
1. 【定义】：概率测度具有连续性。对于事件列，可以通过极限运算来求概率，即概率与极限运算在一定条件下可交换顺序。
2. 原理与推导：基于概率的极限定理，常用于处理无穷多次独立重复试验中的极限事件概率。
3. 【解题步骤】：
   1. 识别无穷序列事件及其极限关系（如“无穷多次发生”或“最终一直发生”）。
   2. 将极限事件转化为事件列的上、下极限表示。
   3. 利用概率的连续性及级数收敛性进行计算或判断。
4. 【易错点】：混淆“无穷多次发生”与“一直发生”的数学表达；忽略 Borel-Cantelli 引理第二部分中独立性的前提条件。

**例题**：
**题目**：抛掷一枚均匀硬币无穷多次。记事件 $H_n$ 表示第 $n$ 次抛掷得到正面。证明以概率 1 会有无穷多次正面出现，并求某时刻之后一直为正面的概率。[Lecture02_handout.pdf P21]
**思路**：判断事件列独立性及概率和，使用 Borel-Cantelli 引理第二部分求无穷多次发生概率；利用对立事件求一直发生概率。
**解答**：已知 $P(H_n) = 1/2$，且每次抛掷独立。
计算级数和：
$$\sum_{n=1}^{\infty} P(H_n) = \sum_{n=1}^{\infty} \frac{1}{2} = \infty$$
由 Borel-Cantelli 引理第二部分，$P(\limsup_{n \to \infty} H_n) = 1$，即以概率 1 会有无穷多次正面出现。
对于“某时刻之后一直为正面”，即 $\liminf_{n \to \infty} H_n$，其对立事件为“无穷多次反面”，即 $\limsup_{n \to \infty} H_n^c$。
同理 $P(\limsup_{n \to \infty} H_n^c) = 1$，故：
$$P(\liminf_{n \to \infty} H_n) = 1 - P(\limsup_{n \to \infty} H_n^c) = 1 - 1 = 0$$
**技巧**：$\limsup A_n$ 表示“无穷多次发生”，$\liminf A_n$ 表示“最终一直发生”。

#### 事件列的上、下极限 【重点】
1. 【定义】：
   - 上极限：$\limsup_{n \to \infty} A_n = \bigcap_{n=1}^{\infty} \bigcup_{k=n}^{\infty} A_k$，表示事件列中有无穷多个事件发生。
   - 下极限：$\liminf_{n \to \infty} A_n = \bigcup_{n=1}^{\infty} \bigcap_{k=n}^{\infty} A_k$，表示事件列中从某项开始后所有事件均发生。
2. 原理与推导：Borel-Cantelli 引理是事件列上极限的重要应用。
   - 若 $\sum_{n=1}^{\infty} P(A_n) < \infty$，则 $P(\limsup_{n \to \infty} A_n) = 0$。
   - 若 $A_n$ 相互独立且 $\sum_{n=1}^{\infty} P(A_n) = \infty$，则 $P(\limsup_{n \to \infty} A_n) = 1$。
3. 【易错点】：误用 Borel-Cantelli 引理第一部分的逆命题（独立性是第二部分的必要条件）。

### （二）条件概率与乘法公式

#### 条件概率 【必背】
1. 【定义】：设 $(\Omega, \mathcal{F}, P)$ 是概率空间，$A, B \in \mathcal{F}$，且 $P(A) > 0$。在事件 $A$ 发生的条件下，事件 $B$ 发生的条件概率定义为：
$$P(B|A) = \frac{P(A \cap B)}{P(A)} \tag{2-1}$$
2. 原理与推导：条件概率本质上是将样本空间缩减为 $A$。此时 $(\Omega, \mathcal{F}, P_A)$ 依然构成一个概率空间，其中 $P_A(\cdot) = P(\cdot|A)$。因此，条件概率满足概率的所有基本性质（非负性、规范性、可列可加性）。
3. 【解题步骤】：
   1. 确定已知条件事件 $A$ 及目标事件 $B$。
   2. 计算 $P(A)$ 和 $P(A \cap B)$（或利用缩减样本空间直接计算）。
   3. 代入定义式求解。
4. 【易错点】：混淆 $P(B|A)$ 与 $P(A|B)$；误认为 $P(B|A)$ 必定大于或小于 $P(B)$。

| 概念 | 公式 | 大小关系 (与 $P(B)$ 比较) |
| :--- | :--- | :--- |
| 包含关系 ($B \subset A$) | $P(B\|A) = P(B)/P(A)$ | $P(B\|A) \ge P(B)$ |
| 互斥关系 ($A \cap B = \emptyset$) | $P(B\|A) = 0$ | $P(B\|A) \le P(B)$ |
| 一般关系 | $P(B\|A) = P(A \cap B)/P(A)$ | 无法确定大小 |

**例题**：
**题目**：52张扑克中任取一张，在已知抽到梅花的条件下，求抽到5的概率。[Lecture02_handout.pdf P23]
**思路**：利用条件概率定义，或者缩减样本空间法。
**解答**：设 $A$ 表示抽到梅花，$B$ 表示抽到5。
$P(A) = 13/52$，$P(A \cap B) = 1/52$。
代入条件概率公式：
$$P(B|A) = \frac{P(A \cap B)}{P(A)} = \frac{1/52}{13/52} = \frac{1}{13}$$
**技巧**：已知条件发生时，样本空间由 52 缩减为 13，在 13 中等可能取 1，直接得 1/13。

**例题**：
**题目**：雷达探测飞机，$A=$ {飞机出现}，$B=$ {雷达报警}。已知 $P(A)=0.05, P(B|A)=0.99, P(B|A^c)=0.10$。求 $P(A|B)$。[Lecture02_handout.pdf P29]
**思路**：画序贯树形图，利用全概率公式求 $P(B)$，再利用条件概率定义求 $P(A|B)$。
**解答**：先求 $P(B)$：
$$P(B) = P(A)P(B|A) + P(A^c)P(B|A^c) = 0.05 \times 0.99 + 0.95 \times 0.10 = 0.1445$$
再求 $P(A \cap B)$：
$$P(A \cap B) = P(A)P(B|A) = 0.05 \times 0.99 = 0.0495$$
代入条件概率公式：
$$P(A|B) = \frac{P(A \cap B)}{P(B)} = \frac{0.0495}{0.1445} \approx 0.34$$
**技巧**：此类“由果溯因”问题本质是贝叶斯公式的应用，通过树形图理清分支概率是关键。

#### 乘法公式 【重点】
1. 【定义】：设 $(\Omega, \mathcal{F}, P)$ 是概率空间，$A_i \in \mathcal{F}, i=1,\dots,n$，且 $P(A_1 \cap \dots \cap A_{n-1}) > 0$，则：
$$P\left(\bigcap_{i=1}^n A_i\right) = P(A_1) \prod_{i=2}^n P\left(A_i \bigg| A_1 \cap \dots \cap A_{i-1}\right) \tag{2-2}$$
2. 原理与推导：由条件概率定义 $P(B|A) = P(A \cap B)/P(A)$ 变形得到 $P(A \cap B) = P(A)P(B|A)$，推广至 $n$ 个事件即得乘法公式。公式可任意换序，共有 $n!$ 种形式。
3. 【解题步骤】：
   1. 将复杂交事件分解为一系列序贯发生的事件。
   2. 构造序贯树形图，在分枝上标注条件概率。
   3. 将路径上的条件概率相乘得到最终概率。
4. 【易错点】：忽略后续事件的条件概率是在前面事件均已发生的条件下计算的；换序时未同步更新条件部分。

**例题**：
**题目**：（配对问题）某人写了 $n$ 封信装入 $n$ 个信封，任意写地址不重复。求没有一个信封地址正确的概率 $q_0$。[Lecture02_handout.pdf P34]
**思路**：设 $A_i$ 为第 $i$ 个信封地址正确，所求为 $P(\bigcap_{i=1}^n A_i^c)$。利用对立事件及容斥原理，结合乘法公式计算交事件概率。
**解答**：$q_0 = 1 - P(\bigcup_{i=1}^n A_i)$。
由乘法公式计算任意 $k$ 个事件同时发生的概率：
$$P(A_1 \cap \dots \cap A_k) = P(A_1)P(A_2|A_1) \dots P(A_k|A_1 \cap \dots \cap A_{k-1}) = \frac{1}{n} \cdot \frac{1}{n-1} \dots \frac{1}{n-k+1} = \frac{(n-k)!}{n!}$$
由容斥原理：
$$P\left(\bigcup_{i=1}^n A_i\right) = \sum_{i}P(A_i) - \sum_{i<j}P(A_i \cap A_j) + \dots + (-1)^{n-1}P(A_1 \cap \dots \cap A_n)$$
$$= \binom{n}{1}\frac{1}{n} - \binom{n}{2}\frac{(n-2)!}{n!} + \dots + (-1)^{n-1}\binom{n}{n}\frac{1}{n!}$$
$$= 1 - \frac{1}{2!} + \frac{1}{3!} - \dots + (-1)^{n-1}\frac{1}{n!}$$
因此：
$$q_0 = 1 - \left(1 - \frac{1}{2!} + \dots + (-1)^{n-1}\frac{1}{n!}\right) = \sum_{k=0}^n \frac{(-1)^k}{k!}$$
**技巧**：配对问题是经典模型，当 $n \to \infty$ 时 $q_0 \to e^{-1} \approx 0.368$。

### （三）全概率公式

#### 全概率公式 【高频】
1. 【定义】：设 $A_1, A_2, \dots, A_n$ 为样本空间 $\Omega$ 的一个分割（完备事件组），且 $P(A_i) > 0$，则对任一事件 $B$，有：
$$P(B) = \sum_{i=1}^n P(A_i)P(B|A_i) \tag{2-3}$$
2. 原理与推导：将复杂事件 $B$ 分解为互不相容的子事件 $A_i \cap B$ 之并，利用概率的有限可加性和乘法公式推导得出。
3. 【解题步骤】：
   1. 寻找样本空间的一个合适分割 $A_1, \dots, A_n$（通常是导致 $B$ 发生的各种“原因”或“阶段”）。
   2. 计算各原因的概率 $P(A_i)$ 及在原因下结果发生的条件概率 $P(B|A_i)$。
   3. 代入公式求和。
4. 【易错点】：分割事件组不满足互斥性或完备性（遗漏情况或重叠）。

### 本章小结
- **本章知识关系**：条件概率是概率论的核心概念，由其定义直接推导出乘法公式（求交事件概率）和全概率公式（由因求果）。事件列的极限与 Borel-Cantelli 引理将概率的连续性推广至无穷序列。
- **本章核心结论**：条件概率本质是概率测度的缩减；乘法公式用于序贯事件；全概率公式用于化整为零。
- **必背公式**：
  - 条件概率：$P(B|A) = \frac{P(A \cap B)}{P(A)}$
  - 乘法公式：$P(\bigcap A_i) = P(A_1) \prod P(A_i | A_1 \cap \dots \cap A_{i-1})$
  - 全概率公式：$P(B) = \sum P(A_i)P(B|A_i)$
- 【记忆】：条件概率缩空间，乘法公式算交积，全概公式分原因。
