### 本章总览
- **核心问题**：在给定某些事件或随机变量信息的条件下，如何定义并计算随机变量的期望与方差。
- **高频考点排序**：全期望定理（全方差公式下章讲）的应用 > $E(X|A)$ 的计算（定理2.1） > 条件分布与贝叶斯公式 > 指数分布无记忆性的应用。
- **方法清单**：1. 条件期望定义法（先求条件分布再求期望）；2. 示性函数法（$E(X|A) = E(XI_A)/P(A)$）；3. 全期望公式法（分割样本空间加权求和）。

---

### （一）条件概率与条件分布回顾

#### 条件概率与条件分布 【重点】
1. 【定义】
   - **事件条件下的条件分布** $P_{X|A}$：
     - 离散型：$p_{X|A}(x) = P(X=x|A) = \frac{P(\{X=x\} \cap A)}{P(A)}$
     - 连续型：$f_{X|A}(x) = \frac{f_X(x) I_A(x)}{P(A)}$ （其中 $I_A(x)$ 为事件 $A$ 的示性函数）
   - **随机变量条件下的条件分布** $P_{X|Y}$：
     - 离散型：$P_{X|Y}(x_i|y_j) = \frac{p_{ij}}{q_j}$
     - 连续型：$f_{X|Y}(x|y) = \frac{f(x,y)}{f_Y(y)} = \frac{f(x,y)}{\int_{-\infty}^{\infty} f(x,y) dx}$
2. 原理与推导：
   - **乘法法则**：$f_{X,Y}(x,y) = f_{X|Y}(x|y) f_Y(y)$
   - **全概率公式**（连续型）：
     $$f_X(x) = \int f_{X|Y}(x|y) f_Y(y) dy \tag{9-1}$$
   - **贝叶斯准则**（连续型）：
     $$f_{X|Y}(x|y) = \frac{f_X(x) f_{Y|X}(y|x)}{f_Y(y)} = \frac{f_X(x) f_{Y|X}(y|x)}{\int f_X(x) f_{Y|X}(y|x) dx} \tag{9-2}$$
3. 【解题步骤】
   - 判别 $X$ 与 $Y$ 的类型（双离散、双连续、一离散一连续）。
   - 选取对应的贝叶斯公式或全概率公式。
   - 注意分母 $f_Y(y)$ 或 $P(Y=y)$ 需要通过全概率公式（对另一个变量求和或积分）获得。
4. 【易错点】
   - 混淆不同类型随机变量组合下的贝叶斯公式（如 $X$ 离散 $Y$ 连续时，分母应为积分，分子为离散概率乘连续密度）。
   - 连续型求边缘密度时，积分变量和积分限写错。

### （二）事件条件下的期望 $E(X|A)$

#### $E(X|A)$ 的定义与计算 【必背】
1. 【定义】
   给定事件 $A$ 发生（且 $P(A)>0$）的条件下，随机变量 $X$ 的条件期望定义为在条件分布下对 $X$ 求期望：
   - 离散型：
     $$E(X|A) = \sum_{i=1}^{\infty} x_i P(X=x_i|A) \tag{9-3}$$
   - 连续型：
     $$E(X|A) = \int_{-\infty}^{\infty} x f_{X|A}(x) dx \tag{9-4}$$
2. 原理与推导：
   条件期望本质上是将条件分布作为权重对随机变量的取值进行加权平均。它是条件分布的数字特征，是一个确定的实数。
3. 【解题步骤】
   - 步骤 1：求出条件概率 $P(X=x_i|A)$ 或条件密度 $f_{X|A}(x)$。
   - 步骤 2：代入期望定义式进行求和或积分。
4. 【易错点】
   - 直接用 $X$ 的无条件分布求和/积分，忘记将分布替换为条件分布。

#### 由无条件期望计算 $E(X|A)$ 【重点】
1. 【定义】
   设 $P(A)>0$，$X$ 是随机变量且 $E(X|A)$ 存在，则：
   $$E(X|A) = \frac{E(X I_A)}{P(A)} \tag{9-5}$$
   其中 $I_A$ 是事件 $A$ 的示性函数。
2. 原理与推导：
   对于连续型非负随机变量，利用 $E(X) = \int_0^\infty P(X>x) dx$，在条件概率 $P(\cdot|A) = P(\cdot \cap A)/P(A)$ 下代入期望定义，可分离出 $E(X I_A)$ 与 $P(A)$ 的比值。该定理极大地简化了条件期望的计算，避免了显式求解条件分布。
3. 【解题步骤】
   - 步骤 1：写出事件 $A$ 对应的示性函数 $I_A$（如 $A=\{X>a\}$，则 $I_A = I_{\{X>a\}}$）。
   - 步骤 2：计算 $E(X I_A)$，通常转化为二重积分或分段积分。
   - 步骤 3：计算 $P(A)$。
   - 步骤 4：作比值求得 $E(X|A)$。
4. 【易错点】
   - 计算 $E(X I_A)$ 时，积分限出错。例如 $A=\{X<Y\}$ 时，应满足 $0<x<y$，外层对 $x$ 积分，内层对 $y$ 从 $x$ 积到 $\infty$。

**例题**：
**题目**：设 $X \sim \mathcal{E}(\lambda)$, 对 $a>0$, 证明 $E(X|X>a) = a + 1/\lambda$。[Lecture 09_handout.pdf P19]
**思路**：利用定理2.1（公式 9-5），将条件期望转化为 $E(X I_{\{X>a\}}) / P(X>a)$。也可利用指数分布的无记忆性直接得出。
**解答**：
利用公式 $E(X|X>a) = \frac{E(X I_{\{X>a\}})}{P(X>a)}$。
分别计算分子分母：
分母：$P(X>a) = e^{-\lambda a}$
分子：
$$E(X I_{\{X>a\}}) = \int_a^\infty x \lambda e^{-\lambda x} dx$$
利用分部积分法：
$$\int_a^\infty x \lambda e^{-\lambda x} dx = \left[ -x e^{-\lambda x} \right]_a^\infty + \int_a^\infty e^{-\lambda x} dx = a e^{-\lambda a} + \frac{1}{\lambda} e^{-\lambda a}$$
作比值：
$$E(X|X>a) = \frac{a e^{-\lambda a} + \frac{1}{\lambda} e^{-\lambda a}}{e^{-\lambda a}} = a + \frac{1}{\lambda}$$
**技巧**：遇到指数分布的条件期望（形如 $X>a$ 或 $X<t$），优先考虑无记忆性：$E(X|X>a) = a + E(X)$，可秒杀。

**例题**：
**题目**：设 $X, Y$ 独立, 且 $X \sim \mathcal{E}(\lambda), Y \sim \mathcal{E}(\mu)$ 计算 $E(X|X<Y)$。[Lecture 09_handout.pdf P20]
**思路**：利用定理2.1，转化为计算 $E(X I_{\{X<Y\}}) / P(X<Y)$。因为 $X,Y$ 独立，联合密度为两者乘积，通过二重积分求解。
**解答**：
利用公式 $E(X|X<Y) = \frac{E(X I_{\{X<Y\}})}{P(X<Y)}$。
分母：$P(X<Y) = \int_0^\infty P(Y>x) f_X(x) dx = \int_0^\infty e^{-\mu x} \lambda e^{-\lambda x} dx = \frac{\lambda}{\lambda+\mu}$
分子：
$$E(X I_{\{X<Y\}}) = \int_0^\infty \int_0^\infty x I_{\{x<y\}} \lambda e^{-\lambda x} \mu e^{-\mu y} dy dx$$
先对 $y$ 积分（$y$ 从 $x$ 到 $\infty$）：
$$= \int_0^\infty x \lambda e^{-\lambda x} \left( \int_x^\infty \mu e^{-\mu y} dy \right) dx = \int_0^\infty x \lambda e^{-\lambda x} e^{-\mu x} dx$$
此积分为期望 $\int_0^\infty x \lambda e^{-(\lambda+\mu)x} dx$，相当于参数为 $\lambda+\mu$ 的指数分布的期望乘以 $\lambda$，即 $\lambda \cdot \frac{1}{(\lambda+\mu)^2}$。
作比值：
$$E(X|X<Y) = \frac{\lambda / (\lambda+\mu)^2}{\lambda / (\lambda+\mu)} = \frac{1}{\lambda+\mu}$$
**技巧**：计算含示性函数的独立变量二重积分时，固定一个变量，将另一个变量的积分限用示性函数条件表示（如 $I_{\{x<y\}}$ 固定 $x$，则 $y \in (x, \infty)$），可大幅简化计算。

### （三）全期望定理

#### 全期望定理 【高频】
1. 【定义】
   设 $A_1, \dots, A_n$ 构成样本空间 $\Omega$ 的一个分割，且 $P(A_i)>0$，则：
   $$E(X) = \sum_{i=1}^n P(A_i) E(X|A_i) \tag{9-6}$$
2. 原理与推导：
   全概率公式的期望推广。将样本空间划分为若干互斥事件，分别计算每个事件下的条件期望，再按事件发生的概率进行加权平均。
3. 【解题步骤】
   - 步骤 1：寻找合适的样本空间分割 $A_1, \dots, A_n$（通常是影响 $X$ 取值的第一步随机试验结果）。
   - 步骤 2：计算各分割事件发生的概率 $P(A_i)$。
   - 步骤 3：在各事件发生的条件下，计算 $X$ 的条件期望 $E(X|A_i)$。
   - 步骤 4：加权求和得到 $E(X)$。
4. 【易错点】
   - 分割不完整（未穷尽所有可能）或分割事件之间不互斥。
   - 计算条件期望时，未正确使用条件分布（如忘记使用无记忆性或条件密度）。

### 本章小结
- **本章知识关系**：本章从条件概率出发，将条件分布推广到数字特征层面，定义了 $E(X|A)$。通过定理2.1将条件期望转化为无条件期望与示性函数的比值，极大简化了计算。最后利用样本空间分割，导出了全期望定理，为求解复杂随机变量的期望提供了分步计算的框架。本章是下章条件期望 $E(X|Y)$ 与条件方差的基础。
- **本章核心结论**：条件期望是条件分布的加权平均；计算条件期望首选 $E(X I_A)/P(A)$；全期望定理允许我们将复杂期望拆解为简单情形的加权平均。
- **必背公式**：
  - $E(X|A) = \frac{E(X I_A)}{P(A)}$
  - $E(X) = \sum P(A_i) E(X|A_i)$
- **【记忆】**：条件期望等于“带示性函数的期望”除以“条件概率”；总期望等于“条件期望的加权平均”。
