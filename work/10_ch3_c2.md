#### 本章总览
- 核心问题：如何计算随机变量及其函数的期望与方差？期望和方差有哪些重要性质与优化解释？
- 高频考点排序：期望的线性性质应用（分解法） > 常见分布的方差计算 > 方差的性质与独立和公式 > 期望的优化解释 > 高阶矩概念。
- 方法清单：定义法直接积分、指示变量分解法、利用性质化简计算、优化极值法。

### 随机变（向）量函数的数学期望

#### 随机变量函数的期望计算 【重点】
1. 【定义】：设 $X$ 为随机变量，$g(x)$ 为实值函数。若 $E(|g(X)|) < \infty$，则 $g(X)$ 的数学期望定义为
$$E(g(X)) = \int_{-\infty}^{\infty} g(x) f(x) dx \tag{8-1}$$
对于二维随机向量 $(X,Y)$，有 $E(g(X,Y)) = \iint_{\mathbb{R}^2} g(x,y) f(x,y) dx dy$。
2. 原理与推导：利用勒贝格积分或黎曼-斯蒂尔杰斯积分定义，将概率加权求和推广到连续情形。计算时无需先求 $g(X)$ 的分布，直接用原变量的密度函数积分即可。
3. 【解题步骤】：
   1. 确定原随机变量 $X$ 或 $(X,Y)$ 的概率密度函数。
   2. 将目标函数 $g(x)$ 与密度函数相乘。
   3. 确定积分区域（注意结合函数的奇偶对称性化简）。
   4. 计算定积分。
4. 【易错点】：
   - 忽略随机变量的取值范围，导致积分上下限错误。
   - 计算二维期望时，未正确写出联合密度（如单位圆面积应为 $\pi$ 而非 1）。
   - 误认为 $E(g(X)) = g(E(X))$，一般情况下两者不相等。

**例题**：设 $X \sim U(0, \pi/2)$，计算 $E(\cos X)$。[Lecture08_handout.pdf P25]
**思路**：已知 $X$ 的密度函数为 $f(x) = \frac{2}{\pi}$ ($0 \le x \le \pi/2$)，直接代入期望定义公式积分。
**解答**：
$$E(\cos X) = \int_{0}^{\frac{\pi}{2}} \cos x \cdot \frac{2}{\pi} dx = \frac{2}{\pi} \int_{0}^{\frac{\pi}{2}} \cos x dx = \frac{2}{\pi} [\sin x]_{0}^{\frac{\pi}{2}} = \frac{2}{\pi} \tag{8-2}$$
**技巧**：计算均匀分布的期望等于函数积分除以区间长度。

**例题**：设 $X \sim N(0,1)$，$\alpha$ 是常数，计算 $E(X^\alpha)$。[Lecture08_handout.pdf P26]
**思路**：代入标准正态密度函数，利用奇偶性化简，并作换元 $x^2/2 = t$ 转化为 Gamma 函数计算。
**解答**：$X$ 的密度为 $\varphi(x) = \frac{1}{\sqrt{2\pi}} e^{-x^2/2}$。对 $\alpha > -1$，有
$$E(X^\alpha) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^{\infty} x^\alpha e^{-x^2/2} dx = \frac{2}{\sqrt{2\pi}} \int_{0}^{\infty} x^\alpha e^{-x^2/2} dx \tag{8-3}$$
令 $x^2/2 = t$，则 $x = (2t)^{1/2}$，$dx = \frac{1}{\sqrt{2t}} dt$，代入得：
$$E(X^\alpha) = \frac{2^\alpha}{\sqrt{\pi}} \int_{0}^{\infty} t^{\frac{\alpha+1}{2}-1} e^{-t} dt = \frac{2^\alpha}{\sqrt{\pi}} \Gamma\left(\frac{\alpha+1}{2}\right) \tag{8-4}$$
特别地，$E(X^2)=1$；$E(|X|)=\sqrt{2/\pi}$；当 $\alpha \le -1$ 时，$E(X^\alpha) = \infty$。
**技巧**：正态分布的高阶矩常通过 Gamma 函数计算，注意利用奇偶对称性化简积分区间。

**例题**：设 $(X, Y)$ 在单位圆 $D=\{(x,y)|x^2+y^2 \le 1\}$ 内均匀分布，计算 $E(X), E(Y)$。[Lecture08_handout.pdf P27]
**思路**：写出联合密度，利用二维期望定义积分。也可利用对称性直接得出。
**解答**：$(X, Y)$ 的联合密度为 $f(x,y) = \frac{1}{\pi} I_D(x,y)$。
$$E(X) = \iint_{\mathbb{R}^2} x f(x,y) dx dy = \frac{1}{\pi} \int_{-1}^{1} \left( \int_{-\sqrt{1-y^2}}^{\sqrt{1-y^2}} x dx \right) dy = 0 \tag{8-5}$$
**技巧**：若联合密度关于某变量是偶函数且积分区域对称，则该变量的期望必为 0。

### 期望的性质

#### 期望的重要性质 【重点】
1. 【定义】：设 $E(X), E(Y)$ 有限，$a, b, C$ 为实数。
2. 原理与推导：
   1. 常数期望：$E(C) = C$
   2. 绝对值不等式：$E(X) \le E(|X|)$
   3. 线性性质：$E(aX+bY) = aE(X)+bE(Y)$
   4. 单调性：若 $X \le Y \ a.s.$，则 $E(X) \le E(Y)$
   5. 独立乘积：若 $X, Y$ 独立，则 $E(XY) = E(X)E(Y)$
3. 【解题步骤】：遇到复杂随机变量的期望，常将其分解为若干简单随机变量（如示性变量）之和，再利用线性性质求解。
4. 【易错点】：
   - 误用 $E(XY) = E(X)E(Y)$，该式仅在独立时成立（或至少不相关）。
   - 混淆 $E(g(X))$ 与 $g(E(X))$。

**例题**：(质量抽查) $N$ 件产品中有 $M$ 件正品，从中无放回任取 $n$ 件，期望抽到几件正品？[Lecture08_handout.pdf P29]
**思路**：直接用超几何分布定义算期望较复杂。引入指示变量 $X_k$ 表示第 $k$ 次是否抽到正品，利用期望的线性性质求解。
**解答**：记 $X$ 为抽到的正品数，$X_k$ 为第 $k$ 次抽到正品与否的指示变量。则 $X = X_1 + \dots + X_n$。
无放回抽样中，每次抽到正品的概率均为 $E(X_1) = \frac{M}{N}$。
由线性性质：
$$E(X) = E(X_1) + \dots + E(X_n) = n E(X_1) = n \cdot \frac{M}{N} \tag{8-6}$$
**技巧**：复杂随机变量期望计算，常将其分解为示性变量之和，利用线性性质绕开复杂的联合分布。

**例题**：(机器人打字问题) 一个机器人在只有26个小写字母的键盘上打字。每次独立等可能选择一个字母，要打出 $k$ 个不同的字母，期望打字多少次？[Lecture08_handout.pdf P30-31]
**思路**：将总打字次数分解为等待每个新字母出现的时间之和，每个等待时间服从几何分布。
**解答**：设 $S_k$ 为出现第 $k$ 个新字母时的总打字次数，$X_k = S_k - S_{k-1}$ 为等待第 $k$ 个新字母的打字次数。总字母数 $N=26$。
出现第 $k-1$ 个新字母后，还有 $N-k+1$ 个新字母，故 $X_k$ 服从参数 $p_k = \frac{N-k+1}{N}$ 的几何分布，$E(X_k) = \frac{1}{p_k} = \frac{N}{N-k+1}$。
由 $S_k = X_1 + \dots + X_k$，得：
$$E(S_k) = \sum_{j=1}^k E(X_j) = N \sum_{j=1}^k \frac{1}{N-j+1} = N \left( \sum_{m=1}^N \frac{1}{m} - \sum_{m=1}^{N-k} \frac{1}{m} \right) \approx N \ln \frac{N+1}{N-k+1} \tag{8-7}$$
**技巧**：将复杂过程分解为独立阶段（如等待时间），是概率论计算期望的核心技巧（即“分解法”）。

#### 期望的优化解释与中位数 【重点】
1. 【定义】：
   - 期望的优化解释（定理2.3）：$E(X) = \arg\min_{c \in \mathbb{R}} E((X-c)^2)$。即期望是使均方误差最小的常数预测。
   - 中位数（定理2.4）：设 $m$ 是 $X$ 的中位数，则 $m = \arg\min_{c \in \mathbb{R}} E|X-c|$。即中位数是使绝对误差最小的常数预测。
2. 原理与推导：对 $E((X-c)^2)$ 求导令其为 0，得 $c = E(X)$；对 $E|X-c|$ 利用次梯度可得中位数。
3. 【易错点】：

| 特征 | 期望 $E(X)$ | 中位数 $m$ |
| :--- | :--- | :--- |
| 优化目标 | 最小化均方误差 $E((X-c)^2)$ | 最小化绝对误差 $E\|X-c\|$ |
| 存在性 | 要求 $E(\|X\|) < \infty$ | 一定存在 |
| 唯一性 | 唯一 | 可能不唯一 |
| 抗极值 | 易受极值影响 | 不易受极值影响 |

#### 高阶矩 【必背】
1. 【定义】：设 $X$ 是随机变量，$m$ 是正整数。若 $E(|X|^m) < \infty$，称 $E(X^m)$ 为 $X$ 的 $m$ 阶原点矩，称 $E((X-E(X))^m)$ 为 $X$ 的 $m$ 阶中心矩。
2. 原理与推导：当 $m > 2$ 时统称为高阶矩。常见的中心矩：
   - $m=2$：方差
   - $m=3$：偏度，衡量分布的非对称性。公式为 $\frac{E((X-E(X))^3)}{(E((X-E(X))^2))^{3/2}}$
   - $m=4$：峰度，衡量分布的尾部厚度。公式为 $\frac{E((X-E(X))^4)}{(E((X-E(X))^2))^2} - 3$

### 方差

#### 方差的定义与计算 【必背】
1. 【定义】：如果随机变量 $X$ 的期望 $\mu = E(X)$ 有限，则称 $E((X-\mu)^2)$ 为 $X$ 的方差，记作 $var(X)$ 或 $\sigma_X^2$。
2. 原理与推导：方差表示 $X$ 在期望附近的集中程度。计算公式推导：
$$var(X) = E((X-E(X))^2) = E(X^2 - 2XE(X) + (E(X))^2) = E(X^2) - (E(X))^2 \tag{8-8}$$
3. 【解题步骤】：
   1. 先求 $E(X)$。
   2. 再求 $E(X^2)$（常通过分解 $X^2 = X(X-1) + X$ 简化离散型计算）。
   3. 代入公式 $var(X) = E(X^2) - (E(X))^2$。
4. 【易错点】：计算离散型分布高阶矩时，未利用 $k(k-1)$ 等因子化简级数。

#### 常见分布的方差 【必背】
1. 【定义】：各类常见分布的方差计算结果。
2. 原理与推导：利用定义法或性质求解。例如泊松分布：$E(X^2) = E(X(X-1)) + E(X) = \lambda^2 + \lambda$，故 $var(X) = \lambda^2 + \lambda - \lambda^2 = \lambda$。
3. 【易错点】：混淆参数关系，如指数分布方差为 $1/\lambda^2$ 而非 $\lambda^2$；Gamma 分布 $\Gamma(\alpha, \beta)$ 方差为 $\alpha/\beta^2$。

| 分布名称 | 记号 | 期望 $E(X)$ | 方差 $var(X)$ |
| :--- | :--- | :--- | :--- |
| 两点分布 | $B(1, p)$ | $p$ | $pq$ |
| 二项分布 | $B(n, p)$ | $np$ | $npq$ |
| 泊松分布 | $\mathcal{P}(\lambda)$ | $\lambda$ | $\lambda$ |
| 几何分布 | $G(p)$ | $1/p$ | $q/p^2$ |
| 均匀分布 | $U(a, b)$ | $(a+b)/2$ | $(b-a)^2/12$ |
| 指数分布 | $\varepsilon(\lambda)$ | $1/\lambda$ | $1/\lambda^2$ |
| 正态分布 | $N(\mu, \sigma^2)$ | $\mu$ | $\sigma^2$ |
| Gamma分布 | $\Gamma(\alpha, \beta)$ | $\alpha/\beta$ | $\alpha/\beta^2$ |

#### 方差的性质 【重点】
1. 【定义】：设 $E(X) = \mu$，$var(X) < \infty$。
2. 原理与推导：
   1. 平移缩放：$var(aX+b) = a^2 var(X)$
   2. 优化解释：$var(X) = E((X-\mu)^2) \le E((X-c)^2), \forall c \in \mathbb{R}$
   3. 退化分布：$var(X) = 0 \iff X = \mu \ a.s.$
   4. 独立可加性：若 $X_1, \dots, X_n$ 相互独立，则 $var(\sum_{k=1}^n X_k) = \sum_{k=1}^n var(X_k)$
3. 【解题步骤】：求多个随机变量和的方差时，先判断独立性。若独立直接相加；若不独立，需展开计算协方差或利用 $E(X^2)$ 求解。
4. 【易错点】：在变量不独立时错误使用方差可加性。

**例题**：(配对问题) 某人写了 $n$ 封信, 将其装入 $n$ 个信封, 并在每个信封上分别任意地写上 $n$ 个收信人的一个地址（不重复）,记 $X$ 为匹配正确的封数。求 $E(X)$ 和 $Var(X)$。[Lecture08_handout.pdf P49]
**思路**：利用指示变量分解。设 $X_i$ 为第 $i$ 封信是否匹配正确，则 $X = \sum X_i$。计算 $E(X_i)$ 和 $E(X_i X_j)$ 进而求方差。
**解答**：设 $X_i = I\{\text{第 } i \text{ 封信配对}\}$，则 $X = X_1 + \dots + X_n$。
对于任意 $i$，$P(X_i = 1) = \frac{1}{n}$，故 $E(X_i) = \frac{1}{n}$。
$$E(X) = \sum_{i=1}^n E(X_i) = n \cdot \frac{1}{n} = 1 \tag{8-9}$$
对于方差，因 $X_i$ 不独立，展开 $X^2 = \sum_{i=1}^n X_i^2 + \sum_{i \neq j} X_i X_j$。
因 $X_i^2 = X_i$，且对于 $i \neq j$，$P(X_i=1, X_j=1) = \frac{1}{n(n-1)}$，故 $E(X_i X_j) = \frac{1}{n(n-1)}$。
$$E(X^2) = \sum_{i=1}^n \frac{1}{n} + \sum_{i \neq j} \frac{1}{n(n-1)} = 1 + n(n-1) \frac{1}{n(n-1)} = 2 \tag{8-10}$$
$$Var(X) = E(X^2) - (E(X))^2 = 2 - 1^2 = 1 \tag{8-11}$$
**技巧**：求和变量的方差时，若变量不独立，不能直接将方差相加。需展开 $Var(X) = E(X^2) - (E(X))^2$，利用指示变量的乘积期望计算。

### 本章小结（本片部分）
- 本章知识关系：本片从期望的定义推广到随机变量函数的期望，进而引出期望的线性性质与优化解释；在此基础上，定义了衡量离散程度的方差，并推导了其性质与常见分布的计算公式。
- 本章核心结论：期望是均方误差下的最优预测，中位数是绝对误差下的最优预测；方差具有平移不变性和缩放平方性；独立和的方差等于方差之和。
- 必背公式：
  - $E(aX+bY) = aE(X)+bE(Y)$
  - $var(X) = E(X^2) - (E(X))^2$
  - $var(aX+b) = a^2 var(X)$
  - 独立时：$var(\sum X_i) = \sum var(X_i)$
- 【记忆】：期望线性可加，方差独立才加；期望最小均方，方差等于平方减期望方。
