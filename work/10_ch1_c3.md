#### 本章总览
- **核心问题**：如何求多维随机变量函数的分布？次序统计量有何分布性质？p分位数如何定义及如何用于随机数生成？
- **高频考点排序**：1. 雅可比变换法求联合密度；2. 次序统计量的定义与分布计算；3. p分位数的定义与逆变换采样；4. 卷积公式与分布函数法（CDF法）。
- **方法清单**：分布函数法（CDF法）、卷积公式、雅可比变换法（增补变量法/微元法）、逆变换法。

#### p分位数的定义与性质 【必背】
1. 【定义】
设 $X$ 是 $(\Omega, \mathcal{F}, \mathbb{P})$ 上的随机变量，$F(x)$ 是其分布函数。对 $p \in [0,1]$，如果满足 $\mathbb{P}(X \le x) \ge p$ 且 $\mathbb{P}(X \ge x) \ge 1-p$，则称 $x$ 为 $X$ 的 $p$-分位数。此直观定义不一定唯一。
为保证唯一性，正式定义为：
$$F^{-1}(p) = \inf \{x : F(x) \ge p\} \tag{7-1}$$
称 $F^{-1}(p)$ 为 $F$ 或 $X$ 的 $p$-分位数，通常用 $\xi_p$ 表示。特别地，$\xi_{1/2}$ 称为中位数。
2. 原理与推导
等价定义：$F^{-1}(p) = \sup \{x : F(x) < p\}$。
p分位数具有以下六大性质（证明依赖于CDF的单调非降性与右连续性）：
- 性质1：$F^{-1}(p)$ 单调非降。
- 性质2：$F^{-1}(F(x)) \le x, \forall x \in \mathbb{R}$。
- 性质3：$F(F^{-1}(p)) \ge p, \forall p \in (0,1)$。
- 性质4：$F^{-1}(p) \le t \iff p \le F(t)$。
- 性质5：$F^{-1}(p)$ 是左连续的。
- 性质6：如果 $F$ 是连续的，则 $F(F^{-1}(p)) = p, \forall p \in (0,1)$。
3. 【解题步骤】
   1. 判断分布类型：若 $F$ 连续，直接令 $F(x) = p$ 解方程求 $F^{-1}(p)$。
   2. 若 $F$ 离散或非严格单调，使用基本定义 $F^{-1}(p) = \inf \{x : F(x) \ge p\}$ 寻找满足条件的最小 $x$。
   3. 涉及不等式变形时，优先利用性质4进行等价转化。
4. 【易错点】
   - 误以为 $F^{-1}(F(x)) = x$ 恒成立。实际上只有当 $F$ 严格单调时等号才成立，一般情况为 $\le$。
   - 混淆 $F^{-1}(F(x))$ 与 $F(F^{-1}(p))$ 的不等号方向。

| 概念 | 公式 | 不等号方向 | 直观理解 |
| :--- | :--- | :--- | :--- |
| 先作用 $F$ 后作用 $F^{-1}$ | $F^{-1}(F(x)) \le x$ | $\le$ | $F$ 可能有平坦段，映射后无法完全还原原值，可能偏小 |
| 先作用 $F^{-1}$ 后作用 $F$ | $F(F^{-1}(p)) \ge p$ | $\ge$ | $F$ 可能有跳跃点，映射后可能跳过原概率值，可能偏大 |

5. 例题
**例题**：来自课件附录证明 [Lecture 07_handout.pdf P65]
**题目**：证明 $F(F^{-1}(p)) \ge p, \forall p \in (0,1)$。
**思路**：利用下确界性质与分布函数的右连续性。
**解答**：
记集合 $A = \{t : F(t) \ge p\}$，由 CDF 单调非降性知 $A$ 必为 $[a, \infty)$ 或 $(a, \infty)$ 形式。
由定义 $F^{-1}(p) = \inf A = a$。
取序列 $a + \frac{1}{n} \in A$，则有 $F(a + \frac{1}{n}) \ge p$。
令 $n \to \infty$，利用 $F(x)$ 的右连续性，得：
$$F(F^{-1}(p)) = F(a) = \lim_{n \to \infty} F\left(a + \frac{1}{n}\right) \ge p$$
**技巧**：处理涉及 $\inf$ 的等式证明时，构造趋于下确界的数列并结合右连续性是标准套路。

#### 随机变量的生成（逆变换法） 【重点】
1. 【定义】
设 $X \sim \mathcal{U}(0,1)$，$F(x)$ 是连续分布函数，则 $Y = F^{-1}(X) \sim F$。此方法称为逆变换法，常用于统计计算中生成指定分布的随机数。
2. 原理与推导
证明利用了性质4：$F^{-1}(p) \le t \iff p \le F(t)$。
$$F_Y(y) = \mathbb{P}(Y \le y) = \mathbb{P}(F^{-1}(X) \le y) = \mathbb{P}(X \le F(y)) = F(y)$$
因此 $Y \sim F$。
3. 【解题步骤】
   1. 求出目标分布的累积分布函数 $F(x)$。
   2. 解方程 $F(x) = u$ （其中 $u \sim \mathcal{U}(0,1)$），得到反函数 $x = F^{-1}(u)$。
   3. 将均匀分布随机数 $u$ 代入反函数，即得目标分布随机数。
4. 【易错点】
   - 忽略 $F(x)$ 的定义域，导致反函数求解错误。
   - 对于离散型分布，需按跳跃点分段处理，将 $(0,1)$ 区间按概率质量进行划分。
5. 例题
**例题**：生成指数分布随机变量 [Lecture 07_handout.pdf P53]
**题目**：设易生成服从均匀分布的随机变量 $U \sim \mathcal{U}(0,1)$，如何生成服从指数分布 $E(\lambda)$ 的随机变量？
**思路**：使用逆变换法，先求指数分布 CDF 的反函数，再代入均匀随机变量。
**解答**：
指数分布 $E(\lambda)$ 的分布函数为 $F(x) = 1 - e^{-\lambda x}$ ($x \ge 0$)。
令 $u = 1 - e^{-\lambda x}$，反解得：
$$x = -\lambda^{-1} \log(1-u)$$
因此，令 $X = -\lambda^{-1} \log(1-U)$，则 $X \sim E(\lambda)$。
由于 $1-U \sim \mathcal{U}(0,1)$，实际计算中也可简化为 $X = -\lambda^{-1} \log U$。
**技巧**：若 $U \sim \mathcal{U}(0,1)$，则 $1-U \sim \mathcal{U}(0,1)$，利用此性质可简化反函数表达式。

#### 四）多维随机变量函数的联合密度（复习与拓展） 【高频】
1. 【定义】
多维随机向量函数的联合密度求解是随机向量函数的分布（一般方法）的核心应用，常通过雅可比变换法处理。
2. 原理与推导
设 $n$ 维随机向量 $\boldsymbol{X}$ 和 $\boldsymbol{Y}$ 有相同的分布，$g: \mathbb{R}^n \to \mathbb{R}^m$ 是可测的，则 $g(\boldsymbol{X})$ 与 $g(\boldsymbol{Y})$ 有相同的分布（定理1.2）。
在极坐标变换等场景中，若 $(X,Y)$ 联合密度已知，通过变换 $x = r\cos\theta, y = r\sin\theta$，雅可比行列式为：
$$J = \begin{vmatrix} \cos\theta & -r\sin\theta \\ \sin\theta & r\cos\theta \end{vmatrix} = r \tag{7-2}$$
联合密度变换公式为 $g_{R,\Theta}(r, \theta) = f_{X,Y}(r\cos\theta, r\sin\theta) |J|$。
3. 【解题步骤】
   1. 写出原随机变量的联合密度。
   2. 给出变换公式，求出雅可比行列式 $|J|$。
   3. 代入变换公式与雅可比行列式，得到新变量的联合密度。
   4. 观察联合密度是否可分离变量，判断独立性并求边缘密度。
4. 【易错点】
   - 雅可比行列式漏加绝对值。
   - 忽略新变量的取值范围（如 $r \ge 0, \theta \in [0, 2\pi)$）。
5. 例题
**例题**：极坐标变换与正态分布生成 [Lecture 07_handout.pdf P55]
**题目**：设 $X, Y$ 独立且 $X, Y \sim \mathcal{N}(0,1)$，$(R, \Theta)$ 由极坐标变换 $X=R\cos\Theta, Y=R\sin\Theta$ 决定，求 $(R, \Theta)$ 的联合密度。
**思路**：使用雅可比变换法，先写出联合密度，再求雅可比行列式并代入。
**解答**：
$(X,Y)$ 的联合密度为 $f_{X,Y}(x,y) = \frac{1}{2\pi} \exp\left(-\frac{x^2+y^2}{2}\right)$。
极坐标变换的雅可比行列式 $|J| = r$。
代入得联合密度：
$$g_{R,\Theta}(r, \theta) = \frac{1}{2\pi} \exp\left(-\frac{r^2}{2}\right) r = \left(r e^{-r^2/2}\right) \left(\frac{1}{2\pi}\right), \quad r \ge 0, \theta \in [0, 2\pi)$$
由此可得 $R$ 与 $\Theta$ 独立，$R$ 服从 Rayleigh 分布，$\Theta \sim \mathcal{U}[0, 2\pi)$。
**技巧**：二维独立标准正态变量转极坐标时，半径和角度必然独立，角度服从均匀分布。

#### 本章小结
- **本章知识关系**：本章从随机向量函数的分布（一般方法）出发，通过分布函数法（CDF法）与雅可比变换法解决了多维变换问题；进而引出次序统计量的分布（基于多项分布思想）；最后将分布函数的反函数应用于 p 分位数与随机数生成，形成完整的逻辑闭环。
- **本章核心结论**：
  - 求解随机向量函数分布的核心是 CDF 法与雅可比变换法。
  - 次序统计量的联合分布是连续函数。
  - 连续型分布的 p 分位数满足 $F(F^{-1}(p)) = p$，且可通过 $Y = F^{-1}(U)$ 生成指定分布的随机数。
- **必背公式**：
  - p 分位数定义：$F^{-1}(p) = \inf \{x : F(x) \ge p\}$
  - 逆变换采样：$Y = F^{-1}(U), U \sim \mathcal{U}(0,1) \implies Y \sim F$
  - 雅可比行列式（极坐标）：$|J| = r$
- **【记忆】**：求分布找 CDF，多维变换找雅可比；次序统计量看组合，生成随机数用逆变换。
