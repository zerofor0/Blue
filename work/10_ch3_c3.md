### 本章总览（第3/3片）
- 核心问题：如何对随机变量进行标准化与高阶矩度量？如何利用概率不等式对分布尾部进行界限估计？
- 高频考点排序：Markov/Chebyshev 不等式放缩计算 > Jensen 不等式证明与应用 > 标准化与高阶矩概念 > 期望方差综合应用（匹配问题）。
- 方法清单：标准化变换、构造示性函数求复杂期望、利用凸函数性质证明不等式、利用二次型非负定性证明内积不等式。

#### 标准化 【必背】
【定义】设随机变量 $X$ 的期望 $\mu = E(X)$ 有限，记 $Var(X) = E[(X-\mu)^2]$ 为 $X$ 的方差，或记作 $\sigma_X^2$。则称 $\sigma_X = \sqrt{Var(X)}$ 为 $X$ 的标准差。
设 $Var(X) < \infty$，令 $Y = \frac{X - E(X)}{\sqrt{Var(X)}}$，则 $E(Y) = 0, Var(Y) = 1$。此时称 $Y$ 是 $X$ 的标准化。
1. 原理与推导：
   $$Y = \frac{X - \mu}{\sigma} \tag{8-1}$$
   $$E(Y) = E\left(\frac{X-\mu}{\sigma}\right) = \frac{1}{\sigma}(E(X) - \mu) = 0 \tag{8-2}$$
   $$Var(Y) = Var\left(\frac{X-\mu}{\sigma}\right) = \frac{1}{\sigma^2}Var(X) = 1 \tag{8-3}$$
   特别地，当 $X \sim \mathcal{N}(\mu, \sigma^2)$ 时，其标准化 $Y = \frac{X-\mu}{\sigma} \sim \mathcal{N}(0,1)$。
2. 【易错点】标准化时除以的是标准差 $\sigma$（即方差的算术平方根），而非方差 $\sigma^2$ 本身。

#### 高阶矩 【必背】
【定义】利用标准化变量 $\frac{X-\mu}{\sigma}$，可以定义衡量分布形状的高阶矩指标：
- 偏度：衡量分布的非对称性。
  $$\text{Skewness} = E\left[ \left(\frac{X-\mu}{\sigma}\right)^3 \right] \tag{8-4}$$
- 峰度：衡量分布尾部厚度（相对于正态分布的超出量）。
  $$\text{Kurtosis} = E\left[ \left(\frac{X-\mu}{\sigma}\right)^4 \right] - 3 \tag{8-5}$$
1. 【易错点】峰度公式最后需要减去 3（这是为了以正态分布作为基准，正态分布的峰度为 0）。忘记减 3 是常见计算错误。

#### Markov 不等式 【高频】
【定义】对随机变量 $X$ 和 $\varepsilon > 0$，若 $X \ge 0$ 且 $\alpha > 0$，有：
$$P(X \ge \varepsilon) \le \frac{E(X^\alpha)}{\varepsilon^\alpha} \tag{8-6}$$
1. 原理与推导：
   利用示性函数 $I$，$P(X \ge \varepsilon) = E[I(X \ge \varepsilon)]$。
   当 $X \ge 0, \alpha > 0$ 时，事件 $\{X \ge \varepsilon\}$ 等价于 $\{X^\alpha \ge \varepsilon^\alpha\}$。
   $$I(X^\alpha \ge \varepsilon^\alpha) \le \frac{X^\alpha}{\varepsilon^\alpha}$$
   两侧求期望：
   $$P(X \ge \varepsilon) = E[I(X^\alpha \ge \varepsilon^\alpha)] \le E\left[\frac{X^\alpha}{\varepsilon^\alpha}\right] = \frac{E(X^\alpha)}{\varepsilon^\alpha}$$
2. 【解题步骤】
   1. 确认随机变量非负 $X \ge 0$。
   2. 根据题目所求概率，选取合适的 $\alpha$ 和 $\varepsilon$。
   3. 计算 $E(X^\alpha)$。
   4. 代入公式求概率上界。
3. 【易错点】Markov 不等式要求 $X$ 是非负随机变量，若 $X$ 可取负值，直接使用会导致错误。

#### Chebyshev 不等式 【高频】
【定义】对任意随机变量 $X$（方差存在），有：
$$P(|X - E(X)| \ge \varepsilon) \le \frac{Var(X)}{\varepsilon^2} \tag{8-7}$$
1. 原理与推导：
   令 $Y = |X - E(X)|$，则 $Y \ge 0$。对 $Y$ 应用 Markov 不等式（取 $\alpha = 2$）：
   $$P(|X - E(X)| \ge \varepsilon) = P(Y \ge \varepsilon) \le \frac{E(Y^2)}{\varepsilon^2} = \frac{E[(X-E(X))^2]}{\varepsilon^2} = \frac{Var(X)}{\varepsilon^2}$$
2. 【解题步骤】
   1. 确认或计算随机变量的期望 $E(X)$ 和方差 $Var(X)$。
   2. 确定偏离阈值 $\varepsilon$。
   3. 代入公式计算概率上界。
3. 【易错点】Chebyshev 不等式估计的是双侧概率 $P(|X - \mu| \ge \varepsilon)$，若题目问单侧概率，直接用此公式会放大上界。

#### 内积不等式 【重点】
【定义】设 $E(X^2) < \infty$ 和 $E(Y^2) < \infty$，则有：
$$|E(XY)| \le \sqrt{E(X^2)E(Y^2)} \tag{8-8}$$
等号成立的充分必要条件是存在不全为零的常数 $a, b$ 使得 $aX + bY = 0 \quad a.s.$
1. 原理与推导：
   对于不全为零的常数 $a, b$，构造二次型：
   $$E[(aX + bY)^2] = a^2E(X^2) + 2abE(XY) + b^2E(Y^2) \ge 0$$
   写成矩阵形式为 $(a, b) \Sigma (a, b)^T \ge 0$，其中 $\Sigma = \begin{pmatrix} E(X^2) & E(XY) \\ E(XY) & E(Y^2) \end{pmatrix}$。
   由 $\Sigma$ 的非负定性知其行列式大于等于 0：
   $$\det(\Sigma) = E(X^2)E(Y^2) - [E(XY)]^2 \ge 0$$
   即得 $|E(XY)| \le \sqrt{E(X^2)E(Y^2)}$。等号成立当且仅当行列式为 0，即存在非零向量 $(a,b)$ 使 $E[(aX+bY)^2] = 0$，亦即 $aX + bY = 0 \ a.s.$。

#### Jensen 不等式 【高频】
【定义】设 $\psi$ 是 $\mathbb{R}^1$ 上的凸函数，且 $X$ 和 $\psi(X)$ 都有有限的期望，则有：
$$\psi(E(X)) \le E(\psi(X)) \tag{8-9}$$
对于严格凸的函数 $\psi$，等号成立当且仅当 $X = E(X) \ a.s.$
1. 原理与推导：
   由凸函数性质，对任意 $a, b$，存在常数 $c = g'_r(a)$（$g'_r(x)$ 为右导数），使得：
   $$\psi(b) - \psi(a) \ge c(b - a)$$
   令 $a = E(X), b = X$，则：
   $$\psi(X) - \psi(E(X)) \ge c(X - E(X))$$
   两侧求期望，因 $E(X - E(X)) = 0$，得：
   $$E(\psi(X)) - \psi(E(X)) \ge 0$$
   即 $E(\psi(X)) \ge \psi(E(X))$。
2. 【解题步骤】
   1. 判断题目中函数 $\psi(x)$ 的凹凸性（二阶导数 $\ge 0$ 为凸，$\le 0$ 为凹）。
   2. 若 $\psi$ 为凸函数，使用 $E(\psi(X)) \ge \psi(E(X))$；若 $\psi$ 为凹函数，不等号反向。
   3. 常用于证明涉及期望的不等式或优化算法（如 EM 算法）中的边界推导。

#### 期望与方差的综合应用：匹配问题 【重点】
**例题**：匹配问题（配对问题）
**题目**：某人写了 $n$ 封信, 将其装入 $n$ 个信封, 并在每个信封上分别任意地写上 $n$ 个收信人的一个地址（不重复）, 记 $X$ 为匹配正确的封数. 求 $E(X)$ 和 $Var(X)$。 [Lecture08_handout.pdf P49]
**思路**：直接求 $X$ 的分布非常困难。引入示性函数 $X_i$ 表示第 $i$ 封信是否匹配正确，则 $X = \sum_{i=1}^n X_i$。利用期望的线性性质求 $E(X)$；利用 $Var(X) = E(X^2) - [E(X)]^2$，将 $X^2$ 展开为 $\sum X_i^2 + \sum_{i \neq j} X_i X_j$ 并分别求期望。
**解答**：
定义 $X_i = I(\text{第 } i \text{ 封信匹配})$，则 $X = \sum_{i=1}^n X_i$。
对于任一封信，匹配概率为 $\frac{1}{n}$，故 $E(X_i) = \frac{1}{n}$。
$$E(X) = \sum_{i=1}^n E(X_i) = n \cdot \frac{1}{n} = 1$$
由于 $X_i$ 是示性函数，$X_i^2 = X_i$。
$$X^2 = \left( \sum_{i=1}^n X_i \right)^2 = \sum_{i=1}^n X_i^2 + \sum_{i \neq j} X_i X_j = \sum_{i=1}^n X_i + \sum_{i \neq j} X_i X_j$$
对于 $i \neq j$，$E(X_i X_j) = P(X_i=1, X_j=1) = \frac{1}{n(n-1)}$。
$$E(X^2) = E\left( \sum_{i=1}^n X_i \right) + E\left( \sum_{i \neq j} X_i X_j \right) = 1 + n(n-1) \cdot \frac{1}{n(n-1)} = 2$$
$$Var(X) = E(X^2) - [E(X)]^2 = 2 - 1^2 = 1$$
**技巧**：复杂求和变量的期望与方差，优先拆解为示性函数之和，利用期望线性性质和交叉项的联合概率求解。

### 本章小结（本片部分）
- 本章知识关系：本片在前两部分期望与方差定义及性质的基础上，引入了标准化与高阶矩以全面刻画分布特征，并利用期望的性质导出了三大核心不等式，为概率界限估计提供了理论工具。
- 本章核心结论：标准化消除了量纲影响；Markov 不等式限定了非负变量的尾部概率；Chebyshev 不等式量化了方差对集中度的贡献；Jensen 不等式建立了凸函数与期望的桥梁。
- 必背公式：
  - 标准化：$Y = \frac{X - E(X)}{\sqrt{Var(X)}}$
  - Markov 不等式：$P(X \ge \varepsilon) \le \frac{E(X^\alpha)}{\varepsilon^\alpha}$
  - Chebyshev 不等式：$P(|X - E(X)| \ge \varepsilon) \le \frac{Var(X)}{\varepsilon^2}$
  - 内积不等式：$|E(XY)| \le \sqrt{E(X^2)E(Y^2)}$
  - Jensen 不等式：$\psi(E(X)) \le E(\psi(X))$ （$\psi$ 为凸函数）
- 【记忆】标准化去量纲，马氏非负控尾端，切氏方差定集中，詹森凸函数看期望。
